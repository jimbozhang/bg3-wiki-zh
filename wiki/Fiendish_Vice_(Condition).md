# 邪魔罪恶（状态）

**邪魔罪恶**

- 受影响实体可以调用恶魔奥术，施放[奇术](Thaumaturgy.md "奇术")、[炼狱叱喝](Hellish_Rebuke.md "炼狱叱喝")、[燃烧之手](Burning_Hands.md "燃烧之手")和[火焰刀](Flame_Blade.md "火焰刀")各一次，直到下一次[长休](Long_Rest.md "长休")。

## 属性

[堆叠ID](Stack_ID.md "堆叠ID")：`MOO_BLOODPOTION_TIEFLING` [如果已应用](Conditions.md#Stack_type "状态")：替换当前

## 赋予

[燃烧之手](Burning_Hands.md "燃烧之手") (+ )
每个易燃目标受到3d6⁠⁠[火焰](Fire.md "火焰")伤害。

[火焰刀](Flame_Blade.md "火焰刀") (+ )
在手中召唤一把燃烧的[弯刀](Flame_Blade_(weapon).md "火焰刀（武器）")，持续10回合，造成3d6⁠⁠[火焰](Fire.md "火焰")伤害。它在3米（10英尺）半径范围内散发明亮光线，在6米（20英尺）半径范围内散发微弱光线。

[炼狱叱喝](Hellish_Rebuke.md "炼狱叱喝") (+ )
以火焰回应你的下一个攻击者，造成2d10⁠⁠[火焰](Fire.md "火焰")伤害。

[奇术](Thaumaturgy.md "奇术")
在[威吓](Intimidation.md "威吓")和[表演](Performance.md "表演")[技能检定](Skill_Check.md "技能检定")上获得[优势](Advantage.md "优势")。
