# 高安保险库8钥匙

不要与[清账屋保险箱钥匙8](Counting_House_Safe_n%C2%BA8_Key.md "清账屋保险箱钥匙8")混淆

高安保险库8钥匙是[第三幕](Act_Three.md "第三幕")中的一个杂物[故事道具](Story_Item.md "故事道具")。

握着这把钥匙会让人胃部翻腾，充满空虚感。

- [杂物](Miscellaneous.md "杂物")

- 稀有度：故事道具

- 价格：1 金币

- UID `S_LOW_CountingHouse_Vault8_Key` UUID `6499560e-4790-4aab-97a0-c0989088a8cb` ## 获取地点

- 在[下城区](Lower_City.md "下城区")[哀伤之邸](House_of_Grief.md "哀伤之邸")下方的[阴影之拥修道院](Cloister_of_Sombre_Embrace.md "阴影之拥修道院")内的一个珠宝盒中 X: -401 Y: -1590

  - [贵族的勒索文件](The_Patriar_Blackmail_Files.md "贵族的勒索文件")
