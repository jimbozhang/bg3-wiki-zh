# 妖精祝福 (状态)

**妖精祝福**

- 由于妖精的保护魔法，[幽影诅咒](Shadow_Curse_(Condition).md) 无法影响此生物。

## 属性

[堆叠ID](Stack_ID.md "Stack ID"): `SCL_SAFE` [若已应用](Conditions.md#Stack_type "状态"): 不应用

[堆叠优先级](Stack_priority.md "Stack priority"): 20

[更多属性](Status_properties.md "状态属性"):

- [禁用战斗日志](Status_properties/DisableCombatlog.md "状态属性/禁用战斗日志")
- [禁用头顶显示](Status_properties/DisableOverhead.md "状态属性/禁用头顶显示")
- [忽略休息](Status_properties/IgnoreResting.md "状态属性/忽略休息")

## 妖精祝福的来源

- [金银丝妖精荒野铃铛](Filigreed_Feywild_Bell.md "金银丝妖精荒野铃铛")

## 具有相同堆叠ID的状态

| 状态 | 效果 |
| --- | --- |
| [月之护盾](Moonshield_(Condition).md "月之护盾 (状态)") | 受到[月亮提灯](Moonlantern.md "月亮提灯")或[暗影提灯](Shadow_Lantern.md "暗影提灯")的光芒保护。[幽影诅咒](Shadow_Curse_(Condition).md "幽影诅咒 (状态)")不会影响此实体。 |
| 妖精祝福 | 由于妖精的保护魔法，[幽影诅咒](Shadow_Curse_(Condition).md "幽影诅咒 (状态)")无法影响此生物。 |
| [幽影诅咒](Shadow_Curse_(Condition).md "幽影诅咒 (状态)") | 若无光源，每回合将受到2d4⁠⁠[黯蚀](Necrotic.md "黯蚀")伤害。此伤害在队伍成员受幽影诅咒影响的每回合**翻倍**。 |

- 若无光源，每回合将受到2d4⁠⁠[黯蚀](Necrotic.md "黯蚀")伤害。此伤害在队伍成员受幽影诅咒影响的每回合**翻倍**。

## 备注

- 通过从[卡'尼斯](Kar'niss.md "卡'尼斯")的[月亮提灯](Moonlantern.md "月亮提灯")中解救[多利多利多利](Dolly_Dolly_Dolly.md "多利多利多利")获得。
- 更改区域（例如在[第一幕](Act_One.md "第一幕")和[第二幕](Act_Two.md "第二幕")之间旅行）可能导致您失去此状态。可以通过与[金银丝妖精荒野铃铛](Filigreed_Feywild_Bell.md "金银丝妖精荒野铃铛")互动、与多利多利多利交谈并礼貌地请求另一个祝福来刷新。
| 状态 | 效果 |
| --- | --- |
| [月之护盾](Moonshield_(Condition).md "月之护盾 (状态)") | 受到[月亮提灯](Moonlantern.md "月亮提灯")或[暗影提灯](Shadow_Lantern.md "暗影提灯")的光芒保护。[幽影诅咒](Shadow_Curse_(Condition).md "幽影诅咒 (状态)")不会影响此实体。 |
| 妖精祝福 | 由于妖精的保护魔法，[幽影诅咒](Shadow_Curse_(Condition).md "幽影诅咒 (状态)")无法影响此生物。 |
| [幽影诅咒](Shadow_Curse_(Condition).md "幽影诅咒 (状态)") | 若无光源，每回合将受到2d4⁠⁠[黯蚀](Necrotic.md "黯蚀")伤害。此伤害在队伍成员受幽影诅咒影响的每回合**翻倍**。 |

- 效果与[月亮提灯](Moonlantern.md "月亮提灯")增益和围绕[月出之塔](Moonrise_Towers.md "月出之塔")的[月之护盾](Moonshield_(Condition).md) 相同。然而，妖精祝福更方便，因为它不占用武器栏位。
