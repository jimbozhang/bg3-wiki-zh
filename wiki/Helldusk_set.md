# 黯狱套装

**黯狱**护甲是一套围绕抵抗和造成[火焰](Fire.md "火焰")伤害的[装备](Equipment.md "装备")。它由[地狱铁](Infernal_Iron.md "地狱铁")或[炼狱合金](Infernal_Alloy.md "炼狱合金")锻造而成。

## 残缺版

残缺版套装是真正的黯狱物品套装，如果[戴摩](Dammon.md "戴摩")被给予[地狱铁](Infernal_Iron.md "地狱铁")或[炼狱合金](Infernal_Alloy.md "炼狱合金")，他就可以制作出来。戴摩给队伍物品的顺序是：残缺的黯狱护甲；残缺的黯狱头盔；最后是残缺的黯狱手套。戴摩曾是[卡里西姆](Carixim.md "卡里西姆")的学徒。

[次级地狱惩戒](Lesser_Infernal_Retribution.md "次级地狱惩戒")
当你被2米（7英尺）内的敌人击中时，它可能受到1d4⁠⁠[火焰](Fire.md "火焰")伤害。
