# 剑圣手套

剑圣手套是一副稀有[手套](Gloves.md "Gloves")，它能提供三种剑类武器的熟练项，并给予近战攻击掷骰加成。

著名的卓尔，拥有金子般心灵的[崔斯特·杜垩登](Drizzt_Do'Urden.md "崔斯特·杜垩登")，并非“双剑流”战斗风格的开创者，尽管他是其中最优秀的实践者之一。

## 属性

- [手套](Gloves.md "Gloves")
- 稀有度：稀有
- 重量：0.5 千克 (1 磅)
- 价格：160 金币 / 460 金币（荣誉模式）
- UID `MAG_OfSwordmaster_Gloves` UUID `66e20a58-4bc3-4e17-a13e-510a19b23c33` Stats `MAG_OfSwordmaster_Gloves` File `ARM_Gloves_Leather_G` ### 特殊

此物品的穿戴者获得：

[刀剑精通](Sword_Mastery.md "刀剑精通")
你获得[短剑](Shortswords.md "短剑")、[长剑](Longswords.md "长剑")和[巨剑](Greatswords.md "巨剑")的[熟练项](Proficiency.md "熟练项")。此外，你的近战[攻击掷骰](Attack_roll.md "攻击掷骰")获得+1加成。

## 获取地点

- [终焉光芒旅店](Last_Light_Inn.md "终焉光芒旅店") X: -33 Y: 164：在[第二章](Act_Two.md "第二章")由[戴摩](Dammon.md "戴摩")出售

## 备注

_关于刀剑精通：_

- 尽管描述如此，+1攻击加成仅适用于武器攻击，而非所有近战攻击，因此不适用于徒手近战攻击。

## 错误

_关于刀剑精通：_

- **刀剑精通**不会增加副手近战攻击的伤害。
