## 参与情况

### 第一幕

当队伍首次通过[翠绿林地](Emerald_Grove.md "翠绿林地")大门接近[临时监狱](Makeshift_Prison.md "临时监狱")时，他们发现萨扎正被[阿尔卡](Arka.md "阿尔卡")用弩指着，阿尔卡想为她死去的弟弟[加农](Kanon.md "加农")报仇。阿尔卡的朋友[门诺斯](Memnos.md "门诺斯")试图劝阻她复仇。队伍成员可以选择通过交谈让她冷静下来或挡在阿尔卡的弩前帮助萨扎；他们也可以让萨扎死在阿尔卡手中。如果阿尔卡和门诺斯离开后队伍攻击萨扎，[_验证_](bg3wiki_colon_Verification.md "bg3wiki:Verification")她会呼叫守卫，守卫会像对待任何其他中立角色被攻击时一样做出反应。

如果队伍帮助了萨扎并与她交谈，她会主动介绍他们认识一位强大的治疗者，[真魂者女祭司迦特](Gut.md "迦特")。萨扎随后会作为[临时伙伴](Companions.md#Temporary_Companions "伙伴")加入，并必须被护送离开林地。如果队伍表明意图[解救地精萨扎](Save_the_Goblin_Sazza.md "解救地精萨扎")她，她承诺会向她的部落为他们美言，帮助他们绕过[地精营地](Goblin_Camp.md "地精营地")和[破碎圣所](Shattered_Sanctum.md "破碎圣所")的检查点。然而，当他们见到[明萨拉](Minthara.md "明萨拉")时，她试图将队伍作为俘虏呈现，以掩盖她在突袭中的失败，并被指出队伍成员都是[真魂者](True_Soul.md "真魂者")。面对惩罚的威胁，队伍可以决定是杀死她还是囚禁她。

- 如果队伍在第二次饶恕萨扎后与她交谈，通过 DC 10 [游说](Persuasion.md "游说") [属性检定](Ability_Check.md "属性检定")可以说服她分享一个秘密藏匿点的位置，坐标 X: -133 Y: 398。
- 如果萨扎未被解救并护送出林地，[解救地精萨扎](Save_the_Goblin_Sazza.md "解救地精萨扎")任务会因她的死亡而结束。
- 如果萨扎被解救并护送出林地后，所有三名地精首领都被杀死，她会在第三名地精首领死亡时立即死亡。
- 如果萨扎被解救，[明萨拉](Minthara.md "明萨拉")未被杀死但昏迷，且其他两名地精首领被杀死，她会存活下来，并可在第二幕中遇到。
- 如果迦特死亡且明萨拉被击倒后萨扎被解救，她会站在明萨拉的房间内，她的任务在[长休](Long_Rest.md "长休")后才会继续，之后明萨拉的尸体消失。过场动画仍会播放，队伍可以选择饶恕萨扎。

### 第二幕

如果萨扎在[第一幕](Act_One.md "第一幕")中存活，她可以在[月出之塔](Moonrise_Towers.md "月出之塔")的接见室中再次出现，与其他几名地精一起等待因未能夺取神器而受到的审判。队伍可以决定是解救她还是处决她（方式残忍程度不一）。如果萨扎在月出之塔被解救，将获得一个成就。

#### 解救地精

- [邪念] _放下你的折磨。解救他们。_ -1 -1 +1 -1 -1
- 守卫。释放他们。 -1 -1 +1 -1 -1

#### 自行处决

- [邪念] 剖开自己。_咔嚓咔嚓！_ 工作时要面带微笑！ +1 -1 -1
- [邪念] 我梦想着能沐浴其中的血池。给我放洗澡水。 +1 -1 -1
- 把你们的内脏洒在地板上。我想看你们流血。 +1 -1 -1
- [邪念] 乞求！窒息！去死！ +1 -1 -1
- 别再说话了。事实上——你们任何一个都别再呼吸了。 +1 -1 -1

#### 决斗审判

- 我会给你一个公平的机会。与我战斗——你要么死，要么杀了我然后自由离开。 -1 +1 +1
- [弃誓者] 这将是一场决斗审判。成为自己命运的主宰。 -1 +1 +1
- [圣武士] 这将是一场决斗审判。如你所愿，你必须为自己而战。 -1 +1 +1
- [狂战士] 我们将在**鲜血**中决定。你战斗得好，就能活。 -1 +1 +1
- [野蛮人] 动物只有在被逼入绝境时才会展现真正的力量。向我展示你的力量。_战斗。_ -1 +1 +1
- [邪念] 我会亲手做这件事，这样感觉总是更好。 -1 +1 +1

如果萨扎在月出之塔被解救，萨扎和其他地精会逃离该区域，之后不再出现。队伍随后会在对话选项中指出，他们很可能因无法获得[月亮提灯](Moonlantern.md "月亮提灯")而死于[幽影诅咒](Shadow_Curse_(Condition).md "幽影诅咒（状态）")。

### 尾声

然而，如果萨扎被饶恕，在[尾声](Epilogue.md "尾声")期间，可以在[感恩之箱](Chest_of_Grateful_Words.md "感恩之箱")中找到她的一封[信件残片](Parchment_from_Sazza.md "萨扎的信件残片")。信中说她一直在游荡，寻找仍然忠于绝对真神的人（不知道她的“女神”已经陨落），最终被关进了笼子，被_脾气暴躁的猫人_（可能是[塔巴克辛](https://forgottenrealms.fandom.com/wiki/Tabaxi)）俘虏。在信中，萨扎请求玩家角色再次拯救她。

## 伙伴好感度

### 第一幕

- 处理[翠绿林地](Emerald_Grove.md "翠绿林地")监狱中被俘的萨扎：
- 允许阿尔卡射杀萨扎：+1+1
- 挡在阿尔卡的弩和萨扎之间：+1
- 阻止阿尔卡射杀萨扎（成功的威吓或游说检定）：+1-1+1-1+1
- 在让阿尔卡放过萨扎后杀死萨扎不会引起任何伙伴好感度变化。
  - 在萨扎被关在笼子里时与她交谈：_“成交。我会把你弄出来。”_ +1 -1 -1
  - 当盖尔在场时化解局势会导致他向玩家角色提供对话，他在其中讲述了在[深水城](Waterdeep.md "深水城")的类似情况。
- 从笼子里解救萨扎后与她交谈：
- 选择_“成交，一言为定”_或_“跟我来。我会保护你的安全”_： -1 -1 -1
- 在地精营地中：
- 在明萨拉面前为萨扎辩护：-1 +1 +1 -1 -1
- 指控萨扎对明萨拉撒谎：+1 -1 +1 +1 +1
