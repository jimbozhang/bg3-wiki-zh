# 尤斯滕

| | |
| --- | --- |
| 体型 | 中级 |

**尤斯滕**是[巴尔邪教](Cult_of_Bhaal.md "巴尔邪教")的一名已故成员，其尸体可在[博德之门](Baldur's_Gate.md "博德之门")的[底城区](Undercity.md "底城区")内的[底城废墟](Undercity_Ruins.md "底城废墟")中找到。

## 相关剧情

根据其尸体旁的[羊皮卷（底城废墟）](Parchment_(Undercity_Ruins).md "羊皮卷（底城废墟）")，尤斯滕曾试图穿越[杀人谷](Manslaughter_Chasm.md "杀人谷")，但最终失败，死于[巴尔·吉斯利夫的弑神者](Ghislev.md "巴尔·吉斯利夫的弑神者")之手。
