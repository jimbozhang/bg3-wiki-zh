# 威尔/不悦

**威尔**不赞成伤害无辜者或任由其悲伤持续，以及任何邪恶行径。

他**不悦**于恶棍行为，例如杀害毫无防备者，或拒绝拯救受虐者。他憎恶对他人的不诚实与残忍，以及协助或纵容邪恶。他自幼被教导支持荣誉与英勇，因此鄙视不公与通过欺骗获利。由于被诱骗与魔鬼订立契约，威尔天生厌恶几乎所有与魔鬼相关的事物。最后，他不欣赏大多数犯罪行为或对抗合法且光荣的权威，也不欣赏忽视他自己的家族恩怨及其与父亲的关系。

**本页列出的故事和对话选项会降低[威尔](../Wyll.md "威尔")的好感度。**

## 第一幕

### 地点

#### [疮痍的海滩](../Ravaged_Beach.md "疮痍的海滩") 与 [路边山崖](../Roadside_Cliffs.md "路边山崖")

- 接受[阿斯代伦](../Astarion.md "阿斯代伦")的道歉，并说你也会做同样的事 (-1) _（[邪念缠身](../Haunted_One.md "邪念缠身")角色在此有独特台词）_
- 作为[邪念缠身](../Haunted_One.md "邪念缠身")，撕下[盖尔](../Gale.md "盖尔")的手 (-1)

#### [翠绿林地](../Emerald_Grove.md "翠绿林地")

_在[林地空洞](../The_Hollow.md "林地空洞")中：_

- 在泽弗洛与阿拉丁争论时攻击泽弗洛 (-1)
- 拒绝帮助泽弗洛说服[卡加](../Kagha.md "卡加")让提夫林留下 (-1)
- 斥责[埃莱吉斯](../Elegis.md "Elegis")害怕几个地精，或告诉她去挖自己的坟墓 (-1) _（任选其一）_
- 告诉[阿伦](../Arron.md "阿伦")提夫林难民会没事的，因为他们有魔鬼般的力量 (-1)
- 告诉由[阿沙拉克](../Asharak.md "阿沙拉克")训练的提夫林孩子们他们快死了 (-1)
- 告诉[潘迪尔娜](../Pandirna.md "潘迪尔娜")闭嘴，否则扭断她的脖子 (-1) _（除[野蛮人](../Barbarian.md "野蛮人")外所有职业可用）_
- 说服[阿尔卡](../Arka.md "阿尔卡")饶过[萨扎](../Sazza.md "萨扎")（使用[游说](../Persuasion.md "游说")、[威吓](../Intimidation.md "威吓")或[侦测思想](../Detect_Thoughts.md "侦测思想")） (-1)
  - 或 未能[威吓](../Intimidation.md "威吓")阿尔卡或对她使用[侦测思想](../Detect_Thoughts.md "侦测思想") (-1) _（任选其一）_
- 向萨扎承诺你会帮她逃脱 (-1) 并确实释放她 (-1)
- 当泽弗洛要求你杀死地精部落的首领时拒绝他 (-1)
- 作为[罗丝誓约卓尔](../Lolth-Sworn_Drow.md "罗丝誓约卓尔")，在捉弄了[莫尔](../Mol.md "莫尔")的一个朋友后与她交谈，威胁要撕出她的脊椎 (-1)
  - 如果因被偷物品而质问莫尔，[威吓](../Intimidation.md "威吓")要扯掉她的尾巴 (-1) _（除[野蛮人](../Barbarian.md "野蛮人")外所有职业可用）_
- 成功为莫尔偷到[西凡那斯神像](../Idol_of_Silvanus.md "西凡那斯神像")后，告诉她你会留着它 (-1)
  - 或者，作为[游荡者](../Rogue.md "游荡者")，建议她不要问，直接拿走 (-1)
- 选择不干预[梅利](../Meli.md "梅利")和[巴尔特](../Barth.md "巴尔特")关于挂坠盒的争论 (-1)
  - 或 告诉梅利交出挂坠盒。当他说没有时，告诉他他只能靠自己了 (-1)
  - 或 说"今天已经有一个小偷要埋了。" (-1)

_**注：** 此台词仅在[阿拉贝拉](../Arabella.md "阿拉贝拉")已死亡时可用_

- 如果从梅利那里拿回挂坠盒，让巴尔特付钱给你 (-1)

- 与[马蒂斯](../Mattis.md "马蒂斯")交谈/交易时，抓住[西尔菲](../Silfy.md "西尔菲")试图扒窃你并抓住她的手 (-1)

_在[圣池](../Sacred_Pool.md "圣池")和德鲁伊洞穴中：_

- 当卡加威胁[阿拉贝拉](../Arabella.md "阿拉贝拉")时袖手旁观 (-1)
- 解决阿拉贝拉的命运后，告诉卡加你本会直接杀了那女孩 (-5)
  - 或 告诉她你只是想看看会发生什么 (-5)
  - 或 告诉她这真是一场好戏 (-5)
  - 或 表扬卡加保护自己人 (-1)
  - 如果阿拉贝拉死了，与[拉斯](../Rath.md "拉斯")交谈并责怪孩子 (-1)
- 戳[奈蒂](../Nettie.md "奈蒂")的鸟两次 (-1)
- 作为[游荡者](../Rogue.md "游荡者")，在[默孔](../Mirkon.md "默孔")被塞壬魅惑时扒他的口袋 (-1)
  - 从塞壬手中救出默孔后，告诉他别再抽泣了 (-1)
- 告诉[阿尔菲拉](../Alfira.md "阿尔菲拉")她应该放弃她的歌 (-1)
  - 偷走阿尔菲拉的备用鲁特琴 (-1)
  - 从阿尔菲拉手中夺走鲁特琴并砸碎它 (-1)
- 踢[廷伯](../Timber.md "廷伯")松鼠 (-1)
  - 如果踢空了，再试一次踢她 (-1)
  - 作为[邪念缠身](../Haunted_One.md "邪念缠身")，当伙伴质问你踢[廷伯](../Timber.md "廷伯")时，回答它朝你冲来或你的本能很敏锐 (-1)

_当翠绿林地准备抵御地精攻击时_

- 告诉潘迪尔娜你是来看戏的 (-1)
- 告诉泽弗洛（如果泽弗洛死了则告诉阿沙拉克）是你带领部落来到林地的，然后说这地方属于[至上真神](../Absolute.md "至上真神") (-10)
  - 或 说部落来了，由卓尔带领——而你和他们在一起 (-10)
  - 或，作为[邪念缠身](../Haunted_One.md "邪念缠身")，告诉泽弗洛地精部落是你的 (-1)
- 告诉默孔你不会再救他了，或他只能靠自己了 (-1) _（措辞取决于你是否从塞壬手中救过默孔）_
- 吹响号角后背叛林地（打开大门）(-10)

#### [日照湿地](../Sunlit_Wetlands.md "日照湿地")/[腐臭沼泽](../Putrid_Bog.md "腐臭沼泽")

- [埃赛尔婶婶](../Auntie_Ethel.md "埃赛尔婶婶")消失后，告诉[德米尔](../Demir.md "德米尔")和[约尔](../Johl.md "约尔")他们只能靠自己了 (-1)
  - 或 作为[游侠](../Ranger.md "游侠")、[术士](../Sorcerer.md "术士")或[法师](../Wizard.md "法师")，告诉兄弟俩"相信更有经验的灵魂" (-1)

_在[蔓生地道](../Overgrown_Tunnel.md "蔓生地道")中：_

- 在巢穴中追逐埃赛尔婶婶时，与[洛林](../Lorin.md "洛林")交谈并向他咆哮 (-1)
- 当埃赛尔投降时，放她走并留下[梅丽娜](../Mayrina.md "梅丽娜")以换取[精灵发丝](../Auntie_Ethel's_Hair.md "埃赛尔婶婶的头发") (-5)
- 如果你杀了埃赛尔婶婶并饶过[悔恨面具](../Mask_of_Regret_(Overgrown_Tunnel).md)，之后与后者交谈并选择攻击她 (-1)
- 允许埃赛尔婶婶取走你的眼睛作为移除你体内蝌蚪的报酬 (-1)
- 复活[康纳·维德巴德](../Connor_Vinderblad.md "康纳·维德巴德")，然后选择将[离愁魔杖](../Bitter_Divorce.md "离愁魔杖")留给自己 (-1)

#### [森林](../Forest.md "森林") 与 [染疫村落](../Blighted_Village.md "染疫村落")

- 通过[动物交谈](../Speak_with_Animals.md "动物交谈")与[斯克拉奇](../Scratch.md "斯克拉奇")交谈时，朝[戈姆威克](../Gomwick.md "戈姆威克")的尸体大喊以证明他确实死了 (-1)
- 在桥上与[阿拉丁·贝诺](../Aradin_Beno.md "阿拉丁·贝诺")交谈时，说你不会在魔鬼杂种身上浪费时间 (-1)
- 打断谷仓里的熊地精和食人魔情侣 (-1)
- 向[费泽克](../Fezzerk.md "费泽克")支付大量金币以通过 (-1)
  - 或 攻击团伙，并在费泽克求饶时饶过他 (-5)
- 将[巴克斯·鲁特](../Barcus_Wroot.md "巴克斯·鲁特")从风车叶片上解下，并向他索要补偿 (-1)

#### [地精营地](../Goblin_Camp.md "地精营地")

_在门口和外部营地：_

- 应[哨兵奥拉克](../Olak.md "哨兵奥拉克")的要求将粪便涂抹在脸上 (-1)
- 告诉[追踪者格利卡](../Grikka.md "追踪者格利卡")你会从囚犯那里得到答案 (-1)
- 使用[动物交谈](../Speak_with_Animals.md "动物交谈")，告诉[凌乱的鸡](../Dishevelled_Chicken.md "凌乱的鸡")闭嘴并玩耍 (-1)
  - 或 指示鸡跑到终点柱 (-1)

_**注：** 这些选项仅在你在[巢穴](../Owlbear_Nest.md "枭熊巢穴")中杀死了[枭熊幼兽](../Owlbear_Cub.md "枭熊幼兽")时可用_

_处理[粉碎者](../Crusher.md "粉碎者")：_

- 亲吻粉碎者的脚 (-1)，然后开始舔它 (-1)
  - 或 直接长舔他的脚 (-1)
  - 如果粉碎者攻击你，击败他，然后说你已经表明立场并扶他起来 (-1)

_在[破碎圣所](../Shattered_Sanctum.md "破碎圣所")内：_

- 告诉[施虐者斯派克](../Spike.md "施虐者斯派克")让你来审问囚犯 (-1)
  - 或 使用基于种族的卓尔台词或[邪念缠身](../Haunted_One.md "邪念缠身")的独特台词 (-1) _（任选其一）_
  - 然后亲自折磨[利亚姆](../Liam.md "利亚姆") (-1)
- 审问利亚姆后，将他留在拷问架上 (-1)
- 说服[明萨拉](../Minthara.md "明萨拉")饶过萨扎，说后者不知道你是谁 (-1)

_**注：** 萨扎必须已从翠绿林地的笼子中被释放_

- 同意与明萨拉一起攻击林地，无论是否要求报酬 (-5)

_在[战狼围栏](../Worg_Pens.md "战狼围栏")中：_

- 告诉地精孩子们继续攻击[熊](../Halsin.md "哈尔辛")，瞄准他的眼睛，或使用更锋利的石块 (-1) _（任选其一）_
- 自己扔石头 (-1) _（使哈尔辛敌对）_
- 释放[哈尔辛](../Halsin.md "哈尔辛")后，拒绝帮助他拯救林地 (-5)

#### [地底深渊](../Underdark.md "地底深渊") 与 [格莱姆格斗场](../Grymforge.md "格莱姆格斗场")

- 同意为[盖克·科尔](../Gekh_Coal.md "盖克·科尔")追捕[苏拉](../Thulla.md "苏拉")，无论是否要求报酬 (-1)
- 在苏拉无助时偷走她的[速度之靴](../Boots_of_Speed.md "速度之靴") (-1)
- 击败灰矮人后，同意为[格鲁特](../Glut.md "格鲁特")消灭君主[斯波](../Spaw.md "斯波") (-1)
  - 如果[巴伦·骨篷](../Baelen_Bonecloak.md "巴伦·骨篷")自己吃了[贵族蘑菇](../Noblestalk.md "贵族蘑菇")，告诉[德里丝·骨篷](../Derryth_Bonecloak.md "德里丝·骨篷")你确切知道他做了什么 (-1)

_在格莱姆格斗场：_

- 询问[守卫马格马](../Ward_Magmar.md "守卫马格马")如何获得自己的奴隶 (-1)
- 杀死马格马和[皮斯特尔](../Pistle.md "皮斯特尔")后，告诉[斯基克皮特](../Skickpit.md "斯基克皮特")使用电梯去地面 (-1) _（说服侏儒的进一步选项不会带来任何点数）_
- 救出[尼尔](../Nere.md "尼尔")后，告诉他处决奴隶，或选择沉默 (-5) _（任选其一）_
- 支持尼尔对抗[布利斯瓦尔长老](../Brithvar.md "布利斯瓦尔长老") (-1)
- 如果你在释放尼尔前杀死了所有灰矮人（包括叛军和至上真神），当尼尔命令你时，杀死侏儒奴隶 (-5)
- 击败尼尔后，询问灰矮人关于侏儒奴隶的事，并说他们是宝贵的资产 (-5)

#### [晋升之路](../The_Risen_Road.md "晋升之路")、[渥金的休眠地](../Waukeen's_Rest.md "渥金的休眠地") 与 [散塔林会窝点](../Zhentarim_Hideout.md "散塔林会窝点")

- 从[布雷姆](../Brem.md "布雷姆")处购买[奥斯卡·费拉斯](../Oskar_Fevras.md "奥斯卡·费拉斯")后，与艺术家交谈并称他为奴隶**两次** (-5)

#### [山隘](../Mountain_Pass.md "山隘") 与 [伊雷珂养育间](../Cr%C3%A8che_Y'llek.md "伊雷珂养育间")

- 在遭遇吉斯洋基巡逻队的地点附近攻击[艾莉卡](../Ellyka.md "艾莉卡") (-1)
  - 或，[伪装](../Disguise_Self.md "伪装术")成吉斯洋基人，激怒她攻击你 (-1)

_在伊雷珂养育间：_

- 告诉[养育官寇库](../Varsh_Ko'kuu.md "养育官寇库")如果他给你[吉斯洋基蛋](../Githyanki_Egg.md "吉斯洋基蛋")，你会以最高价卖掉它或给自己做个煎蛋卷 (-1) _（任选其一）_
- 压碎吉斯洋基蛋 (-1) _（[吉斯洋基人](../Githyanki.md "吉斯洋基人")角色不可用）_

_**注：** 任何伤害或摧毁蛋的威胁都会使寇库和孵化间守卫敌对_

- 当[训练官凯斯克](../Sa'varsh_Kethk.md "训练官凯斯克")要求挑战者时，主动提出与[青年瓦尔](../Youth_Varrl.md "青年瓦尔")战斗 (-1)，然后确实杀死他 (-1)
- 告诉[维拉基斯](../Vlaakith.md "维拉基斯")你是为她带来遗物的朋友 (-1)
