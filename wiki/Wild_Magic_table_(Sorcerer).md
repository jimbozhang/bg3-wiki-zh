# 狂野魔法表（术士）

[狂野魔法术士](Wild_Magic_Sorcerer.md "狂野魔法术士")在施放1级或更高等级的法术后，有机会触发以下效果之一。术士的*角色等级*决定了可能出现的效果。

## 狂野魔法术士效果

*以下部分可以通过`{{:Wild Magic table (Sorcerer)}}`嵌入其他页面。*

| 效果 | 最低等级 | 受控混沌 | 描述 |
| --- | --- | --- | --- |
| [动作如潮](Wild_Magic_colon__Action_Surge.md "狂野魔法：动作如潮") | 1 | ✓ | 你在此回合获得一个额外动作。 |
| [朦胧术](Wild_Magic_colon__Blur.md "狂野魔法：朦胧术") | 1 |  | 9米（30英尺）内的每个生物变得[模糊](Blurred_(Condition).md "模糊（状态）")。 |
| [燃烧](Wild_Magic_colon__Burning.md "狂野魔法：燃烧") | 2 | ✓ | 6米（20英尺）内的每个生物和物品开始燃烧，每回合受到1d6⁠⁠[火焰](Fire.md "火焰")伤害。 |
| [坎比翁](Wild_Magic_colon__Cambion.md "狂野魔法：坎比翁") | 10 | ✓ | 从九层地狱的火焰枢纽召唤一个坎比翁。它对所有人都是敌对的。 |
| [猫和狗](Wild_Magic_colon__Cats_and_Dogs.md "狂野魔法：猫和狗") | 2 | ✓ | 9米（30英尺）内的每个生物随机变身为猫或狗。 |
| [武器附魔](Wild_Magic_colon__Enchant_Weapons.md "狂野魔法：武器附魔") | 1 |  | 为6米（20英尺）内的每个生物的武器附魔。他们的下一次攻击是[重击](Critical_Hit.md "重击")并额外造成1d4⁠⁠[力场](Force.md "力场")伤害。 |
| [缩小](Wild_Magic_colon__Enlarge/Reduce.md "缩小") | 2 |  | 9米（30英尺）内的每个生物随机[巨化](Enlarged_(Condition).md "巨化（状态）")或[缩小](Reduced_(Condition).md "缩小（状态）")。 |
| [纠缠术](Wild_Magic_colon__Entangle.md "狂野魔法：纠缠术") | 2 | ✓ | 在自身周围创建藤蔓地表，减缓生物速度，可能[诱捕](Ensnared_(Condition).md "诱捕（状态）")它们。 |
| [爆炸治疗](Wild_Magic_colon__Explosive_Healing.md "狂野魔法：爆炸治疗") | 1 |  | 当你用法术击中目标时，治疗3米（10英尺）内的所有生物，每使用一个[法术位](Spell_Slot.md "法术位")等级恢复1d4⁠⁠[治疗](Healing.md "治疗")。 |
| [飞行术](Wild_Magic_colon__Flight.md "狂野魔法：飞行术") | 2 |  | 你可以[飞行](Fly_(class_action).md "飞行（职业动作）")直到回合结束。 |
| [雾](Wild_Magic_colon__Fog.md "狂野魔法：雾") | 1 | ✓ | 在自身周围创建一团雾。雾内的生物被[重度遮蔽](Heavily_Obscured.md "重度遮蔽")并[目盲](Blinded_(Condition).md "目盲（状态）")。 |
| [变形术](Wild_Magic_colon__Polymorph.md "狂野魔法：变形术") | 2 | ✓ | 你变身为一只野兽。 |
| [弹力法球](Wild_Magic_colon__Resilient_Sphere.md "狂野魔法：弹力法球") | 2 | ✓ | 将自己包裹在一个奥术球体中。你不会受到球体外攻击或效果的伤害，也无法伤害球体外的任何东西。[移动速度](Movement_speed.md "移动速度")减半。 |
| [护盾术](Wild_Magic_colon__Shield.md "狂野魔法：护盾术") | 2 |  | [护甲等级](Armour_Class.md "护甲等级")提高5，并且你免疫[魔法飞弹](Magic_Missile.md "魔法飞弹")的效果。 |
| [缓慢术](Wild_Magic_colon__Slow.md "狂野魔法：缓慢术") | 2 | ✓ | 你被[减速](Slowed_(Condition).md "减速（状态）")。 |
| [术法点](Wild_Magic_colon__Sorcery_Points.md "狂野魔法：术法点") | 2 |  | 直到你的下个回合结束，你施放的每个法术都会恢复等于其[法术位](Spell_Slot.md "法术位")等级的术法点。 |
| [动物交谈](Wild_Magic_colon__Speak_with_Animals.md "狂野魔法：动物交谈") | 2 |  | 获得理解并与野兽交流的能力。 |
| [荆棘丛生](Wild_Magic_colon__Spike_Growth.md "狂野魔法：荆棘丛生") | 2 | ✓ | 将自身周围的一片地面塑造成硬刺。走在尖刺上的生物每移动1.5米（5英尺）受到2d4⁠⁠[穿刺](Piercing.md "穿刺")伤害。 |
| [召唤魔蝠](Wild_Magic_colon__Summon_Mephit.md "狂野魔法：召唤魔蝠") | 1 | ✓ | 召唤一个敌对的魔蝠。 |
| [交换](Wild_Magic_colon__Swap.md "狂野魔法：交换") | 1 |  | 每次你施放法术或[戏法](Cantrip.md "戏法")时，与一个目标交换位置。 |
| [心灵遥控](Wild_Magic_colon__Telekinesis.md "狂野魔法：心灵遥控") | 2 |  | 能够用意念举起并投掷物品和生物，直到你的回合结束。 |
| [传送](Wild_Magic_colon__Teleport_(Sorcerer).md "狂野魔法：传送（术士）") | 1 |  | 直到你的下个回合结束，你可以使用附赠动作传送至多9米（30英尺）。 |
| [驱散魔法](Wild_Magic_colon__Turn_Magic.md "狂野魔法：驱散魔法") | 2 |  | 在每回合开始时，触发一个新的随机魔法效果。 |
