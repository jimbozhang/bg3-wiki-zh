# 多余的套装

The **Unwanted Set** is a set of masterwork armour. It consists of the following pieces:

| 名称 | 插槽 | 效果 |
| --- | --- | --- |
| 多余的杰作铁手套 | [手套](Gloves.md "Gloves") ([中甲](Handwear#Medium.md#Medium "Handwear")) | [攻击掷骰](Attacks#Attack_rolls.md#Attack_rolls "Attacks") +1 [力量](Strength.md "Strength") [豁免检定](Saving_throw.md "Saving Throw") +1 |
| 多余的杰作护胫 | [鞋类](Footwear.md "Footwear") ([中甲](Footwear#Medium.md#Medium "Footwear")) | [运动](Athletics.md "Athletics") +1 |
| 多余的杰作鳞甲 | [护甲](Armour.md "Armour") ([中甲](Medium_Armour.md "Medium Armour")) | [特制材料](Exotic_Material.md "Exotic Material")将你的完整敏捷[调整值](Ability_Modifier.md "Ability Modifier")加到你的[护甲等级](Armour_Class.md "Armour Class")上。此外，此护甲不会对隐匿[检定](Ability_Check.md "Ability Check")施加[劣势](Disadvantage.md "Disadvantage")。 [抗性](Resistance.md "Resistance")至⁠[火焰](Fire.md "Fire")伤害 |

[特制材料](Exotic_Material.md "Exotic Material")
将你的完整敏捷[调整值](Ability_Modifier.md "Ability Modifier")加到你的[护甲等级](Armour_Class.md "Armour Class")上。此外，此护甲不会对隐匿[检定](Ability_Check.md "Ability Check")施加[劣势](Disadvantage.md "Disadvantage")。
[抗性](Resistance.md "Resistance")至⁠[火焰](Fire.md "Fire")伤害
