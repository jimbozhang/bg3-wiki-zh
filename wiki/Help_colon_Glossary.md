这是关于缩写词、缩略语和其他晦涩术语的**术语表**，您可能会在游戏内或关于[博德之门3](Baldur's_Gate_3.md "Baldur's Gate 3")的讨论中看到这些术语。它并非完整列表，但如果您遇到不理解的术语，可能会有所帮助。

## \*

- **5e** : [龙与地下城第五版](D&D_5e_rule_changes.md "D&D 5e rule changes")的简称，游戏基于该版本的D&D。

## A

- **AC** : [护甲等级](Armour_Class.md "Armour Class")的简称。

- **AoE** : 效果范围的简称。指影响特定区域内事物的能力。

- **ASI** : 属性值增加或属性值提升的简称。

## B

- **BG3 / BGIII** : [博德之门3](Baldur's_Gate_3.md "Baldur's Gate 3")的简称。
- **BGWiki** : [博德之门维基](https://baldursgate.fandom.com/wiki/)的简称。

## C

- **CHA** : [魅力](Charisma.md "Charisma")的简称。

- **Check** : [属性检定](Ability_Check.md "Ability Check")或技能检定的简称。

- **CON** : [体质](Constitution.md "Constitution")的简称。

## D

- **d4 / d6 / d8 / d10 / d12 / d20** : 指特定类型的骰子：d4是四面骰，d6是六面骰，依此类推。_3d6_ 表示掷三个六面骰。更多信息请参见[掷骰](Die_Rolls.md "Die Rolls")。

- **D &D / DnD**: 龙与地下城的简称。

- **DC** : [难度等级](Difficulty_Class.md "Difficulty Class")的简称，您必须达到该数字才能在属性检定、豁免检定或攻击掷骰中成功。

- **DDS** : 直接绘制表面，游戏在[模组](Modding_the_Game.md "Modding the Game")时使用的纹理。

- **DEX** : [敏捷](Dexterity.md "Dexterity")的简称。

## E

- **EA** : [抢先体验](Early_Access.md "Early Access")的简称。

## F

- **FRWiki** : [被遗忘的国度维基](https://forgottenrealms.fandom.com/wiki/)的简称。

## G

- **Gish** : 粉丝术语，指擅长物理战斗和施法的角色或构建。在游戏世界观中，它是吉斯洋基人的术语，指接受过武器和魔法双重训练的人，首次使用于1981年的《魔鬼图鉴》（指吉斯洋基人战士/法师的多职业角色）。

## H

- **HP** : [生命值](Hit_Points.md "Hit Points")的简称。

## I

- **INT** : [智力](Intelligence.md "Intelligence")的简称。

## M

- **Mod** :
  - [属性值调整值](Ability_score_modifier.md "Ability Score Modifier")的简称。
  - 模组的简称，指用户对游戏进行的修改。更多信息请参见[模组资源](Modding_Resources.md "Modding Resources")。

## N

- **NPC** : [非玩家角色](Non-player_characters.md "Non-Player Characters")的简称。

## P

- **PC** : 玩家角色的简称。可指任何可控角色，包括伙伴，或仅指在角色创建中选择的角色。

- **PFH** : [地狱小组](Panel_from_Hell.md "Panel from Hell")的简称，[抢先体验](Early_Access.md "Early Access")期间关于游戏状态的一系列演示。

- **PHB** : 玩家手册的简称，D&D第五版的核心规则书之一。

## S

- **Save** : [豁免检定](Saving_throw.md "Saving Throw")的简称。

- **STR** : [力量](Strength.md "Strength")的简称。

## T

- **Tav** : [自定义角色](Origins.md#Custom "Origins")的默认名称。有时因此用来指代他们。

- **TPK** : 全队击杀的简称，指队伍中所有人死亡。通常随后是游戏结束。

## U

- **UUID** : 通用唯一标识符，游戏在[模组](Modding_the_Game.md "Modding the Game")时使用。

## W

- **WIS** : [感知](Wisdom.md "Wisdom")的简称。
