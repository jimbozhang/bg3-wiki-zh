# 治疗套装

治疗套装是一套能够增强[治疗](Healing.md "治疗")效果、在治疗/被治疗时提供加成，并为治疗盟友或赋予他们[临时生命值](Temporary_Hit_Points.md "临时生命值")提供方式的装备。与[自我治疗套装](Self-Heal_set.md "自我治疗套装")互动，此处不重复描述。
