# 魅惑（状态组）

本文介绍的是状态组。关于具体状态，请参见[魅惑（状态）](Charmed_(Condition)..md)

**魅惑**是一个[状态组](Status_group.md "状态组")。该组中的状态是影响受影响生物行为的精神效果。其中最严重的状态会导致受影响生物被另一个生物完全控制。

## 移除

以下法术/动作可移除此状态组中的所有状态：

- [反制善恶：解除惑控](Dispel_Evil_And_Good_colon__Break_Enchantment.md "反制善恶：解除惑控")
- [心如止水](Stillness_of_Mind.md "心如止水")
- [高等复原术](Greater_Restoration.md "高等复原术")

## 免疫

以下物品可使角色免疫此状态组中的所有状态：

- [星界银剑](Silver_Sword_of_the_Astral_Plane.md "星界银剑")

以下法术/动作可使角色免疫此状态组中的所有状态：

- [安定心神](Calm_Emotions.md "安定心神")
- [奉献灵光](Aura_of_Devotion.md "奉献灵光")

以下被动特性可使角色免疫此状态组中的所有状态：

- [诱导防御](Beguiling_Defences.md "诱导防御")
- [心脏燃烧](Heart_Ablaze.md "心脏燃烧")
- [无我狂暴](Mindless_Rage.md "无我狂暴")
- [自然防护](Nature's_Ward.md "自然防护")

## 优势与劣势

以下物品可使角色在对抗此状态组中所有状态的豁免检定上获得[优势](Advantage.md "优势")：

- [卓越专注灵药](Elixir_of_Peerless_Focus.md "卓越专注灵药")

以下被动特性可使角色在对抗此状态组中所有状态的豁免检定上获得[优势](Advantage.md "优势")：

- [妖精血统](Fey_Ancestry.md "妖精血统")

## 其他

- [防护善恶](Protection_from_Evil_and_Good.md "防护善恶")说明其可免疫来自特定生物类型的魅惑效果，但实际上并不适用于魅惑。

## 魅惑状态列表

- 对魅惑者及其盟友友好。受到伤害会结束此状态。

[魅惑](Charmed_(Condition).md "魅惑（状态）")
|

- 无法攻击施法者。施法者在对话中进行[魅力](Charisma.md "魅力")[检定](Ability_Check.md "属性检定")时具有[优势](Advantage.md "优势")。

[胁迫](Coerced_(Condition).md "胁迫（状态）")
|

- 被迫为尼讷而战，每回合受到1d6⁠⁠[心灵](Psychic.md "心灵")伤害。
- 受到任何非心灵伤害时结束。

[疯狂](Crown_of_Madness_(Condition).md "疯狂（状态）")
|

- 被灌注了魔法疯狂。会攻击除施法者外最近的生物。

[恶魔之灵疯狂](Demonspirit_Madness_(Condition).md "恶魔之灵疯狂（状态）")
|

- 受影响实体被魔法疯狂驱使陷入狂怒。它会在自己的回合攻击最近的生物。
- 每回合结束时，受影响实体进行一次[感知](Wisdom.md "感知")[豁免检定](Saving_throw.md "豁免检定")。豁免成功则移除此状态。

[受控](Dominated_(Condition).md "受控（状态）")
|

- 受影响实体将跟随并协助施放此法术的施法者战斗。
- 每次受到伤害时，可能通过成功的感知[豁免检定](Saving_throw.md "豁免检定")打破施法者的控制。

\_(状态)[受控](Dominated_(Dominate_Beast)_(Condition).md "受控（支配野兽）（状态）")
|

- 受影响实体将跟随并协助施放此法术的施法者战斗。
- 每次受到伤害时，可能通过成功的感知[豁免检定](Saving_throw.md "豁免检定")打破施法者的控制。

[邪魔服从](Fiendish_Obeisance_(Condition).md "邪魔服从（状态）")
|

- 受影响实体必须服从直接控制它的邪魔的命令。

[交友术](Friends_(Condition).md "交友术（状态）")
|

- 对目标生物进行[魅力](Charisma.md "魅力")[检定](Ability_Check.md "属性检定")时具有[优势](Advantage.md "优势")。

在更高难度模式下，目标可能会指控你对其施放了惑控法术。
[盖尔狂欢](Galerian_Revelry_(Condition).md "盖尔狂欢（状态）")
|

- 被迫拖着脚步跳舞。
- 受影响实体无法移动或执行动作。对其的[攻击掷骰](Attack_roll.md "攻击掷骰")具有[优势](Advantage.md "优势")，而其攻击掷骰和敏捷[豁免检定](Saving_throw.md "豁免检定")具有[劣势](Disadvantage.md "劣势")。

[催眠凝视](Hypnotic_Gaze_(Condition).md "催眠凝视（状态）")
|

- 无法移动或执行动作、附赠动作或反应。
- 施法者在对话中进行[属性检定](Ability_Check.md "属性检定")时具有[优势](Advantage.md "优势")。
- 受到伤害时移除。
- 持续时间可通过[维持催眠凝视](Maintain_Hypnotic_Gaze.md "维持催眠凝视")延长。

[催眠](Hypnotised_(Condition).md "催眠（状态）")
|

- 无法[移动](Movement_speed.md "移动速度")或执行[动作](Action.md "动作")、[附赠动作](Bonus_action.md "附赠动作")或[反应](Reaction.md "反应")。
- 受到伤害、被[推击](Shove.md "推击")或被[协助](Help.md "协助")时移除。

[梦魔魅惑](Incubus_Charmed_(Condition).md "梦魔魅惑（状态）")
|

- 你被魔鬼魅惑，必须在战斗中支持它。受到伤害后，可对此效果再进行一次[豁免检定](Saving_throw.md "豁免检定")。

[诱惑](Lured_(Condition).md "诱惑（状态）")
|

- 受影响实体被鸟妖的诱捕旋律吸引，并在其回合移动靠近鸟妖，可能引发其他鸟妖的借机攻击。
- 受影响实体必须在回合开始时通过[DC](Dice_rolls.md#Save_DCs "骰子掷骰") 13的[感知](Wisdom.md "感知")[豁免检定](Saving_throw.md "豁免检定")，否则保持被诱惑状态。
- 受到伤害或被[推击](Shove.md "推击")会移除此状态。

[心智奴役](Mind_Mastery_(Condition).md "心智奴役（状态）")
|

- 被灰矮人心智奴役者强迫对其一名盟友进行武器攻击。
- 攻击后或被击中时结束。

[奥图迷舞](Otto's_Irresistible_Dance_(Condition).md "奥图迷舞（状态）")
|

- 无法执行[动作](Action.md "动作")或[移动](Movement_speed.md "移动速度")
- 攻击者进行[攻击掷骰](Attack_roll.md "攻击掷骰")时具有[优势](Advantage.md "优势")
- 其攻击掷骰和[敏捷](Dexterity.md "敏捷")[豁免检定](Saving_throw.md "豁免检定")具有[劣势](Disadvantage.md "劣势")

[异界誓缚束缚](Seized_by_Planar_Binding_(Condition).md "异界誓缚束缚（状态）")
|

- 受影响实体跟随并协助束缚它的施法者战斗。
- 每次受到伤害时，可能通过成功的[感知](Wisdom.md "感知")[豁免检定](Saving_throw.md "豁免检定")打破施法者的控制。

[观察者眼魔魅惑术](Spectator's_Charm_(Condition).md "观察者眼魔魅惑术（状态）")
|

- 微弱的惑控效果阻止此生物攻击观察者眼魔。任何伤害都可能打破它...

## 备注

- [反制善恶：解除惑控](Dispel_Evil_And_Good_colon__Break_Enchantment.md "反制善恶：解除惑控")仅移除由天界生物、元素生物、妖精、邪魔或不死生物造成的魅惑效果。
