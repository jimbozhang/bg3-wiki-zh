# 报应正义 (状态)

**报应正义**

- 这个巴尔的回声正在汲取新的、可怕的力量来为沙洛佛克·安基夫的死亡复仇。

## 赋予

[枯萎术](Blight.md "Blight") (+ )
对目标造成 8d8⁠⁠[黯蚀](Necrotic.md "Necrotic") 伤害。植物受到此法术的最大伤害，并在对抗它的[豁免检定](Saving_throw.md "Saving Throw")上具有[劣势](Disadvantage.md "Disadvantage")。

[死云术](Cloudkill.md "Cloudkill") (+ )
制造一团大型云雾，每回合造成 5d8⁠⁠[中毒](Poison.md "Poison") 伤害。你可以每回合重新定位云雾。

[曳光弹](Guiding_Bolt.md "Guiding Bolt") (+ )
下一次对该目标的攻击掷骰具有[优势](Advantage.md "Advantage")。

[重伤术](Harm.md "Harm") (+ )
降低目标的最大[生命值](Hit_Points.md "Hit Points")，但永远不会低于 1。

[致伤术](Inflict_Wounds.md "Inflict Wounds") (+ )
用充满你双手的[黯蚀](Necrotic.md "Necrotic")能量腐化一个生物。

[精魂守卫](Spirit_Guardians.md "Spirit Guardians") (+ )
召唤灵体来保护你。

[吸血鬼之触](Vampiric_Touch.md "Vampiric Touch") (+ )
触碰一个敌人以吸取其生命力，并恢复一半的[生命值](Hit_Points.md "Hit Points")。

## 注释

- 如果[沙洛佛克·安基夫](Sarevok_Anchev.md "Sarevok Anchev")在[阿米利桑的回声](Echo_of_Amelyssan.md "Echo of Amelyssan")、[伊拉塞拉的回声](Echo_of_Illasera.md "Echo of Illasera")和[森代的回声](Echo_of_Sendai.md "Echo of Sendai")之前被杀死，则会影响它们，允许它们使用进攻能力。
  - 阿米利桑的回声获得[死云术](Cloudkill.md "Cloudkill")和[吸血鬼之触](Vampiric_Touch.md "Vampiric Touch")
  - 伊拉塞拉的回声获得[曳光弹](Guiding_Bolt.md "Guiding Bolt")和[精魂守卫](Spirit_Guardians.md "Spirit Guardians")
  - 森代的回声获得[致伤术](Inflict_Wounds.md "Inflict Wounds")、[枯萎术](Blight.md "Blight")和[重伤术](Harm.md "Harm")
