# 闪电火花套装

闪电火花套装由一系列授予或与[闪电充能](Lightning_Charges.md "闪电充能")互动的物品组成，由[闪电火花](Yrre.md "闪电火花")伊尔尔发明。与[闪电套装](Lightning_set.md "闪电套装")中的物品协同作用，此处不重复列出。该套装包含以下部件：

| 名称 | 插槽 | 效果 |
| --- | --- | --- |
| 闪电护符 | [护符](Amulet.md "护符") | [闪电释能](Lightning_Blast.md "闪电释能") () 将你血脉中的电流集中，强化你的下一个法术或[戏法](Cantrip.md "戏法")。 ( 重充：[长休](Long_Rest.md "长休")。) |
| 闪电射手 | [主手远程](Main_Hand_Ranged.md "主手远程") ([长弓](Longbows.md "长弓")) | [闪电之血](Electric_Blood.md "闪电之血")当使用者使用此武器造成伤害时，他们获得 2 点[闪电充能](Lightning_Charges_(Condition).md "闪电充能（状态）")。 |
| 带电马甲 | [护甲](Armour.md "护甲") ([中甲](Medium_Armour.md "中甲")) | [上等材质](Superior_Material_(-1).md "上等材质 (-1)")你受到的[挥砍](Slashing.md "挥砍")伤害减少 1 点。[电流反击](Countershock.md "电流反击")当穿戴者在拥有[闪电充能](Lightning_Charges_(Condition).md "闪电充能（状态）")时受到伤害，攻击者必须通过[敏捷](Dexterity.md "敏捷")[豁免检定](Saving_throw.md "豁免检定")，否则将陷入[感电](Shocked_(Condition).md "感电（状态）")。 |
| 赋命者 | [帽子](Headgear.md "帽子") ([非护甲](Headwear#Non-Armour.md#Non-Armour "头部装备")) | [生命充能](Lifecharger.md "生命充能")当穿戴者获得[闪电充能](Lightning_Charges_(Condition).md "闪电充能（状态）")时，他们同时获得 3 点[临时生命值](Temporary_Hit_Points.md "临时生命值")。如果穿戴者的闪电充能耗尽，临时生命值将消失。 |
| 守护火花壁垒 | [护甲](Armour.md "护甲") ([服装](Clothing.md "服装")) | [高阶施法](High_Spellcasting_(+1).md "高阶施法 (+1)")你的[法术豁免 DC](Spell_save_DC.md "法术豁免 DC")获得 +1 加值。[火花壁垒护甲](Sparkswall_Armour.md "火花壁垒护甲")只要穿戴者拥有[闪电充能](Lightning_Charges_(Condition).md "闪电充能（状态）")，其[护甲等级](Armour_Class.md "护甲等级")和[豁免检定](Saving_throw.md "豁免检定")获得 +1 加值。 |
| 现实火花壁垒 | [副手近战](Off-Hand_Melee.md "副手近战") [盾牌](Shields.md "盾牌") | [闪电灵光](Lightning_Aura.md "闪电灵光") () 消耗 3 点[闪电充能](Lightning_Charges_(Condition).md "闪电充能（状态）")，释放一道电流冲击波，对附近敌人造成伤害并使其[触电](Jolted_(Condition).md "触电（状态）")。 ( 重充：[长休](Long_Rest.md "长休")。) |
| 火花之手 | [手套](Gloves.md "手套") ([非护甲](Handwear#Non-Armour.md#Non-Armour "手部装备")) | [导电攻击](Conductive_Strikes.md "导电攻击")当使用徒手攻击命中时，穿戴者获得 2 点[闪电充能](Lightning_Charges_(Condition).md "闪电充能（状态）")。[高效导体](Effective_Transmitter.md "高效导体")当被[闪电充能](Lightning_Charges_(Condition).md "闪电充能（状态）")灌注时，对金属[构装生物](Construct.md "构装生物")和穿戴金属护甲的敌人发动攻击时获得[优势](Advantage.md "优势")。 |
| 火花壁垒 | [戒指](Ring.md "戒指") | 对[闪电](Lightning.md "闪电")伤害具有[抗性](Resistance.md "抗性")。[电火花遮隔](Spark_Blocker.md "电火花遮隔")穿戴者不会被[电击](Electrocuted_(Condition).md "电击（状态）")。 |
| 闪电三叉戟 | [主手近战](Main_Hand_Melee.md "主手近战") ([三叉戟](Tridents.md "三叉戟")) | [闪电之血](Electric_Blood.md "闪电之血")当使用者使用此武器造成伤害时，他们获得 2 点[闪电充能](Lightning_Charges_(Condition).md "闪电充能（状态）")。 |
| 闪电飞靴 | [鞋靴](Footwear.md "鞋靴") ([中甲](Footwear#Medium.md#Medium "鞋靴")) | [运动](Athletics.md "运动") +1 [疾速火花](Speedy_Sparks.md "疾速火花")当穿戴者在战斗中[疾走](Dash.md "疾走")或采取类似动作时，他们获得 3 点[闪电充能](Lightning_Charges_(Condition).md "闪电充能（状态）") |
| 法术火花 | [主手近战](Main_Hand_Melee.md "主手近战") ([长棍](Quarterstaves.md "长棍")) | [闪电血脉](Electric_Veins.md "闪电血脉")当使用者使用法术或戏法造成伤害时，他们获得 2 点[闪电充能](Lightning_Charges_(Condition).md "闪电充能（状态）")。 |
| 水中火花 | [鞋靴](Footwear.md "鞋靴") ([非护甲](Footwear#Non-Armour.md#Non-Armour "鞋靴")) | [电化液体](Electrify_Water.md "电化液体")当穿戴者在战斗中站在[水](Water_(surface).md "水（地表）")中时，水会变成[带电的水](Electrified_Water.md "带电的水")。[水中火花](Watersparks.md "水中火花")如果穿戴者在[带电的水](Electrified_Water.md "带电的水")地表上开始其回合，他们获得 3 点[闪电充能](Lightning_Charges_(Condition).md "闪电充能（状态）")。
