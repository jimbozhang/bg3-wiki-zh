# 戴摩的信

**戴摩的信**是玩家角色在[终幕](Epilogue.md "终幕")中可以收到的信件之一。根据[卡菈克](Karlach.md "卡菈克")是否存活、死亡或被选为[出身](Origin.md "Origin")，它有三种变体。

一张朴素无华的便条。

## 文本

_如果卡菈克存活_ ( UID `S_EPI_Letter_DammonKarlachComp` / UUID `cf300cca-153a-44c4-a347-5cc7bbed6e90` )：

嘿 -

能麻烦你把这个转交给卡菈克吗？万分感谢！

卡菈克，

如果你正在读这封信，说明你还活着。无论用什么方法，你找到了出路。感谢诸神。我时常想起你，想知道你过得怎么样。给我写信，好吗？拜托了。

你忠实的，

戴摩

_如果卡菈克死亡_ ( UID `S_EPI_Letter_DammonKarlachDead` / UUID `82179ce0-263e-466e-b786-fefe00713438` )：

你好，

希望你一切安好，但我实在无法想象。我听说在你们击败大脑后不久，卡菈克就去世了。虽然我了解她不及你的一半，但她令人难忘。我不禁自责——想知道自己还能多做些什么。相信我，我不会再措手不及了——我已经把学习所有关于地狱金属加工的知识当作自己的事业，以防再有另一个卡菈克造访我的锻炉。但话说回来，怎么可能有另一个呢？她绝对是独一无二的。

致以同情，

戴摩

_如果玩家扮演卡菈克_ ( UID `S_EPI_Letter_DammonKarlachAvatar` / UUID `d0de24c8-2df4-4375-9ce4-494ae72ef593` )：

你忠实的，

戴摩
