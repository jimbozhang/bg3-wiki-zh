# 毒素抗性灵药

毒素抗性灵药是一种[灵药](Elixirs.md "Elixirs")，使用后可获得[中毒](Poison.md "Poison")[抗性](Damage_types.md#Resistance "伤害类型")。

在政治家和冒险者中都很受欢迎。

## 属性

- [灵药](Elixirs.md "Elixirs")
- 单次使用
- 稀有度：不常见
- 重量：0.2 千克 (0.4 磅)
- 价格：16 金币
- UID `CONS_Potion_Poison_Resistance` UUID `ae28b14c-0881-45fe-8237-9fcb1c951d3d` Stats `OBJ_Potion_Of_Poison_Resistance` ### 效果

[附赠动作](Actions.md#Resources "动作")

| ⁠[钝击](Bludgeoning.md "钝击") | 易伤 |
| --- | --- |
| ⁠[穿刺](Piercing.md "穿刺") | 抗性 |
| ⁠[黯蚀](Necrotic.md "黯蚀") | 抗性 |
| ⁠[中毒](Poison.md "中毒") | 免疫 |
| ⁠[心灵](Psychic.md "心灵") | 免疫 |
| ⁠[光耀](Radiant.md "光耀") | 抗性 |

- 获得对 ⁠[中毒](Poison.md "中毒") 伤害的[抗性](Damage_types.md#Resistance "伤害类型")，并对[中毒](Poisoned_(Condition).md "中毒 (状态)")免疫。

- [投掷](Throw.md "投掷")药水可为小范围内所有生物提供对 ⁠[中毒](Poison.md "中毒") 伤害的[抗性](Damage_types.md#Resistance "伤害类型")，并对[中毒](Poisoned_(Condition).md "中毒 (状态)")免疫。

  - 范围效果：1 米 (3 英尺) 半径
  - 持续时间：直至[长休](Long_Rest.md "长休")

## 获取地点

- 由[艾隆](Arron.md "艾隆")在[翠绿林地](Emerald_Grove.md "翠绿林地")出售
- 通过[炼金术](Alchemy.md "炼金术")合成，将[泥泞粘物](Suspension_of_Muddy_Goo.md "泥泞粘物")（从[泥魔蝠的翅膀](Mud_Mephit_Wing.md "泥魔蝠的翅膀")获取）与任意[升华物](Alchemy.md#Extractions "炼金术")结合

## 备注

- 同时提供对[恶心](Nauseous_(Condition).md "恶心 (状态)")的免疫。
- 炼金术制作所需的成分名称与提取物不同。

## 被动特性

毒素抗性灵药拥有 1 点[生命值](Hit_Points.md "生命值")。
