## 概述

每件武器要么是[_近战_](List_of_melee_weapons.md "近战武器列表")，要么是[_远程_](List_of_ranged_weapons.md "远程武器列表")，并造成特定类型的物理伤害：⁠[钝击](Bludgeoning.md "钝击")、⁠[穿刺](Piercing.md "穿刺")或⁠[挥砍](Slashing.md "挥砍")。

武器进一步分类为_军用_或_简易_。军用武器比简易武器造成更多伤害，但需要更专业的训练才能正确使用。

此外，每件武器都有一组_属性_，用于决定武器的使用方式。

### 武器栏位

每个角色都有一个_主手_栏位和一个_副手_栏位，两者都可以装备武器。

武器分为_单手_、_双手_或_两用_。单手武器只占用一个武器栏位，而双手武器占用两个。两用武器可以作为单手或双手武器使用，后者造成更大伤害。

### 熟练项

职业授予武器或武器类别的熟练项，允许使用者在攻击掷骰中添加其熟练项加值。

主手装备武器的熟练项会解锁其武器动作。副手武器不会授予武器动作。

## 简易武器

另见：[简易武器列表](List_of_simple_weapons.md "简易武器列表")

简易武器易于学习和使用。除德鲁伊、法师和术士外的所有[职业](Class.md "职业")在1级时都精通所有简易武器。否则，可以在4级时通过选择合适的[专长](Feat.md "专长")来获得简易武器熟练项。生物的[种族](Race.md "种族")和职业也可能授予其特定类型简易武器的熟练项；参见下文的[种族和职业授予的熟练项](#proficiencies-granted-by-race-and-class)。

- **德鲁伊：** [短棒](Club.md "短棒")、[匕首](Dagger.md "匕首")、[标枪](Javelin.md "标枪")、[硬头锤](Mace.md "硬头锤")、[长棍](Quarterstaff.md "长棍")、[镰刀](Sickle.md "镰刀")、[矛](Spear.md "矛")
- **法师、术士：** [匕首](Dagger.md "匕首")、[长棍](Quarterstaff.md "长棍")、[轻弩](Light_Crossbow.md "轻弩")

## 军用武器

另见：[军用武器列表](List_of_martial_weapons.md "军用武器列表")

## 即兴武器

并非实际武器的物品仍可作为[即兴武器](Improvised_Melee_Weapon.md "即兴近战武器")使用。请注意，这与使用非武器物品（因其外观足够像武器而允许作为武器使用）不同。例如，当[腊肠](Salami.md "腊肠")用作武器时，它实际上作为[短棒](Clubs.md "短棒")发挥作用，而不是作为即兴武器。

| [惑控学派](Enchantment.md "惑控学派") (+1, +2, +3) | 此武器被魔法附魔。使用此武器进行的任何攻击和伤害掷骰也会添加相应的加值（+1、+2等）。 |
| [灵巧](Finesse.md "灵巧") | 你使用力量和敏捷中较高的一个进行攻击和伤害掷骰。 |
| [轻型](Light_(weapon_property).md "轻型（武器属性）") | 如果主手武器也是轻型的，可以作为副手武器在副手使用。 |
| 范围 | 每件远程武器都有一个正常范围和一个长范围。超出正常范围但在长范围内的生物可以被攻击，但攻击者具有[劣势](Disadvantage.md "劣势")。超出长范围的生物无法被攻击。 |
| [额外范围](Extra_Reach.md "额外范围") | 具有此属性的近战武器可以攻击最多2.5米（8英尺）外的敌人，而不是常规的1.5米（5英尺）。 |
| 银化 | (未实现。) 有些怪物对非魔法武器免疫或有抗性，但对银制武器易受伤害。 |
| 特殊 | 此武器适用各种特殊规则，详见其描述。 |
| [投掷](Thrown.md "投掷") | 具有此属性的近战武器可以被投掷[[](#cite_note-3 "[")3][[](#cite_note-4 "[")4][[](#cite_note-5 "[")5]进行远程攻击，并正常受益于[属性值调整值](Ability_score_modifier.md "属性值调整值")（力量）和[熟练项加值](Proficiency_Bonus.md "熟练项加值")，而不是被视为即兴武器。这与灵巧属性结合：如果投掷具有投掷和灵巧属性的[匕首](Dagger.md "匕首")，则投掷者的敏捷调整值用于攻击和伤害掷骰。 |
| [双手](Two-Handed.md "双手") | 表示这是双手武器。使用它进行攻击时必须用双手持握，但仍可以单手持握。 |
| [两用](Versatile.md "两用") | 表示这是两用武器。它可以单手或双手使用。通常，双手使用两用武器可提供更高的最大伤害。 |

1. [↑](#cite_ref-1) 双持客[专长](Feat.md "专长")移除了此限制，并允许双持两用武器，这些武器从不轻型。

1. [↑](#cite_ref-3) 在大多数情况下，投掷具有[投掷](Thrown.md "投掷")标签的武器只会添加该武器的[基础伤害](Thrown.md#Notes "投掷")以及使用者的[属性值调整值](Ability_score_modifier.md "属性值调整值")（力量）和[熟练项加值](Proficiency_Bonus.md "熟练项加值")。额外的伤害属性，例如来自[闪亮的碎颅锤](Shining_Staver-of-Skulls.md "闪亮的碎颅锤")的[光耀](Radiant.md "光耀")伤害，不会被应用。
1. [↑](#cite_ref-4) 在大多数情况下，投掷具有[投掷](Thrown.md "投掷")标签的武器不会添加来自[武器动作](Weapon_actions.md "武器动作")的额外效果。
1. [↑](#cite_ref-5) 明确说明投掷时会添加特定类型额外伤害的物品，例如来自[闪电混语](Lightning_Jabber.md "闪电混语")的[闪电](Lightning.md "闪电")伤害，将添加其额外伤害。
1. [↑](#cite_ref-6) 扩展范围增加的距离尚未精确测量，但增加量不超过标准范围的5%。
