# 命令术

**命令术**是一个[1级惑控学派法术](Spells.md "Spells")。它允许施法者命令目标逃离、靠近、静止不动、趴下或丢弃武器。

## 描述

命令一个生物逃离、靠近、静止不动、趴下或丢弃其武器。目标必须通过一次[感知](Wisdom.md "Wisdom") [豁免检定](Saving_throw.md "Saving Throw")才能抵抗效果。

对不死生物无效。

## 备注

- 生物不会服从直接有害的命令，例如通过伤害性地表使用命令术：靠近。
- 可以说这是游戏中最有用的法术之一，该法术不需要[专注](Concentration.md "专注")，并且可以在[序幕](Prologue.md "序幕")中于[鹦鹉螺](Nautiloid.md "鹦鹉螺")上立即使用——并在游戏的[全部](Act_One.md "第一幕")[三](Act_Two.md "第二幕")[幕](Act_Three.md "第三幕")中——暂时[瘫痪](Command_colon__Drop.md "命令术：丢弃")（大多数）对手或迫使他们[丢弃](Command_colon__Drop.md "命令术：丢弃")武器，通常能产生高度有利的效果。
- [术士](Sorcerer.md "术士")可以使用[超魔：延长法术](Metamagic_colon__Extended_Spell.md "超魔：延长法术")将命令术的持续时间加倍。值得注意的是，目标只在法术初始施放时有几率进行豁免检定；如果失败，他们实际上会被瘫痪两轮，且没有机会再次检定。
