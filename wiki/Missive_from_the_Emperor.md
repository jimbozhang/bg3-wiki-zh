# 君主的信函

**君主的信函**是玩家角色在[终章](Epilogue.md "终章")中可以收到的信件之一。根据玩家角色或[卡菈克](Karlach.md "Karlach")是否转变为[夺心魔](Mind_flayer.md "夺心魔")，以及与[君主](The_Emperor.md "君主")是否存在持续合作，共有四种变体。

一张朴素无华的便条。

## 属性

- [笔记](Notes.md "笔记")

- 作者：[君主](The_Emperor.md "君主")

- 稀有度：普通

- 重量：0.05 千克 (0.1 磅)

- 价格：14 金币

## 文本

_如果玩家角色支持君主且无人转变为夺心魔_ ( UID `S_EPI_Emperor` / UUID `9eecd32e-ade2-4397-ac3e-cdc9f091e05e` )：

我的盟友，

尽管书面交流效率不高，但足以弥补我们如今心灵之间的距离。我写信表达我的遗憾——我将无法出席你的庆典，尽管我感谢你想起我。我未曾忘记你。在我恢复影响力的过程中，当需要时我必定会召唤你，我希望你会响应这样的召唤。我们的合作非常有益，我看不到任何终止进一步合作机会的理由。

你的朋友，

君主

_如果玩家角色支持君主且卡菈克转变为夺心魔，或玩家角色转变为夺心魔并选择与君主分道扬镳_ ( UID `S_EPI_EmperorPlayerMF` / UUID `646bc376-97dc-420c-8804-318c29b73e47` )：

我的朋友，

我们不像灵吸怪那样交流，但我会允许这样。将其视为对你过去生活的一种放纵，一种你仍执着于“庆典”概念的生活。我无法责怪你——我自己也曾如此。我花了一些时间才接受自己的进化，并认识到这确实是一种进化。

今晚创造你必须创造的记忆，以缓解你向未来的过渡。终有一天，你如此珍视的人们将变成仅仅是——美好的回忆，仅此而已。当那一刻到来时，召唤我。我的心智将永远向你敞开。

君主

_如果玩家角色支持君主，转变为夺心魔并决定重建护盾骑士团_ ( UID `S_EPI_Letter_EmperorAllies` / UUID `9c3e2dfb-d399-4c44-ade8-db6181d92700` )：

随着你掌握新力量，你将学会像灵吸怪那样交流——但我要承认，偶尔提笔写信是一种愉快的新奇体验。

我已经对我的总部——我们的总部——做了必要的调整。护盾骑士团必须重建，房间需要及时重新安置人员，但目前精灵之歌下方的房间将提供安全的港湾，以及在城市中不被察觉地移动的方法。

今晚从你的老伙伴那里寻求你能得到的慰藉，不要因为他们那些只有你灵吸怪心智才能看到的、过于人性的缺点而评判他们。你的旧生活在这段适应期会给你带来安慰——只要你不让它阻碍你走向新的生活。

君主

\_如果玩家角色支持君主，转变为夺心魔**并**选择“我们需要想得更远大”_ ( UID `S_EPI_Letter_EmperorBigPlans` / UUID `f5304063-f97b-4076-809a-ecfc323eea40` )：

我写信在你庆典之夜带来好消息。我已完成对剑湾辛迪加的巡视。安姆、科米尔、深水城和卡利姆沙都显示出有希望的线索。我已与散塔林会的灵吸怪尼希尔交流，并使其成为盟友。通过他，我们将控制深水城的财富流动，并将我们的影响力延伸至遥远的北方路斯坎。同时，在南方，我在卡利姆沙建立了“联系人”——他们完全听命于我。内陆的科米尔将是最棘手的——我认为在那里我们必须巧妙地绕过法律限制，但只要我们控制了合适的代表，那也将落入我们手中。与你的前盟友尽情欢乐吧，因为我们可能还需要他们。保持亲近，等待我的消息。很快，我的朋友，剑湾上的一切言语和思想，若非我们知晓并同意，将无法传播。

直至我们重聚，

君主

## 注释

- 取决于玩家角色是否支持君主。
- 在信件的一个版本中，君主提到 ⁠[尼希尔](https://forgottenrealms.fandom.com/wiki/Nihiloor) 作为他最近在剑湾旅行期间结成的盟友。尼希尔是 ⁠[散塔林会](https://forgottenrealms.fandom.com/wiki/Xanathar%27s_Thieves_Guild) 在[深水城](Waterdeep.md "深水城")的灵吸怪同伙，出现在冒险 ⁠[深水城：龙之劫](https://forgottenrealms.fandom.com/wiki/Waterdeep:_Dragon_Heist) 中。
