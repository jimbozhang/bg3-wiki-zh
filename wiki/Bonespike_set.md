# 骨刺套装

**骨刺**是一套完整的[装备](Equipment.md "Equipment")部件，通常在[狂暴](Rage.md "Rage")时为使用者提供增益，或在穿戴时提供新的能力或被动效果。它们没有熟练项要求。

[凶蛮跳跃](Brutal_Leap.md "凶蛮跳跃") (+ )
跃向目标并可能将其击至[倒伏](Prone_(Condition).md "倒伏（状态）")。（充能：每回合。）

[直觉闪避](Evasive_Instinct.md "直觉闪避")
只要你不穿戴[护甲](Armour.md "Armour")或手持[盾牌](Shields.md "Shields")，你的[护甲等级](Armour_Class.md "护甲等级")和[豁免检定](Saving_throw.md "豁免检定")获得+1加值。

[灵巧跳跃](Refined_Vaulting.md "灵巧跳跃")
[跳跃](Jump.md "跳跃")距离增加1.5米（5英尺）。
