# 精通诗人激励 (职业动作)

本文介绍由阿尔菲拉授予的职业动作。关于吟游诗人在高等级获得的被动特性，请参见 [精通诗人激励 (被动特性)](Improved_Bardic_Inspiration_(passive_feature).md)。其他用法，请参见 [精通诗人激励 (消歧义)](Improved_Bardic_Inspiration_(disambiguation).md)。

**精通诗人激励** 是一种职业动作，可由 [阿尔菲拉](Alfira.md "阿尔菲拉") 授予，前提是她存活至 [第二幕](Act_Two.md "第二幕")。此能力允许吟游诗人激励其盟友，类似于 [诗人激励](Bardic_Inspiration_(class_action).md "诗人激励 (职业动作)")，但骰子始终为 +1d12，且无需消耗资源，每长休可使用一次。

## 描述

通过你的表演激励盟友超越其能力。他们可为下一次 [攻击掷骰](Attack_roll.md "攻击掷骰")、[属性检定](Ability_Check.md "属性检定") 或 [豁免检定](Saving_throw.md "豁免检定") 添加 +1d12 奖励。

## 属性

消耗
[附赠动作](Actions.md#Resources "动作")
详情
射程：18 米（60 英尺）
目标：1 名尚未拥有 [诗人激励](Bardic_Inspiration_(Condition).md "诗人激励 (状态)") 的盟友
充能：[长休](Long_Rest.md "长休")

## 状态：精通诗人激励

**[精通诗人激励](Improved_Bardic_Inspiration_(Condition).md "精通诗人激励 (状态)")**

持续时间：直至 [长休](Long_Rest.md "长休")

- 可使用 [诗人激励](Bardic_Inspiration_(class_action).md "诗人激励 (职业动作)") 为下一次 [攻击掷骰](Attack_roll.md "攻击掷骰")、[属性检定](Ability_Check.md "属性检定") 或 [豁免检定](Saving_throw.md "豁免检定") 添加 +1d12 奖励。

## 学习方式

其他学习途径：

- 仅当 [阿尔菲拉](Alfira.md "阿尔菲拉") 存活至第二幕结束，且对话队伍成员同意在 [月出之塔](Moonrise_Towers.md "月出之塔") 王座厅内为提夫林孩子们表演时，才能获得此职业动作。
  - 在第一幕 [翠绿林地](Emerald_Grove.md "翠绿林地") 遇到阿尔菲拉时，不得告知她放弃吟游诗人身份。
  - 对话队伍成员也必须是 [吟游诗人](Bard.md "吟游诗人")。
  - 队伍中多名吟游诗人可通过依次与阿尔菲拉对话获得此能力。

## 注释

- 此效果通过切换被动触发，而非 [反应](Actions.md#Reactions "动作")，因此玩家对此奖励的使用时机和方式控制较少。例如，豁免检定奖励会在受影响角色首次被 *任何* 攻击或能力（无论敌友）针对时立即消耗，而非仅在敌方攻击或能力可能阻止或减轻效果时使用。
- 在开发者注释中，此职业动作内部被称为 _Shining Dawn_（闪耀黎明），可能参考了她在第一幕的歌曲 _Weeping Dawn_（哭泣黎明）。

## 错误

- 由于编码错误，此状态还会授予诗人激励资源和 d6 诗人激励的反应。使用其中一个反应会消耗诗人激励次数，但不会移除精通诗人激励状态，导致非吟游诗人角色在状态结束前拥有一个空的、无法补充的诗人激励次数槽。
