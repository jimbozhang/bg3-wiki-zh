# 束缚 (状态组)

本文介绍的是状态组。关于具体状态，请参见 [束缚 (状态)](Restrained_(Condition)..md)

**SG_Restrained** 是一个 [状态组](Status_group.md "状态组")。该组包含所有会物理限制生物移动的状态。

## 移除

以下法术/动作会移除此状态组中的所有状态：

- [协助](Help.md "Help")
- [解缚打击](Unshackling_Strike.md "Unshackling Strike")

以下状态会移除此状态组中的所有状态：

- [必然决心](Inevitable_Resolve_(Condition).md "必然决心 (状态)")

## 免疫

以下物品会使角色免疫此状态组中的所有状态：

- [自由行动药水](Elixir_of_Guileful_Movement.md "自由行动药水")
- [自由动作戒指](Ring_of_Free_Action.md "自由动作戒指")

以下法术/动作会使角色免疫此状态组中的所有状态：

- [行动自如](Freedom_of_Movement.md "行动自如")

以下被动特性会使角色免疫此状态组中的所有状态：

- [自然步伐](Nature's_Step.md "自然步伐")

以下状态会使角色免疫此状态组中的所有状态：

- [吸收精华：伊拉萨拉](Absorbed_Essence_colon__Illasera_(Condition).md "吸收精华：伊拉萨拉 (状态)")
- [穿行者](Weavewalker_(Condition).md "穿行者 (状态)")

## 优势与劣势

以下物品会使角色在对抗此状态组中所有状态的 [豁免检定](Saving_throw.md "豁免检定") 时获得 [优势](Advantage.md "优势")：

- [力量布衣](The_Mighty_Cloth.md "力量布衣")

## SG_Restrained 中的状态列表

- 被黑暗触须束缚。受影响实体无法移动，每回合受到 3d6⁠⁠[钝击](Bludgeoning.md "钝击") 伤害。

- 对该实体的 [攻击掷骰](Attack_roll.md "攻击掷骰") 具有 [优势](Advantage.md "优势")，而该实体的攻击掷骰和 [敏捷](Dexterity.md "敏捷") [豁免检定](Saving_throw.md "豁免检定") 具有 [劣势](Disadvantage.md "劣势")

[卡尔萨斯锁链](Chains_of_Karsus_(Condition).md "卡尔萨斯锁链 (状态)")
|

- 耐色脑过于强大，即使是瞬间之神的魔法也无法完全控制它——[卡尔萨斯强迫术](Karsus'_Compulsion.md "卡尔萨斯强迫术") 无法完全支配它。

- 该法术反而暂时束缚了大脑的意志。要完成支配，你必须将其击倒。

[绞缠](Constricted_(Condition).md "绞缠 (状态)")
|

- 受影响实体无法移动。对其的 [攻击掷骰](Attack_roll.md "攻击掷骰") 具有 [优势](Advantage.md "优势")，而其攻击掷骰和 [敏捷](Dexterity.md "敏捷") [豁免检定](Saving_throw.md "豁免检定") 具有 [劣势](Disadvantage.md "劣势")。
- 该实体每回合受到 2d6+2⁠⁠[穿刺](Piercing.md "穿刺") 和 1d4⁠⁠[黯蚀](Necrotic.md "黯蚀") 伤害。
- 可通过 [协助](Help.md "协助") 移除。

[诱捕](Ensnared_(Condition).md "诱捕 (状态)")
|

- 受影响实体无法移动。
- 对其的 [攻击掷骰](Attack_roll.md "攻击掷骰") 具有 [优势](Advantage.md "优势")，而其攻击掷骰和 [敏捷](Dexterity.md "敏捷") [豁免检定](Saving_throw.md "豁免检定") 具有 [劣势](Disadvantage.md "劣势")。

\_(状态)[诱捕](Ensnared_(Net)_(Condition).md "诱捕 (渔网) (状态)")

- ⁠[挥砍](Slashing.md "挥砍") 和 ⁠[火焰](Fire.md "火焰") 伤害会移除渔网。

\_(状态)[诱捕](Ensnared_(Ensnaring_Strike)_(Condition).md "诱捕 (诱捕打击) (状态)")

[绞杀](Garrotted_(Condition).md "绞杀 (状态)")
|

- 受影响实体正被绞杀术勒住。其处于 [沉默](Silenced_(Condition).md) 状态，每回合受到 3d6⁠⁠[钝击](Bludgeoning.md "钝击") 伤害。
- 在状态结束前，该实体和绞杀者均无法移动。
- 如果该实体和绞杀者被强制移动超过 5 米（17 英尺）的距离，此状态结束。

- 对其的 [攻击掷骰](Attack_rolls.md "攻击掷骰") 具有 [优势](Advantage.md "优势")，而该实体的攻击掷骰和 [敏捷](Dexterity.md "敏捷") [豁免检定](Saving_throw.md "豁免检定") 具有 [劣势](Disadvantage.md "劣势")。

- [抗性](Resistant.md "抗性") ⁠[火焰](Fire.md "火焰") 伤害。

[自然之怒](Nature's_Wrath_(Condition).md "自然之怒 (状态)")

- 无法移动。
- 对受影响生物的 [攻击掷骰](Attack_roll.md "攻击掷骰") 具有 [优势](Advantage.md "优势")。
