# 烂肉症（状态组）

**烂肉症**是一个[状态组](Status_group.md "状态组")。疾病通常是普通的，可以自行痊愈，或在各种药水和治疗能力的帮助下治愈。

## 移除

以下物品会移除此状态组中的所有状态：

- [活力药水](Potion_of_Vitality.md "活力药水")
- [詹尼尔的手套](Jhannyl's_Gloves.md "詹尼尔的手套")

以下法术/动作会移除此状态组中的所有状态：

- [圣疗：治愈](Lay_on_Hands_colon__Cure.md "圣疗：治愈")
- [恢复](Lesser_Restoration.md "恢复")
- [治疗](Heal.md "治疗")

## 免疫

以下法术/动作会使角色免疫此状态组中的所有状态：

- [假死术](Feign_Death.md "假死术")
- [英雄盛宴](Heroes'_Feast.md "英雄盛宴")

以下被动特性会使角色免疫此状态组中的所有状态：

- [神佑](Divine_Health.md "神佑")
- [自然防护](Nature's_Ward.md "自然防护")
- [百病不侵](Purity_of_Body.md "百病不侵")

## 烂肉症状态列表

- 在[攻击掷骰](Attack_roll.md "攻击掷骰")和[属性检定](Ability_Check.md "属性检定")上具有[劣势](Disadvantage.md "劣势")。
- 每回合必须成功通过一次[豁免检定](Saving_throw.md "豁免检定")，否则毒素将进入下一阶段。成功3次后痊愈。失败3次后，会感染与所施放的[疫病术](Contagion.md "疫病术")变体相对应的疾病。

[蠕行啃咬](Crawling_Gnaw_(Condition).md "蠕行啃咬（状态）")
|

- 目标感染了僵尸粘液。如果受影响的实体在感染消失前死亡，它将暂时作为新生僵尸复活。
- 新生僵尸在数据上与[被召唤的僵尸](Summoned_Zombie.md "被召唤的僵尸")相同，只是它们只有10点[生命值](Hit_Points.md "生命值")，并且具有[新生僵尸](Newborn_Zombie_(Condition).md "新生僵尸（状态）")状态，导致它们每回合受到1⁠⁠[黯蚀](Necrotic.md "黯蚀")伤害。
- 新生僵尸会跟随原始的被召唤僵尸。

[发烧](Feverish_(Condition).md "发烧（状态）")
|

- 体温逐渐升高。在使用力量的[属性检定](Ability_Check.md "属性检定")和[豁免检定](Saving_throw.md "豁免检定")上具有[劣势](Disadvantage.md "劣势")。
- 持续时间结束时，进行一次[DC](Dice_rolls.md#Save_DCs "掷骰") 15的[体质](Constitution.md "体质")[豁免检定](Saving_throw.md "豁免检定")。失败将施加[污秽热](Filth_Fever_(Condition).md "污秽热（状态）")状态。

[污秽热](Filth_Fever_(Condition).md "污秽热（状态）")
|

- 被高烧压垮。在[攻击掷骰](Attack_roll.md "攻击掷骰")，以及使用力量的[属性检定](Ability_Check.md "属性检定")和[豁免检定](Saving_throw.md "豁免检定")上具有[劣势](Disadvantage.md "劣势")。
- 持续时间结束时，进行一次[DC](Dice_rolls.md#Save_DCs "掷骰") 15的[体质](Constitution.md "体质")[豁免检定](Saving_throw.md "豁免检定")。失败将重新施加污秽热状态。
- 持续时间：25回合

[烂肉症](Flesh_Rot_(Condition).md "烂肉症（状态）")
|

- 肉体正在腐烂。
- 在使用魅力的[属性检定](Ability_Check.md "属性检定")和[豁免检定](Saving_throw.md "豁免检定")上具有[劣势](Disadvantage.md "劣势")。
- 对所有伤害[易伤](Damage_types.md "伤害类型")。
- 持续时间结束时，进行一次[DC](Dice_rolls.md#Save_DCs "掷骰") 15的[体质](Constitution.md "体质")[豁免检定](Saving_throw.md "豁免检定")以移除状态，否则将重新施加。

[重伤术](Harm_(Condition).md "重伤术（状态）")
|

- 受影响实体的最大[生命值](Hit_Points.md "生命值")已因[重伤术](Harm.md "重伤术")法术造成的伤害而减少
- 此状态是一种**疾病**

[感染](Infected_(Condition).md "感染（状态）")
|

- 在体质[豁免检定](Saving_throw.md "豁免检定")上具有[劣势](Disadvantage.md "劣势")，且[体质](Constitution.md "体质")降低1。

\_(状态)[感染](Infested_(Bursting_Brood)_(Condition).md "感染（爆裂幼蛛）（状态）")
|

- 爬满微型蜘蛛，每回合造成1d4穿刺伤害和1d4黯蚀伤害。
- 在你的回合开始时，你可以进行一次[敏捷](Dexterity.md "敏捷")[豁免检定](Saving_throw.md "豁免检定")以尝试移除感染。

[感染](Infested_(Condition).md "感染（状态）")
|

- 爬满微型蜘蛛，每回合造成1d4穿刺伤害和1d4中毒伤害。
- 当被近战攻击击中时，感染会传播给攻击者。死亡时，感染会传播给附近所有生物。
- 被⁠[火焰](Fire.md "火焰")或⁠[强酸](Acid.md "强酸")伤害移除。
- 不伤害[伊特怪](Ettercap.md "伊特怪")或蜘蛛。

[腐烂](Rotting_(Condition).md "腐烂（状态）")
|

- 开放性伤口开始腐烂。
- 在使用魅力的[属性检定](Ability_Check.md "属性检定")和[豁免检定](Saving_throw.md "豁免检定")上具有[劣势](Disadvantage.md "劣势")。
- 持续时间结束时，进行一次[DC](Dice_rolls.md#Save_DCs "掷骰") 15的[体质](Constitution.md "体质")[豁免检定](Saving_throw.md "豁免检定")以移除状态。失败将导致你获得[烂肉症](Flesh_Rot_(Condition).md "烂肉症（状态）")。

[海疫](Sea_Plague_(Condition).md "海疫（状态）")
|

- 受影响实体布满粘稠油汗，在使用[力量](Strength.md "力量")的[属性](Ability.md "属性")[属性检定](Ability_Check.md "属性检定")、[豁免检定](Saving_throw.md "豁免检定")和[攻击掷骰](Attack_roll.md "攻击掷骰")上具有[劣势](Disadvantage.md "劣势")。

[败血症](Septic_(Condition).md "败血症（状态）")
|

- 受影响实体的[体质](Constitution.md "体质")降低1，并且在体质[豁免检定](Saving_throw.md "豁免检定")上具有[劣势](Disadvantage.md "劣势")

[罹患失明症](Stricken_With_Blinding_Sickness_(Condition).md "罹患失明症（状态）")
|

- 灼痛的指甲刺入受影响实体的眼睛。它在[感知](Wisdom.md "感知")[属性检定](Ability_Check.md "属性检定")和[豁免检定](Saving_throw.md "豁免检定")上具有[劣势](Disadvantage.md "劣势")，并且还[目盲](Blinded_(Condition)..md)

[罹患污秽热](Stricken_With_Filth_Fever_(Condition).md "罹患污秽热（状态）")
|

- 受影响实体布满粘稠油汗，在使用[力量](Strength.md "力量")的[属性检定](Ability_Check.md "属性检定")、[豁免检定](Saving_throw.md "豁免检定")和[攻击掷骰](Attack_roll.md "攻击掷骰")上具有[劣势](Disadvantage.md "劣势")。

[罹患烂肉症](Stricken_With_Flesh_Rot_(Condition).md "罹患烂肉症（状态）")
|

- 在自身恶臭的皮肤缸中腐烂，受影响实体在使用[魅力](Charisma.md "魅力")的[属性检定](Ability_Check.md "属性检定")上具有[劣势](Disadvantage.md "劣势")，并且对所有伤害[易伤](Damage_types.md#Vulnerability "伤害类型")。

[罹患心火](Stricken_With_Mindfire_(Condition).md "罹患心火（状态）")
|

- 受影响实体的大脑在沸腾的热血炖汤中烹煮。它在使用[智力](Intelligence.md "智力")的[属性检定](Ability_Check.md "属性检定")上具有[劣势](Disadvantage.md "劣势")，并且在战斗中表现得像[困惑](Confused_(Condition).md "困惑（状态）")。

[罹患癫痫](Stricken_With_Seizures_(Condition).md "罹患癫痫（状态）")
|

- 不自主的抽搐掠过受影响实体，使其在使用[敏捷](Dexterity.md "敏捷")的[属性检定](Ability_Check.md "属性检定")、[豁免检定](Saving_throw.md "豁免检定")和[攻击掷骰](Attack_roll.md "攻击掷骰")上具有[劣势](Disadvantage.md "劣势")。

[罹患粘稠厄运](Stricken_With_Slimy_Doom_(Condition).md "罹患粘稠厄运（状态）")
|

- 受影响实体渗出蜡质、球状的血液。它在使用[体质](Constitution.md "体质")的[属性检定](Ability_Check.md "属性检定")和[豁免检定](Saving_throw.md "豁免检定")上具有[劣势](Disadvantage.md "劣势")。
- 此外，每当它受到伤害时，会[震慑](Stunned_(Condition)直到其下一回合结束.md)。
