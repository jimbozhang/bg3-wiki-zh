# 神秘遗物

一个由黑铁制成的多面体盒子，上面刻有脉动的符文。内部有东西在骚动。

## 属性

- [杂物](Miscellaneous.md "杂物")
- 稀有度：故事道具
- 重量：0.5 千克 (1 磅)
- 价格：1 金币
- UID `S_GLO_PuzzleBox` UUID `326324f9-0920-8f0f-3e85-3781fbf48282` ## 状态：誓缚

**[誓缚](Bound_(Condition).md "誓缚 (状态)")**

- 这件遗物被强大的灵气绑定给其当前所有者。
- 誓缚物品无法从你的物品栏中移除。

## 获取地点

- 由 [影心](Shadowheart.md "影心") 携带

## 备注

- 如果影心死亡、未被招募，或在首次接近 [地精营地](Goblin_Camp.md "地精营地") 或 [山隘](Mountain_Pass.md "山隘") 时不在队伍中，遗物会被传送至一名队伍成员的物品栏中，以在触发的过场动画中保护他们免受 [至上真神](Absolute.md "至上真神") 的影响。
- 可以与遗物互动并调查，但无法打开。
  - 上面覆盖着 [吉斯洋基语](Githyanki_vocabulary.md "吉斯洋基语")。
  - 这是游戏中唯一具有 **誓缚** 属性的物品（除了 [塞尔死灵法术](Necromancy_of_Thay.md "塞尔死灵法术")，它仅在 [特定情况](Unlock_the_Ancient_Tome.md "解锁古代典籍") 下才会被誓缚）：_"这件遗物被强大的灵气绑定给其当前所有者。"_
  - 在物品栏界面拾起并持有时，它会发出心跳声。
- [吉斯洋基人](Githyanki.md "吉斯洋基人") 在 [晋升之路](The_Risen_Road.md "晋升之路") 上的巡逻队正在搜寻这件遗物，但它——或其中包含的存在——有自己的意志，并表明它不希望被他们找到。
- 如果其他角色设法占有了遗物，在影心在队伍中（且在附近）时与之互动将不起作用。
- 遗物的形状像一个 [二十面体](https://zh.wikipedia.org/wiki/二十面体)（也称为 d20——《龙与地下城》及其他一些桌面角色扮演游戏中的主要骰子）。它被错误地称为棱镜，而实际上它是一个正多面体。 [二十面体棱镜](https://polytope.miraheze.org/wiki/Icosahedral_prism) 需要在基础二十面体之外额外 20 个三角形。
- 遗物总是在 [第一幕](Act_One.md "第一幕") 结束前获得。如果扮演的角色拒绝所有招募影心的机会，并且没有从她那里拿走它，它会在扎营时出现。
- 开发《博德之门3》的拉瑞安工作室 CEO 斯文·温克（Swen Vincke）曾表示，确保玩家角色最终拥有这件遗物是多么困难（Gamespot on YouTube: [The Box That Broke Baldur's Gate 3](https://www.youtube.com/watch?v=BHW_KFP7i8I)）。
- 饰演影心的詹妮弗·英格利希（Jennifer English）曾表示，她在动作捕捉表演中使用一盒伯爵茶作为遗物。
