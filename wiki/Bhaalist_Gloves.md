# 巴尔信徒手套

巴尔信徒手套是一副非常稀有的[Gloves](Gloves.md "手套")，允许穿戴者对目标使用[Garrotte](Garrotte.md "绞杀术")。

当谋杀之主巴尔（Bhaal）被凡人化时，他知道自己时日无多。他卑劣的阴谋为他树敌众多。那些日子终有尽头，巴尔确实被杀死了，杀死他的不是神，而是一个名叫[赛瑞克](https://forgottenrealms.fandom.com/wiki/Cyric)的鹰钩鼻男子。

## 属性

- [Gloves](Gloves.md "手套")
- 稀有度：非常稀有
- 重量：0.5 kg (1 lb)
- 价格：540 gp
- UID `MAG_Bhaalist_Gloves` UUID `afd74d05-7c24-45ec-8033-84f365e6ea5f` Stats `MAG_Bhaalist_Gloves` File `ARM_Gloves_Leather_G` ### 特殊

此物品的穿戴者获得：

[Attack Rolls](Attack_roll.md "攻击掷骰") +1

[Garrotte](Garrotte.md "绞杀术") ()
用阴影绳索缠绕[Humanoid](Humanoid.md "类人生物")生物的喉咙，开始对其施加[Garrotting](Garrotted_(Condition).md "绞杀（状态）")。（充能：[Short rest](Short_rest.md "短休")。）
