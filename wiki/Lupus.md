# 天狼座

| | |
| --- | --- |
| 体型 | 中级 |
| 类型 | [野兽](Beast.md "野兽") |
| 种族 | [狼](Wolf.md "狼") |
| 等级 | 3 |
| 生命值 | 11 |
| 护甲等级 | 13 |
| 先攻 | +2 |

**天狼座**是通过[驯兽师](Beast_Master.md "驯兽师")的[狼伙伴](Wolf_Companion.md "狼伙伴")能力召唤的[狼](Wolf.md "狼")，这是[游侠伙伴](Ranger's_Companion.md "游侠伙伴")的变体。当天狼座达到5级时，天狼座“升级”为[天狼座](Lupus_Major.md "天狼座")，当天狼座达到11级时，成为[天狼座](Lupus_Optumus.md "天狼座")。

## 战斗

### 攻击与能力

[啃咬](Bite_(Wolf).md "啃咬 (狼)") ()

伤害: 4~10

2d4 + 2⁠[穿刺](Piercing.md "穿刺")

啃咬目标。

近战: 1.5 m (5 ft)

[飞扑啃咬](Lunging_Bite.md "飞扑啃咬") ()

伤害: 3~6

1d4 + 2⁠[穿刺](Piercing.md "穿刺")

扑向目标啃咬它，并可能将其击至[倒伏](Prone_(Condition).md "倒伏 (状态)")。

[力量](Strength.md "力量") [豁免检定](Saving_throws.md "豁免检定")
近战: 1.5 m (5 ft)

由于[猎物的气味](Prey's_Scent.md "猎物的气味")被动特性，驯兽师游侠可能希望攻击被[猎人印记](Hunter's_Mark.md "猎人印记")标记的目标以获得最佳伤害。

## 备注

- 天狼座以及其他基础动物伙伴的[熟练项加值](Proficiency_Bonus.md "熟练项加值")为0。这可能是一个错误。
