# 负重

**负重**是衡量角色能携带多少重量的指标。

- 接近此值时会对移动和其他能力造成惩罚；
| 效果 | 调整值 | 来源 | 持续时间 |
| --- | --- | --- | --- |
| [缩小](Reduced_(Condition).md "缩小 (状态)") | x0.75 | [收缩油](Oil_of_Diminution.md "收缩油") | 持续时间：2回合 |
| [塞山](Sethan.md "塞山") ([塞山：缩小](Sethan_colon__Reduce.md "塞山：缩小")) | 持续时间：5回合 |  |  |
| [缩小](Reduce.md "缩小") | 持续时间：10回合 |  |  |
| [巨化](Enlarged_(Condition).md "巨化 (状态)") | x1.25 | [巨化](Enlarge.md "巨化") | 持续时间：10回合 |
| [巨化（灰矮人）](Enlarge_(Duergar).md "巨化（灰矮人）") |  |  |  |
| [大男孩的磨牙玩具](Bigboy's_Chew_Toy.md "大男孩的磨牙玩具") ([谁是大块头？](Whossa_Large_Fellow_q_.md "谁是大块头？")) |  |  |  |
| [巨像灵药](Elixir_of_The_Colossus_(Condition).md "巨像灵药 (状态)") | [巨像灵药](Elixir_of_the_Colossus.md "巨像灵药") | 持续时间：直到[长休](Long_Rest.md "长休") |  |
| [人类通才](Human_Versatility.md "人类通才") | [人类](Human.md "人类") 种族被动特性。 | 永久 |  |
| [维普拉克的贪婪](Vaprak's_Greed.md "维普拉克的贪婪") | [巨人（野蛮人副职）](Giant_(barbarian_subclass).md "巨人（野蛮人副职）") 被动。 | 使用该副职时 |  |
| [维普拉克的贪婪](Vaprak's_Greed.md "维普拉克的贪婪") + [人类通才](Human_Versatility.md "人类通才") | x1.50 |  | 特例 |
| [牛之蛮力](Bull's_Strength_(Condition).md "牛之蛮力 (状态)") | x2.00 | [强化属性：牛之蛮力](Bull's_Strength.md "牛之蛮力") | 持续时间：直到[长休](Long_Rest.md "长休") |
| [力量布衣](The_Mighty_Cloth.md "力量布衣") | 装备时 |  |  |
| [野兽面貌：熊](Aspect_of_the_Beast_colon__Bear.md "野兽面貌：熊") | [荒蛮之心](Wildheart.md "荒蛮之心") 野兽面貌选择。 | 选择时 |  |
| [星界重力](Astral_Gravity_(Condition).md "星界重力 (状态)") | x3.00 | （情境性） | 在[星界](Astral_Plane.md "星界")时 |

  - 超过此值的80%将导致[超重](Encumbered_(Condition).md "超重 (状态)")。
  - 超过此值的93.33%（当然，是循环小数）将导致[严重超重](Heavily_Encumbered_(Condition).md "严重超重 (状态)")。
  - 超过此值的100%（未降低限制）通常不被允许，但有时可通过[容器](Containers.md "容器")实现。
- 初始计算为 (40+10×力量)公斤 或等价于 (80+20×力量)磅，并在力量变化时相应更新。
  - 此值随后可乘以下表中的**单个最高调整值**；多个效果不叠加，但[维普拉克的贪婪](Vaprak's_Greed.md "维普拉克的贪婪")和[人类通才](Human_Versatility.md "人类通才")的情况除外。
    - 例如，一个力量为27且拥有[牛之蛮力](Bull's_Strength_(Condition).md "牛之蛮力 (状态)")的角色将拥有 2.00×(40+(10×27))公斤=620公斤（或1240磅）的负重；此外，作为人类、变为[缩小](Reduced_(Condition).md "缩小 (状态)")或[巨化](Enlarged_(Condition).md "巨化 (状态)")、或获得[巨像灵药](Elixir_of_The_Colossus_(Condition).md "巨像灵药 (状态)")或[野兽面貌：熊](Aspect_of_the_Beast_colon__Bear.md "野兽面貌：熊")都不会进一步改变此值。
    - 因此，人类实际上免疫负重降低。
