# 魔网套装

_魔网_ 套装是一套通过 **奥术增强** 提升穿戴者法术相关掷骰的装备：获得对 [法术豁免 DC](Saving_throw.md#The_Difficulty_Class_of_Saving_Throws "豁免检定") 和 [法术](Spell.md "法术") [攻击掷骰](Attack_roll.md "攻击掷骰") 的加成。曾由法师 Tenser 和 Edith Cluckle 佩戴。

| 名称 | 装备栏位 | 效果 |

| 魔网兜帽 | [帽子](Headgear.md "帽子") ([非护甲](Headwear#Non-Armour.md#非护甲 "头部装备")) | [奥术增强](Arcane_Enchantment_(+2).md "奥术增强 (+2)") 你获得对 [法术豁免 DC](Spell_save_DC.md "法术豁免 DC") 和 [法术攻击](Spell_Attack.md "法术攻击") 掷骰的 +2 加成。 |
| 魔网长袍 | [护甲](Armour.md "护甲") ([服装](Clothing.md "服装")) | [护甲等级](Armour_Class.md "护甲等级") +2 [奥术增强](Arcane_Enchantment_(+1).md "奥术增强 (+1)") 你获得对 [法术豁免 DC](Spell_save_DC.md "法术豁免 DC") 和 [法术攻击](Spell_Attack.md "法术攻击") 掷骰的 +1 加成。 [魔网食客](Weave_Eater.md "魔网食客") 每当穿戴者成功对法术进行 [豁免检定](Saving_throw.md "豁免检定") 时，他们恢复 1d6⁠⁠[生命值](Healing.md "治疗")。
