# 逃离（状态组）

**SG_Fleeing** 是一个[状态组](Status_group.md "状态组")。此状态组中的状态会使受影响实体逃离施加状态者。

## 免疫

以下物品使玩家免疫此状态组中的所有状态：

- [权威服饰](Cloth_of_Authority.md "权威服饰")
- [沙洛佛克的角盔](Sarevok's_Horned_Helmet.md "沙洛佛克的角盔")

## SG_Fleeing 中的状态列表

| 状态 | 效果 |
| :--- | :--- |
| [恐惧](Fearful_(Condition).md "恐惧（状态）") | - 必须以最快可用方式尽可能远离施法者。<br>- 无法采取[动作](Action.md "动作")、[附赠动作](Bonus_action.md "附赠动作")或[反应](Reaction.md "反应")。 |
| [恐惧](Fearful_(Condition).md "恐惧（状态）") | - 在[属性检定](Ability_Check.md "属性检定")和[攻击掷骰](Attack_roll.md "攻击掷骰")上具有[劣势](Disadvantage.md "劣势")。<br>- 必须逃离恐惧来源，且无法采取任何额外动作。<br>- 如果目标在其恐惧来源的视线之外结束回合，可进行[豁免检定](Saving_throw.md "豁免检定")以摆脱此效果。 |
| [失踪宠物](Missing_Pets_(Condition).md "失踪宠物（状态）") | - 无法移动。<br>- 受影响实体在[属性检定](Ability_Check.md "属性检定")和[攻击掷骰](Attack_roll.md "攻击掷骰")上具有[劣势](Disadvantage.md "劣势")。 |
| [被驱散](Turned_(Condition).md "被驱散（状态）") | - 被驱散的生物必须在其回合中尝试尽可能远离将其驱散的生物。<br>- 无法自愿移动到距离该生物 9 米（30 英尺）以内的空间。<br>- 也无法采取[反应](Reaction.md "反应")。<br>- 对于其[动作](Action.md "动作")，只能使用[疾走](Dash.md "疾走")动作或尝试逃离阻止其移动的效果。<br>- **受到伤害时状态结束。** |
