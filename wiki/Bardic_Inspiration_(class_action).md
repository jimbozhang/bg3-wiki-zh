# 诗人激励 (职业动作)

本文介绍基础吟游诗人职业动作。其他用法，请参见 [诗人激励 (消歧义)](Bardic_Inspiration_(disambiguation).md)。对于允许重掷失败技能检定的无关游戏机制，请参见 [激励点](Inspiration.md "激励点")。

**诗人激励**是 [吟游诗人](Bard.md "吟游诗人") 的被动特性和职业动作。此能力允许吟游诗人激励其盟友并影响其掷骰结果。在5级及以上时，此特性将被 **精通诗人激励** 替代。

## 描述

通过你的表演激励一名盟友超越其能力。他们可以在下一次 [攻击掷骰](Attack_roll.md "攻击掷骰")、[属性检定](Ability_Check.md "属性检定") 或 [豁免检定](Saving_throw.md "豁免检定") 中获得 +1d6 奖励。

## 属性

消耗
[附赠动作](Actions.md#Resources "动作") + [诗人激励](Bardic_Inspiration_(resource).md)
详情
范围：18米（60英尺）
目标：一名尚未拥有 [诗人激励](Bardic_Inspiration_(Condition).md "诗人激励 (状态)") 的盟友

## 更高等级

- 当吟游诗人达到5级时，奖励增加至 +1d8。
- 当吟游诗人达到10级时，奖励增加至 +1d10。

## 状态：诗人激励

**[诗人激励](Bardic_Inspiration_(Condition).md "诗人激励 (状态)")**

持续时间：直至 [长休](Long_Rest.md "长休")

- 可以使用诗人激励在下一次 [攻击掷骰](Attack_roll.md "攻击掷骰")、[属性检定](Ability_Check.md "属性检定") 或 [豁免检定](Saving_throw.md "豁免检定") 中获得 +1d6 奖励。
- 奖励取决于来源 [吟游诗人](Bard.md "吟游诗人") 的等级

## 学习方式

职业：

- 职业等级1：[吟游诗人](Bard.md "吟游诗人")

## 备注

- 在第二幕结束时帮助 [阿尔菲拉](Alfira.md "阿尔菲拉") 为提夫林孩子们演奏，允许使用独特的 **[精通诗人激励](Improved_Bardic_Inspiration_(class_action).md "精通诗人激励 (职业动作)")** 职业动作一次（每长休一次），仅消耗附赠动作。此功能与诗人激励类似，但使用 d12 并通过切换被动触发，而非反应。

## 错误

- 当用于以 +1d10 奖励修改攻击掷骰时，诗人激励状态不会从角色身上移除。
- 诗人激励可能以已受 +1d10 版本影响的角色为目标，浪费效果，因为它们不叠加。

## 相关物品

- [慈善外套](Blazer_of_Benevolence.md "慈善外套")：当你使用诗人激励激励盟友时，获得4点临时生命值。
- [响亮之靴](Boots_of_Brilliance.md "响亮之靴")：演奏你的乐器以恢复一个诗人激励使用次数（充能：长休）。
- [疗愈软帽](Cap_of_Curing.md "疗愈软帽")：当你使用诗人激励激励盟友时，他们还会恢复1d6生命值。
- [惊奇手套](Wondrous_Gloves.md "惊奇手套")：你的护甲等级增加1。此外，如果你拥有诗人激励，你会获得1次额外使用次数。
