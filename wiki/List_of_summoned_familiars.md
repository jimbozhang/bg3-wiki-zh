## 独特召唤魔宠

### 布布

主条目：[布布](Boo.md "布布")

| [力量](Strength.md "力量") 10 | [敏捷](Dexterity.md "敏捷") 20 (+5) | [体质](Constitution.md "体质") 10 | [智力](Intelligence.md "智力") 6 (-2) | [感知](Wisdom.md "感知") 12 (+1) | [魅力](Charisma.md "魅力") 6 (-2) |
| --- | --- | --- | --- | --- | --- |
|  |  |  |  |  |  |
| 生命值 | 20 |  |  |  |  |
| 护甲等级 | 20 |  |  |  |  |
| 体型 | 微型 |  |  |  |  |

召唤[明斯克](Minsc.md "明斯克")独特、可爱、偶尔暴力且永远忠诚的伙伴布布，它可以使用[啃咬](Bite_(Rat_Combatant).md "啃咬（老鼠战斗员）")。

如果被明斯克投掷，它有几率使目标[目盲](Blinded_(Condition).md "目盲（状态）")（引用明斯克的名言“布布，去啄他们的眼睛！”）。如果目标被击中，必须通过[DC](Dice_rolls.md#Save_DCs "掷骰") 15 [敏捷](Dexterity.md "敏捷") [豁免检定](Saving_throw.md "豁免检定")，否则将目盲1回合。

_仅[明斯克](Minsc.md "明斯克")可以召唤[布布](Boo.md "布布")，且无法与其他普通游侠魔宠同时召唤。_

### 挠挠

主条目：[挠挠](Scratch.md "挠挠")

| [力量](Strength.md "力量") 11 | [敏捷](Dexterity.md "敏捷") 13 (+1) | [体质](Constitution.md "体质") 12 (+1) | [智力](Intelligence.md "智力") 3 (-4) | [感知](Wisdom.md "感知") 13 (+1) | [魅力](Charisma.md "魅力") 6 (-2) |
| --- | --- | --- | --- | --- | --- |
|  |  |  |  |  |  |
| 生命值 | 11 |  |  |  |  |
| 护甲等级 | 11 |  |  |  |  |

召唤最棒的男孩。挠挠敏锐的鼻子可以发现世界上隐藏的许多东西，它可以使用[啃咬](Bite_(dog).md "啃咬（狗）")和[协助](Help.md "协助")。如果召唤的挠挠死亡，可以再次召唤，这不会影响[营地伙伴](Camp_Follower.md "营地伙伴")挠挠。

挠挠无法在营地召唤。

_任何拥有[挠挠的球](Scratch's_Ball.md "挠挠的球")的角色都可以召唤[挠挠](Scratch.md "挠挠")。如果召唤者返回营地，挠挠会自动解散。_
