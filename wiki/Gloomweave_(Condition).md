# 幽影编织 (状态)

| 状态 | 效果 |
| --- | --- |
| 幽影编织 | 在[攻击掷骰](Attack_roll.md "攻击掷骰")上具有[优势](Advantage.md "优势")，并对敌人的攻击掷骰施加[劣势](Disadvantage.md "劣势")。攻击或施放非[阴影潜行](Shadowslip.md "阴影潜行")的法术会结束此状态。 |
| [至上真神之手](Hand_of_the_Absolute_(Condition).md "至上真神之手 (状态)") | 为你的攻击增加 1d4⁠⁠[心灵](Psychic.md "心灵")伤害。每有一个敌人被你的灵能瘟疫半径捕获，你获得一个增益。 |
| [地狱束缚](Infernal_Binding_(Condition).md "地狱束缚 (状态)") | 希望在被囚禁于[希望之邸](House_of_Hope.md "希望之邸")时对所有伤害[免疫](Immune.md "免疫")。免疫[流放](Banished_(Condition).md "流放 (状态)")。 |
| [隐形](Invisible_(Misty_Escape)_(Condition).md "隐形 (雾遁) (状态)") | 在[攻击掷骰](Attack_roll.md "攻击掷骰")上具有[优势](Advantage.md "优势")，并对敌人的攻击掷骰施加[劣势](Disadvantage.md "劣势")。如果隐形实体攻击、施放另一个法术、执行动作或受到伤害，隐形状态会提前结束。 |
| [过热](Overheating_(Condition).md "过热 (状态)") | 这个过热的装置下次使用时会爆炸，对附近所有生物造成 8d8⁠⁠[火焰](Fire.md "火焰")伤害。想办法冷却它或远离它！ |
| [理性之声](Voice_of_Reason_(Condition).md "理性之声 (状态)") | 你的誓言在脑海中回响，暂时将[埃赛尔婶婶](Auntie_Ethel.md "埃赛尔婶婶")的低语挡在外面。 |

**幽影编织**

- 在[攻击掷骰](Attack_roll.md "攻击掷骰")上具有[优势](Advantage.md "优势")，并对敌人的攻击掷骰施加[劣势](Disadvantage.md "劣势")。攻击或施放非[阴影潜行](Shadowslip.md "阴影潜行")的法术会结束此状态。

## 赋予

[阴影潜行](Shadowslip.md "阴影潜行")
影灵传送到一个它能看到的未被占据的空间。（充能：每回合。）
