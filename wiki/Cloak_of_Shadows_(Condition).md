# 幽影斗篷 (状态)

| 状态 | 效果 |
| --- | --- |
| [引导神力：幽影斗篷](Channel_Divinity_colon__Cloak_of_Shadows_(Condition).md "引导神力：幽影斗篷 (状态)") | 在[攻击掷骰](Attack_roll.md "攻击掷骰")上具有[优势](Advantage.md "优势")，并对敌人的攻击掷骰施加[劣势](Disadvantage.md "劣势")。如果隐形实体攻击、施放另一个法术、执行动作或受到伤害，则提前结束隐形。 |
| 幽影斗篷 | 在[攻击掷骰](Attack_roll.md "攻击掷骰")上具有[优势](Advantage.md "优势")，并对敌人的攻击掷骰施加[劣势](Disadvantage.md "劣势")。攻击、施放法术或进入明亮光照区域会结束此状态。 |
| [妖火](Faerie_Fire_(Condition).md "妖火 (状态)") | 散发微光。受到的攻击具有[优势](Advantage.md "优势")。无法处于[隐形](Condition#Invisible.md#Invisible "状态")状态。 |
| [高等隐形术](Greater_Invisibility_(Condition).md "高等隐形术 (状态)") | 实体在[攻击掷骰](Attack_roll.md "攻击掷骰")上具有[优势](Advantage.md "优势")。针对它的攻击掷骰具有[劣势](Disadvantage.md "劣势")。当与物品互动、施放法术或攻击时，实体需要成功通过[隐匿](Stealth.md "隐匿")[检定](Ability_Check.md "属性检定")以维持隐形。每次成功尝试后，检定难度都会增加。 |
| [环境融合](Hide_in_Plain_Sight_(Condition).md "环境融合 (状态)") | 受影响的实体处于隐形状态。它在[攻击掷骰](Attack_roll.md "攻击掷骰")上具有[优势](Advantage.md "优势")。其隐匿检定获得+10加值。如果隐形实体移动、攻击、施放另一个法术、执行动作或受到伤害，则提前结束隐形。 |
| [隐形](Invisible_(Condition).md "隐形 (状态)") | 在[攻击掷骰](Attack_roll.md "攻击掷骰")上具有[优势](Advantage.md "优势")，并对敌人的攻击掷骰施加[劣势](Disadvantage.md "劣势")。如果隐形实体攻击、施放另一个法术、与物体互动、变得濡湿、执行动作或附赠动作，或受到伤害，则提前结束隐形。 |
| [与影同化](One_with_the_Shadows_(Condition).md "与影同化 (状态)") | 受影响的实体处于隐形状态。它在[攻击掷骰](Attack_roll.md "攻击掷骰")上具有[优势](Advantage.md "优势")，并对敌人的攻击掷骰施加[劣势](Disadvantage.md "劣势")。如果隐形实体移动、攻击、施放另一个法术、执行动作或受到伤害，则提前结束隐形。 |
| [暗影幔](Umbral_Shroud_(Condition).md "暗影幔 (状态)") | 在[攻击掷骰](Attack_roll.md "攻击掷骰")上具有[优势](Advantage.md "优势")，并对敌人的攻击掷骰施加[劣势](Disadvantage.md "劣势")。如果隐形实体攻击、施放另一个法术、执行动作或受到伤害，则提前结束隐形。 |

关于[诡术领域](Trickery_Domain.md "诡术领域")牧师职业动作[引导神力：幽影斗篷](Channel_Divinity_colon__Cloak_of_Shadows.md "引导神力：幽影斗篷")授予的类似效果，请参见[引导神力：幽影斗篷](Channel_Divinity_colon__Cloak_of_Shadows_(Condition).md "引导神力：幽影斗篷 (状态)")。

**幽影斗篷**

- 在[攻击掷骰](Attack_roll.md "攻击掷骰")上具有[优势](Advantage.md "优势")，并对敌人的攻击掷骰施加[劣势](Disadvantage.md "劣势")。
- 攻击、施放法术或进入明亮光照区域会结束此状态。

## 属性

[堆叠ID](Stack_ID.md "堆叠ID"): `INVISIBILITY` [状态组](Status_groups.md "状态组"): [SG_Invisible](SG_Invisible.md "SG Invisible"), [SG_RemoveOnRespec](SG_RemoveOnRespec.md "SG RemoveOnRespec")

[持续时间结束](Conditions.md#Duration "状态"): 回合结束

[堆叠优先级](Stack_priority.md "堆叠优先级"): 2

[更多属性](Status_properties.md "状态属性"):

- [UnavailableInActiveRoll](Status_properties/UnavailableInActiveRoll.md "状态属性/UnavailableInActiveRoll")
