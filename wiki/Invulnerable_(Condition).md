# 无敌（状态）

| 状态 | 效果 |
| --- | --- |
| [永恒誓缚](Eternally_Bound_(Condition).md "永恒誓缚（状态）") | 卡扎多尔不会让这个子嗣死亡 - 它对所有伤害[免疫](Immune.md "免疫")。 |
| [无敌](Invulnerable_(Globe_of_Invulnerability)_(Condition).md "无敌（法术无效结界）（状态）") | 受影响实体对所有伤害[免疫](Immune.md "免疫")。在所有[豁免检定](Saving_throw.md "豁免检定")上获得[优势](Advantage.md "优势")。 |
| 无敌 | 受影响实体对所有伤害[免疫](Immune.md "免疫")。 |
| [无敌](Invulnerable_(Ketheric_Thorm)_(Condition).md "无敌（凯瑟里克·索姆）（状态）") | 凯瑟里克在囚禁[暗夜之歌](Nightsong.md "暗夜之歌")期间对所有伤害[免疫](Immune.md "免疫")。 |
| [暗影假面](Shadow_Visage_(Condition).md "暗影假面（状态）") | 此假面漂浮在暗影虚空之中。在被拉入战斗前无法受到伤害。 |
| [银色壁垒](Silvered_Bulwark_(Condition).md "银色壁垒（状态）") | 受影响实体对所有伤害[免疫](Immune.md "免疫")。此守护由屏障内的所有生物和物体共享。 |

本文讨论的是标准状态。关于凯瑟里克·索姆的状态，请参见[无敌（凯瑟里克·索姆）（状态）](Invulnerable_(Ketheric_Thorm)_(Condition).md)。关于法术[法术无效结界](Globe_of_Invulnerability.md "法术无效结界")授予的状态，请参见[无敌（法术无效结界）（状态）](Invulnerable_(Globe_of_Invulnerability)_(Condition).md)。

**无敌**

- 受影响实体对所有伤害[免疫](Immune.md "免疫")。
