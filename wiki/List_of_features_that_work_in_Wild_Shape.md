# 在荒野形态中生效的特性列表

本页面介绍在荒野形态中生效或不生效的特性。关于荒野形态的属性和效果详情，请参见 [荒野形态](Wild_Shape.md "荒野形态") 或 [战斗荒野形态](Combat_Wild_Shape.md "战斗荒野形态")。

大多数装备、许多专长以及部分职业被动效果在 [荒野形态](Wild_Shape.md "荒野形态") 下不会生效。此列表阐明了哪些生效、哪些不生效。最初整理自 Reddit。<sup>[\[1\]](#cite_note-1)</sup>

[月照护甲](Armour_of_Moonbasking.md "月照护甲") - 少数在荒野形态中生效的物品之一

## 基本信息

- 在荒野形态中既无法施放法术，也无法采取任何超出荒野形态可用动作范围的行动。此类动作仅在其效果在变形后持续存在时才会在此列表中提及。
- 装备的物品会融入野兽形态，大多数物品在变回原形前不会产生任何效果。如果任何装备未在此处列出，则可能在荒野形态中不生效。
- 作为一般规则，荒野形态会用生物的属性替换角色的力量、敏捷和体质。此列表会注明永久增益或专长在荒野形态中实际改变这些属性值的例外情况。
- 由于编码方式，许多反应在荒野形态中不会触发。然而，在变形前使用 [伪装术](Disguise_Self.md "伪装术") 或 [伪装工具箱](Disguise_Kit.md "伪装工具箱") 可以使其中一些反应生效。
- 此列表并不意味着所有这些特性都*应该*与荒野形态一起工作——其中许多由于第五版 D&D 的规则本不应生效。
- 此列表远非完整。请添加注意到的任何缺失信息。

### 法术

- 在荒野形态中无法施放 [法术](Spells.md "法术")。
- 这包括消耗反应的法术（例如 [护盾术](Shield_(spell).md "护盾术") 或 [法术反制](Counterspell.md "法术反制")）或由动作触发的法术（例如 [至圣斩](Divine_Smite.md "至圣斩")），即使变形前使用了 [伪装术](Disguise_Self.md "伪装术")。
