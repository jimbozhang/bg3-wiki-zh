# 阿尔菲拉的信

**阿尔菲拉的信**是玩家角色在[终章](Epilogue.md "终章")中可以收到的信件之一。根据是否选择了[邪念](The_Dark_Urge.md "邪念")以及在未选择邪念的情况下[拉克里萨](Lakrissa.md "拉克里萨")是否存活，它有三个不同的版本。

一张朴素无华的便条。

## 文本

_如果拉克里萨在第三幕中存活_ ( UID `S_EPI_Letter_AlfiraAndLakrissa` / UUID `0c07c8d3-3b26-44ad-bd04-22d85e6b936a` )：

你好，朋友！

抱歉这么久没联系，但我有最棒的消息要告诉你。你知道拉克里萨在精灵之歌酒馆工作得有多疯狂吧？嗯，有一天早上她兴奋地把我叫醒，说要给我一个惊喜。她带我去了下城区的一栋房子，外面挂着一块牌子。

牌子上刻着你和莉哈拉的名字——下面写着“音乐学院”。

我当场就哭了出来。简直不敢相信。

这是你见过最可爱的房子，楼下是宽敞的开放空间，楼上有个小阁楼，我们可以睡觉。

消息传开后，我还没反应过来，房间里就挤满了从烛堡到深水城来的见习吟游诗人！拉克里萨不得不辞去精灵之歌酒馆的工作来帮忙。我日夜教学，每晚都筋疲力尽地倒在床上。

而我从未如此快乐过。

有空一定要来看看我们。我们很欢迎你！

爱你的，

阿尔菲拉

_如果拉克里萨已死_ ( UID `S_EPI_Letter_AlfiraAtElfsong` / UUID `4b3f1501-b966-4569-bdcd-4587c54243fd` )：

你好，朋友！

抱歉这么久没联系——事实证明，经营一所吟游诗人学校比我想象的要棘手一点（谁知道吟游诗人这么戏剧化？）。我在下城区买了一栋破旧的小房子，花了几个星期铺地板、换窗户。事实证明我用锤子和钉子还挺在行的！

当我（相对）确定屋顶不会塌后，我决定是时候开张了。我贴了海报，还在博德之口登了广告。天哪，我紧张极了——我老是做噩梦，担心没人会来。

但我的天哪，他们真的来了！从贵族子弟到街头小贩，孩子们排满了整条街。队伍几乎排到了利文顿！

事实证明，和博德之门的英雄做朋友确实有好处。

我日夜教学，每晚都筋疲力尽地倒在床上。

而我从未如此快乐过。

谢谢你，我的朋友。

爱你的，

阿尔菲拉。

_如果选择了邪念作为出身_ ( UID `S_EPI_Letter_AlfiraDUSaved` / UUID `33d08041-5581-4d26-8b76-92c13ca2b643` )：

你好，朋友！

我已经记不清这是第几次开始写这封信了，所以我直接说重点吧。

我听说了你是什么，以及你克服了什么。我从你看着我的眼神中看到了杀意——但你收手了，不是吗？

我不会告诉你我在哪里（原因显而易见），但我很安全。我很开心。我没有浪费你饶恕我的每一秒生命。

谢谢你。

怀着爱与宽恕，

阿尔菲拉
