# 阿玛尼塔·扎尔

| | |
| --- | --- |
| 种族 | [吸血鬼](Vampire.md "吸血鬼") 或 [吸血鬼衍体](Vampire_spawn.md "吸血鬼衍体") |

## 概述

她在1477年的笔记中记录了自己的历史。根据这些笔记，阿玛尼塔·扎尔在十三岁时受邀前往卡扎多尔的宫殿，首次被邀请进入舞厅。<sup>[\[1\]](#cite_note-1)</sup> 她后来被转化为[吸血鬼](Vampire.md "吸血鬼")，但拒绝参与她所谓的“家族仪式”。作为惩罚，她被囚禁在阁楼中。<sup>[\[2\]](#cite_note-2)</sup> 阿玛尼塔在受诅咒后拒绝使用阿玛尼塔·扎尔这个名字。她将自己改名为*隐秘女士*，并选择留在阁楼中。<sup>[\[3\]](#cite_note-3)</sup>
