# 界限突破手套

界限突破手套是一双稀有的[手套](Gloves.md "Gloves")，允许穿戴者以承受一定伤害为代价，在一回合内获得一次额外攻击并使移动速度翻倍。

'你趴下给我做三十个俯卧撑，你这混蛋；你以为熊地精会翻过身来让你揉它的肚子吗？一二三四——继续，蛆虫！更强，更好，更健壮！' —— 任何[宝剑海岸](https://forgottenrealms.fandom.com/wiki/Sword_Coast)的教官，无一例外

## 属性

- [手套](Gloves.md "Gloves")
- 稀有度：稀有 / 非常稀有（荣誉模式）
- 重量：0.5 kg (1 lb)
- 价格：160 gp / 580 gp（荣誉模式）
- UID `MAG_Martial_Exertion_Gloves` UUID `2c6da257-8ca8-4e7c-847d-3e03bb32f260` Stats `MAG_Martial_Exertion_Gloves` File `ARM_Gloves_Leather_F` ### 特殊

穿戴此物品获得：

[界限突破](Martial_Exertion.md "Martial_Exertion")
突破你的身体极限，承受6d6[穿刺](Piercing.md "Piercing")伤害，以获得每回合一次额外攻击，并使你的[移动速度](Movement_speed.md "Movement Speed")翻倍。（充能：[短休](Short_rest.md "Short rest")。）
