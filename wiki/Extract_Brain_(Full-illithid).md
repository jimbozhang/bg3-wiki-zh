# 提取大脑 (完整的灵吸怪)

本文介绍的是玩家在经历完整的蜕变后获得的能力版本。关于夺心魔NPC使用的版本，请参见 [提取大脑](Extract_Brain.md "提取大脑")。

**提取大脑** 是 [完整的灵吸怪](Full-illithid.md "完整的灵吸怪") 可用的动作和 [灵吸怪力量](Illithid_power.md "灵吸怪力量")。此能力让夺心魔尝试提取昏迷或震慑生物的大脑，造成巨额伤害并治疗自身。

## 描述

对附近处于 [震慑](Stunned_(status_group).md)、[倒伏](Prone_(status_group).md)、[沉睡](SG_Sleeping.md "SG Sleeping") 或 [昏迷](Unconscious_(status_group).md) 状态的生物进行脑叶切除。

## 属性

消耗
[动作](Actions.md#Resources "动作")
伤害：55~100 + 调整值

5d10 + 50 + [灵巧](Finesse.md "灵巧")⁠[穿刺](Piercing.md "穿刺")

治疗：6~36

6d6⁠[治疗](Healing.md "治疗")

详情
近战法术 [攻击掷骰](Attack_roll.md "攻击掷骰")
近战：1.5 米 (5 英尺)
目标：[震慑](Stunned_(status_group).md)、[倒伏](Prone_(status_group).md)、[沉睡](SG_Sleeping.md "SG Sleeping") 或 [昏迷](Unconscious_(status_group).md) 的生物
充能：每回合

## 学习方式

由生物使用：[完整的灵吸怪](Full-illithid.md "完整的灵吸怪")

## 备注

- 与 [提取大脑](Extract_Brain.md "提取大脑")（此动作的NPC版本）相比，此版本可以额外以 [倒伏](Prone_(Condition).md "倒伏 (状态)") 生物为目标。其伤害为 5d10 + 50⁠⁠[穿刺](Piercing.md "穿刺")，而非 10d10⁠⁠[穿刺](Piercing.md "穿刺")。
