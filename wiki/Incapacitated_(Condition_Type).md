# 失能 (状态组)

本页面介绍的是状态组。关于具体状态，请参见 [失能 (状态)](Incapacitated_(Condition)..md)

**失能**是一个[状态组](Status_group.md "状态组")。该状态组包含任何能够改变或阻止角色正常使用其动作的[状态](Condition.md "Condition")。

## 失能状态列表

- 从当前位面被放逐。
- 无法被选中。
- 无法移动或进行[动作](Action.md "Action")、[附赠动作](Bonus_action.md "Bonus Action")或[反应](Reaction.md "Reaction")

[膨胀](Bloated_(Condition).md "膨胀 (状态)")
|

- 因疾病和肿胀而被束缚。

[茧缚](Cocooned_(Condition).md "茧缚 (状态)")
|

- 被困在蜘蛛网中。无法移动或进行[动作](Action.md "Action")、[附赠动作](Bonus_action.md "Bonus Action")或[反应](Reaction.md "Reaction")。
- 被茧缚时，受影响实体对所有伤害具有[抗性](Resistance.md "Resistance")。
- 此外，其在进行[力量](Strength.md "Strength")[豁免检定](Saving_throw.md "Saving Throw")时具有[劣势](Disadvantage.md "Disadvantage")，并自动在[敏捷](Dexterity.md "Dexterity")[豁免检定](Saving_throw.md "Saving Throw")中失败。
- 该状态在受到伤害时结束，或实体成功通过[DC](Dice_rolls.md#Save_DCs "Dice rolls") 12的[力量](Strength.md "Strength")[豁免检定](Saving_throw.md "Saving Throw")时结束。

[命令：停步](Command_colon__Halt_(Condition).md "命令：停步 (状态)")
|

- 无法移动。
- 无法进行[动作](Action.md "Action")、[附赠动作](Bonus_action.md "Bonus Action")或[反应](Reaction.md "Reaction")。

[被诅咒：充满恐惧](Cursed_colon__Filled_with_Dread_(Condition).md "被诅咒：充满恐惧 (状态)")
|

- 屈服于诅咒赋予的恐惧，无法移动或进行动作、附赠动作或反应。

[倒地](Downed_(Condition).md "倒地 (状态)")
|

- 如果生命值降至0，你会陷入[昏迷](Unconscious.md "Unconscious")并必须进行[死亡豁免检定](Death_Saving_Throw.md "死亡豁免检定")。3次成功后，你停止流血并变得稳定。3次失败后，你[死亡](Dead_(Condition).md "死亡 (状态)")。
- 昏迷时受到的任何伤害均计为一次失败。
- 如果你恢复任何[生命值](Hit_Points.md "Hit Points")，该状态将被移除。如果盟友[协助](Help.md "Help")你，你将恢复1点生命值。

[假死](Feigning_Death_(Condition).md "假死 (状态)")
|

- 陷入足以模仿死亡的深度魔法昏迷
- 对所有伤害[抗性](Damage_types.md#Resistance "伤害类型")，除了[心灵](Psychic.md "Psychic")伤害，且疾病和中毒不再生效
- 被[协助](Help.md "Help")时移除

[冰冻](Frozen_(Condition).md "冰冻 (状态)")
|

- 受影响实体完全被冰包裹且[失能](Incapacitated.md "失能")。如果受到钝击、雷鸣或力场伤害，冰会碎裂，状态结束。
- 获得对[钝击](Bludgeoning.md "钝击")、[雷鸣](Thunder.md "雷鸣")和[力场](Force.md "力场")伤害的[易伤](Vulnerability.md "Vulnerability")。
- 获得对[火焰](Fire.md "Fire")伤害的[抗性](Resistance.md "Resistance")。
- 对[燃烧](Burning_(Condition).md "燃烧 (状态)")和[冷冻](Encrusted_with_Frost_(Condition).md "冷冻 (状态)")免疫。

[蛛丝墓穴](Gossamer_Tomb_(Condition).md "蛛丝墓穴 (状态)")
|

- 该生物被[感染](Infested_(Condition).md "感染 (状态)")的蛛丝包裹，无法移动或进行动作。除非用[强酸](Acid.md "强酸")或[火焰](Fire.md "Fire")伤害移除蛛丝，否则蛛丝会爆炸，对被包裹的生物造成8d10[中毒](Poison.md "中毒")伤害。

[被擒抱](Grappled_(Condition).md "被擒抱 (状态)")
|

- 受影响实体失能，且在擒抱期间无法被伤害。[协助](Help.md "Help")它以将其解放！

\_(状态)[被擒抱](Grappled_(Tentacle)_(Condition).md "被擒抱 (触手) (状态)")
|

- 每回合开始时，目标进行[力量](Strength.md "Strength")[豁免检定](Saving_throw.md "Saving Throw")以尝试挣脱。豁免失败时，受到2d6[心灵](Psychic.md "心灵")伤害。

[怪诞膨胀](Grotesque_Distention_(Condition).md "怪诞膨胀 (状态)")
|

- 膨胀到无可救药。轻微触碰即会爆炸。
- 触碰受此状态影响的实体将导致半径3米（10英尺）范围内爆发8d6[中毒](Poison.md "中毒")伤害，形成相同大小的[毒云](Poison_Cloud.md "毒云")，并彻底摧毁该实体。

[心形恐怖：蜕变](Heartform_Terror_colon__Ceremorphosis_(Condition).md "心形恐怖：蜕变 (状态)")
|

- 因转变为灵吸怪而充满恐惧，受影响实体无法移动或进行动作。

[怪物定身术](Hold_Monster_(Condition).md "怪物定身术 (状态)")
|

- 受影响实体无法[移动](Movement_speed.md "移动速度")或进行[动作](Action.md "Action")、[附赠动作](Bonus_action.md "Bonus Action")或[反应](Reaction.md "Reaction")。
- 对该实体的攻击具有[优势](Advantage.md "优势")。
- 如果攻击者在3米（10英尺）范围内，对实体的攻击总是[重击](Critical_Hit.md "重击")。
- 受影响实体自动在所有[力量](Strength.md "Strength")和[敏捷](Dexterity.md "Dexterity")[豁免检定](Saving_throw.md "Saving Throw")中失败。
- 每回合结束时，受影响实体进行[感知](Wisdom.md "Wisdom")[豁免检定](Saving_throw.md "Saving Throw")。豁免成功时，状态被移除。

[人类定身术](Hold_Person_(Condition).md "人类定身术 (状态)")
|

- 受影响实体无法[移动](Movement_speed.md "移动速度")或进行[动作](Action.md "Action")、[附赠动作](Bonus_action.md "Bonus Action")或[反应](Reaction.md "Reaction")。
- 对该实体的攻击自动成功通过[攻击掷骰](Attack_roll.md "攻击掷骰")。
- 如果攻击者在射程：3米（10英尺）内，对实体的攻击总是[重击](Critical_Hit.md "重击")。
- 受影响实体自动在所有[力量](Strength.md "Strength")和[敏捷](Dexterity.md "Dexterity")[豁免检定](Saving_throw.md "Saving Throw")中失败。
- 每回合结束时，受影响实体进行[感知](Wisdom.md "Wisdom")[豁免检定](Saving_throw.md "Saving Throw")。豁免成功时，状态被移除。

[霍慈的烈酒：石化](Hoots'_Hooch_colon__Stoned_(Condition).md "霍慈的烈酒：石化 (状态)")
|

- 暂时[石化](Petrified_(Condition).md "石化 (状态)") - 可能带有残留副作用。

[催眠凝视](Hypnotic_Gaze_(Condition).md "催眠凝视 (状态)")
|

- 无法移动或进行动作、附赠动作或反应。
- 施法者在对话中进行[属性检定](Ability_Check.md "属性检定")时具有[优势](Advantage.md "优势")。
- 受到伤害时移除。
- 持续时间可通过[维持催眠凝视](Maintain_Hypnotic_Gaze.md "维持催眠凝视")延长。

[被催眠](Hypnotised_(Condition).md "被催眠 (状态)")
|

- 无法[移动](Movement_speed.md "移动速度")或进行[动作](Action.md "Action")、[附赠动作](Bonus_action.md "Bonus Action")或[反应](Reaction.md "Reaction")。
- 受到伤害、被[推击](Shove.md "推击")或被[协助](Help.md "Help")时移除。

[失能](Incapacitated_(Condition).md "失能 (状态)")
|

- 无法进行[动作](Action.md "Action")或[反应](Reaction.md "Reaction")。
- 如果维持专注法术，会失去[专注](Concentration.md "专注")。

[地狱震慑](Infernal_Stun_(Condition).md "地狱震慑 (状态)")
|

- 该生物无法移动或行动，拉斐尔可随意[吸取灵魂](Soul_Drain.md "吸取灵魂")。

[玛门的掌控](Mammon's_Hold_(Condition).md "玛门的掌控 (状态)")
|

- 受影响实体暂时转变为黄金。无法移动或进行[动作](Action.md "Action")、[附赠动作](Bonus_action.md "Bonus Action")或[反应](Reaction.md "Reaction")。
- 对[中毒](Poison.md "中毒")和[疾病](Disease.md "疾病")免疫。
- 无法被[缴械](Weapon_Dropped!_(Condition).md "武器掉落！ (状态)")。

\_(状态)[麻痹](Paralysed_(Karabasan's_Gift)_(Condition).md "麻痹 (噩梦馈赠) (状态)")
|

- 生物处于[失能](Incapacitated_(Condition)状态.md)
- 对[挥砍](Slashing.md "挥砍")、[穿刺](Piercing.md "穿刺")和[钝击](Bludgeoning.md "钝击")伤害[易伤](Vulnerable.md "Vulnerable")。
- 自动在所有[力量](Strength.md "Strength")和[敏捷](Dexterity.md "Dexterity")[豁免检定](Saving_throw.md "Saving Throw")中失败
- 3米（10英尺）范围内的攻击者对该生物进行的任何攻击掷骰自动命中且为自动[重击](Critical_Hit.md "重击")

[麻痹](Paralysed_(Condition).md "麻痹 (状态)")
|

- 受影响实体无法移动或进行[动作](Actions.md#Resources "动作")、[附赠动作](Actions.md#Resources "动作")或[反应](Actions.md#Reactions "动作")。
- 如果攻击者在3米（10英尺）范围内，对实体的攻击总是[重击](Critical_Hit.md "重击")。
- 对实体的攻击掷骰具有[优势](Advantage.md "优势")。
- 受影响实体自动在所有[力量](Strength.md "Strength")和[敏捷](Dexterity.md "Dexterity")[豁免检定](Saving_throw.md "Saving Throw")中失败。

[石化](Petrified_(Condition).md "石化 (状态)")
|

- 变为石头。无法移动或进行[动作](Actions.md#Resources "动作")、[附赠动作](Actions.md#Resources "动作")或[反应](Actions.md#Reactions "动作")。

[仪式组件](Ritual_Component_(Condition).md "仪式组件 (状态)")
|

- 阿斯代伦作为亵渎仪式的一部分处于[失能](Incapacitated_(Condition)状态.md)，直到被[协助](Help.md "Help")。

[献祭羔羊](Sacrificial_Lamb_(Condition).md "献祭羔羊 (状态)")
|

- 实体被吸血鬼魔法包裹时，无法移动或进行动作、附赠动作或反应。
- 如果卡扎多尔三次从以此方式束缚的所有子嗣中汲取力量，他将升魔。
- 获得对所有伤害类型的[抗性](Resistance.md "Resistance")。
- 无法被[中毒](Poisoned_(status_group)..md)

[被吸取](Sapped_(Condition).md "被吸取 (状态)")
|

- 目标已被灵能吸取。

[灵魂囚笼](Soul_Caged_(Condition).md "灵魂囚笼 (状态)")
|

- 灵魂囚笼将[暗夜之歌](Aylin.md "暗夜之歌")束缚为凯瑟里克[免疫](Immunity.md "免疫")所有伤害的引擎。
- 暗夜之歌可被[协助](Help.md "Help")脱离灵魂囚笼。

[感染爬行者粘液](Stricken_with_Crawler_Mucus_(Condition).md "感染爬行者粘液 (状态)")
|

- 受影响实体[中毒](Poisoned_(Condition).md "中毒 (状态)")且[麻痹](Paralysed_(Condition).md "麻痹 (状态)")
- 持续到受影响实体成功通过[体质](Constitution.md "体质")[豁免检定](Saving_throw.md "Saving Throw")为止。

[转变为黄金](Turned_to_Gold_(Condition).md "转变为黄金 (状态)")
|

- 受影响生物转变为黄金。无法移动或进行动作、附赠动作或反应。
