找到账本。

将账本交给往日身影。

- 我们找到了账本。我们应该把它交给往日身影。

任务完成

- 我们将账本交给了往日身影，并惩罚了凶手的罪行。他很满意。
- 我们拒绝将账本交给往日身影，并不知道他的下落。

- 我们离开了暗影之地，可能永远无法知道往日身影的下落。

## 流程攻略

松动的木板下藏着账本。

一个 >)⁠[影灵](https://forgottenrealms.fandom.com/wiki/Shadar-kai_(elf))，被称为往日身影，可以在[终焉光芒旅店](Last_Light_Inn.md "终焉光芒旅店")以东的废弃战场找到，坐标 X: 126 Y: 105。他设置了一个仪式圈，正在对一具尸体使用[死者交谈](Speak_with_Dead.md "死者交谈")。往日身影最初让队伍离开，但他的白渡鸦暗示他们或许能提供帮助。他解释说，那个女人是[玛德琳](Madeline.md "玛德琳")，她是一名谋杀犯。她在[残月酒厂](The_Waning_Moon.md "残月酒厂")工作，并记录了所有顾客的日志，将他们的秘密出卖给[暗夜法官](Dark_Justiciar.md "暗夜法官")。他要求队伍在酒馆找到[玛德琳的账簿](Madeline's_Ledger.md "玛德琳的账簿")并带回来。

往日身影

残月酒厂位于[雷斯文小镇](Reithwin_Town.md "雷斯文小镇")，在收费站以东，通往博德之门传送点的道路以西。账本在吧台后面，坐标 X: -223 Y: -78 的一块松动木板下，需要通过[察觉技能](Perception.md "察觉技能")检定才能发现。

如果将账本交给往日身影，他会进行一个仪式，将玛德琳的灵魂引导到他自己的身体里，让队伍可以与她交谈。

玛德琳声称，暗夜法官内塔莎指示她报告一切，无论大小；她太害怕了不敢拒绝，这就是为什么她报告了她的朋友们醉酒时对凯瑟里克的抱怨。暗夜法官承诺她只会和那两个男人聊聊。然而，她给了他们刀，并告诉他们互相刺伤，直到她说停止——但她从未说过停止。玛德琳对此感到后悔，并对她的行为导致朋友们的死亡感到恐惧。

### 处理玛德琳

在了解她的故事后，有几种方法可以处理玛德琳：

- [邪念] 我曾结束过一个我珍视之人的生命。没有任何惩罚能为此赎罪。
- [游说] 任何事？那就像本和马克那样伤害自己——刺伤自己。（DC 14）
- [野蛮人] [游说] 你本可以反抗。你不配得到原谅，懦夫！（DC 10）
- [吟游诗人] [游说] 你的故事是懦弱的经典案例。如果你是我故事中的一个角色，我会立刻杀掉你。（DC 10）
- [吟游诗人] [游说] 当我讲述这个故事时，你不会是反派——我保证。该受责备的是暗夜法官。（DC 10）
- [武僧] [游说] 你本有办法阻止它——你*选择*不这样做。你让恐惧和懦弱毁了你。（DC 10）
- [武僧] [游说] 宽恕源于内心——我无法将其赠予你。但要知道，如果我能，我会的。（DC 10）
- [游说] 可悲——你谋杀了你的朋友。你是个懦夫。（DC 10）
- [游说] 你无需做任何事。这不是你的错——我保证。（DC 10）
- [善良牧师] [游说] 凭我所侍奉之光的名义，我原谅你。这不是你的所为。（DC 10）
- [邪恶牧师] [游说] 刺伤自己。撕开你的肚子。感受你朋友们的痛苦——只有这样你才能得到解脱。（DC 6）

如果让玛德琳刺伤自己一次，或者羞辱她为懦夫，那么往日身影会对结果感到满意，并给予队伍[渡鸦手套](Raven_Gloves.md "渡鸦手套")作为奖励。这会使拥有[侍僧](Acolyte.md "Acolyte")背景的队伍成员获得[邪恶不得安息](Acolyte.md#No_Rest_for_the_Wicked "侍僧")激励点。

如果原谅她，或者让她刺伤自己第二次，他会变得激怒并攻击队伍。每次刺伤造成5点伤害，这会延续到战斗中。拥有[侍僧](Acolyte.md "Acolyte")背景的队伍成员会获得[让沉睡的狗躺下](Acolyte.md#Let_Sleeping_Dogs_Lie "侍僧")激励点。

如果任何检定失败，连接会消失，虽然往日身影有些失望，但他仍然会奖励队伍的努力。

[邪念](The_Dark_Urage.md "邪念")如果之前杀死了自己最珍视的伙伴，与玛德琳有特殊互动。他们与她分享这一点，并说这样的行为无法赎罪，此时玛德琳会因内疚而崩溃消失，结果与羞辱她为懦夫相同，但不需要游说检定。往日身影也会评论邪念的内疚，但不会攻击。

[圣武士](Paladin.md "圣武士")在此任务中如果采取某些行动，会打破他们的誓言：

- [复仇之誓](Oath_of_Vengeance.md "复仇之誓")：原谅玛德琳，阻止她刺伤自己，或在仪式开始前攻击往日身影
- [奉献之誓](Oath_of_Devotion.md "奉献之誓") / [古贤之誓](Oath_of_the_Ancients.md "古贤之誓") / [誓言之冠](Oath_of_the_Crown.md "誓言之冠")：让玛德琳刺伤自己，或称她为懦夫

## 伙伴好感度

- 首次与往日身影对话时：

  - 告诉他“我怀疑你不是出于好心才来这里的。”+1 +1
- 同意帮助他，*不*说明你想得到报酬 +1 +1 +1 +1 +1 +1 +1

- 审判玛德琳时：

- 让她刺伤自己 -1 +1 +1 -1 -1 +1 -1 +1
  - 告诉她“*开始刺，直到我说停为止。*”或“*证明你后悔了。为本和马克这么做。*” -1 +2 +1 -1 -1 +1 -1 +1
- 让她停下 +1 -2 -1 +1 +1 -1 +1 -1
- 称她为懦夫 +1 -1 -1 +1 +1 -1 +1 -1
- 原谅她 +1 -1 -1 +1 +1 -1 +1 -1
