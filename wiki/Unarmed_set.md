# 徒手套装

徒手套装能增强徒手攻击。[久希戈套装](Kushigo_set.md "久希戈套装")和[灵魂套装](Soul_set.md "灵魂套装")属于此套装的一部分，但此处不重复列出。另请参阅[与徒手打击互动的特性与物品列表](List_of_features_and_items_that_interact_with_unarmed_strikes.md "与徒手打击互动的特性与物品列表")和[武僧套装](Monk_set.md "武僧套装")。

[天生拳师](Natural_Pugilist.md "天生拳师")
你获得徒手近战[攻击掷骰](Attack_roll.md "攻击掷骰")和伤害的+1加值。

[无阻抗性](Unhampered_Resistance.md "无阻抗性")
当装备者未穿着护甲时，他们获得[豁免检定](Saving_throw.md "豁免检定")的+2加值。
