# 状态属性/失控

另见 [状态属性](../Status_properties.md "状态属性")。

**LoseControl** 是一种 [状态属性](../Status_property.md "状态属性")。

## 拥有失控属性的状态列表

| --- | --- |
|

- 受影响实体无法控制其行动，会漫无目的地四处游荡。

[失心狂怒](../Berserk_(Condition).md "Berserk (Condition)")
|

- 狂乱的怒火折磨着受影响实体的肉体。其 [多重攻击](../Multiattack_(Flesh).md) 每次攻击额外造成 1d8⁠⁠[钝击](../Bludgeoning.md "Bludgeoning") 伤害，并使目标 [倒伏](../Prone_(Condition).md "Prone (Condition)")
- 它还获得对挥砍、穿刺和钝击伤害的 [抗性](../Resistance.md "Resistance")，并在力量 [检定](../Ability_Check.md "Ability Check") 和 [豁免检定](../Saving_throw.md "Saving Throw") 上具有 [优势](../Advantage.md "Advantage")。
- 如果受影响实体恢复全部生命值，此狂怒状态结束。

[冠军挑战](../Champion_Challenge_(Condition).md "Champion Challenge (Condition)")
|

- 受影响实体被迫攻击挑战它的圣武士。任何针对圣武士以外目标的攻击均具有 [劣势](../Disadvantage.md "Disadvantage")。
- 如果实体移动到距离圣武士 9 米（30 英尺）以外，状态结束。

[胁迫](../Coerced_(Condition).md "Coerced (Condition)")
|

- 被迫为尼讷而战，每回合受到 1d6⁠⁠[心灵](../Psychic.md "Psychic") 伤害。
- 在受到任何非心灵伤害后结束。

[命令术：靠近](../Command_colon__Approach_(Condition).md "Command: Approach (Condition)")
|

- 必须以最短路径朝施法者移动。
- 无法使用 [动作](../Action.md "Action")、[附赠动作](../Bonus_action.md "Bonus Action") 或 [反应](../Reaction.md "Reaction")。

[强令对决](../Compelled_Duel_(Condition).md "Compelled Duel (Condition)")
|

- 对除施法者外的任何目标进行 [攻击掷骰](../Attack_rolls.md "Attack Rolls") 时具有 [劣势](../Disadvantage.md "Disadvantage")。

[被迫对决](../Compelled_to_Duel_(Condition).md "Compelled to Duel (Condition)")
|

- 对除施法者外的任何目标进行 [攻击掷骰](../Attack_rolls.md "Attack Rolls") 时具有 [劣势](../Disadvantage.md "Disadvantage")。如果被决斗者击中，可能开始 [流血](../Bleeding_(Condition).md "Bleeding (Condition)")。

[困惑](../Confused_(Condition).md "Confused (Condition)")
|

- 受影响实体对所有人 [敌对](../Hostile.md "Hostile")，会漫无目的地游荡，并偶尔在恍惚中 [跳过对话](../Skip.md "Skip")。

[疯狂](../Crown_of_Madness_(Condition).md "Crown of Madness (Condition)")
|

- 被注入魔法疯狂。会攻击除施法者外最近的生物。

[恶魔之灵疯狂](../Demonspirit_Madness_(Condition).md "Demonspirit Madness (Condition)")
|

- 受影响实体被魔法疯狂驱使陷入狂乱。它会在其回合攻击最近的生物。
- 每回合结束时，受影响实体进行一次 [感知](../Wisdom.md "Wisdom") [豁免检定](../Saving_throw.md "Saving Throw")。如果豁免成功，则移除此状态。

[受控](../Dominated_(Condition).md "Dominated (Condition)")
|

- 受影响实体将跟随并协助施法者战斗。
- 每次受到伤害时，它可能通过成功的感知 [豁免检定](../Saving_throw.md "Saving Throw") 打破施法者的控制。

\_(Condition)[受控](../Dominated_(Dominate_Beast)_(Condition).md "Dominated (Dominate Beast) (Condition)")
|

- 受影响实体将跟随并协助施法者战斗。
- 每次受到伤害时，它可能通过成功的感知 [豁免检定](../Saving_throw.md "Saving Throw") 打破施法者的控制。

[啃咬群](../Gnawing_Horde_(Condition).md "Gnawing Horde (Condition)")
|

- 与你并肩作战的贪婪食尸鬼。

[挑衅咆哮](../Goaded_(Condition).md "Goaded (Condition)")
|

- 必须攻击挑衅咆哮的生物（如果可能）。
- 对除施法者外的目标进行攻击掷骰时具有 [劣势](../Disadvantage.md "Disadvantage")。

[鬼婆的仆从](../Hag's_Pawn_(Condition).md "Hag's Pawn (Condition)")
|

- 伊瑟尔已取得控制权。状态持续到面具被移除或施法者死亡。

[梦魔魅惑](../Incubus_Charmed_(Condition).md "Incubus Charmed (Condition)")
|

- 你被魔鬼魅惑，必须在战斗中支持它。在受到伤害后，你可以对此次效果进行另一次 [豁免检定](../Saving_throw.md "Saving Throw")。

[狂乱之吻](../Insanity's_Kiss_(Condition).md "Insanity's Kiss (Condition)")
|

- 在状态持续期间，对所有其他生物 [敌对](../Hostile.md "Hostile")。

[诱惑](../Lured_(Condition).md "Lured (Condition)")
|

- 受影响实体被鸟妖的诱捕旋律吸引，并使用其回合向鸟妖靠近，可能引发其他鸟妖的借机攻击。
- 受影响实体必须在回合开始时通过一次 [DC](../Dice_rolls.md#Save_DCs "Dice rolls") 13 [感知](../Wisdom.md "Wisdom") [豁免检定](../Saving_throw.md "Saving Throw")，否则保持被诱惑状态。
- 受到伤害或被 [推击](../Shove.md "Shove") 会移除此状态。

[疯狂](../Madness_(Condition).md "Madness (Condition)")
|

- 对附近所有人 [敌对](../Hostile.md "Hostile")。

[邪术附身](../Malefic_Possession_(Condition).md "Malefic Possession (Condition)")
|

- 受影响实体根据 [转化](../Possession.md "Possession") 幽魂的意志行动，每回合受到 4d8⁠⁠[黯蚀](../Necrotic.md "Necrotic") 伤害，直到死亡或被 [驱散](../Turned_(Condition).md "Turned (Condition)")。

[心智奴役](../Mind_Mastery_(Condition).md "Mind Mastery (Condition)")
|

- 被灰矮人心智奴役者强迫对一名盟友进行武器攻击。
- 攻击后或被击中后结束。

[新生僵尸](../Newborn_Zombie_(Condition).md "Newborn Zombie (Condition)")
|

- 由蠕行啃咬产生的僵尸无法持久。它们每回合受到 1⁠⁠[黯蚀](../Necrotic.md "Necrotic") 伤害，并且只跟随感染它们的僵尸。

[黑色之翼](../On_Black_Wings_(Condition).md "On Black Wings (Condition)")
|

- 一群乌鸦会自行行动。

[异界誓缚束缚](../Seized_by_Planar_Binding_(Condition).md "Seized by Planar Binding (Condition)")
|

- 受影响实体跟随并为束缚它的人而战。
- 每次受到伤害时，它可能通过成功的 [感知](../Wisdom.md "Wisdom") [豁免检定](../Saving_throw.md "Saving Throw") 打破施法者的控制。

[暗影附身](../Shadow_Possession_(Condition).md "Shadow Possession (Condition)")
|

- 受影响实体被狂乱暗影附身，会攻击最近的生物。其武器攻击额外造成 1d4⁠⁠[黯蚀](../Necrotic.md "Necrotic") 伤害。

[幽影诅咒不死生物](../Shadow-Cursed_Undead_(Condition).md "Shadow-Cursed Undead (Condition)")
|

- 死于幽影中的人被诅咒永远留在那里。
- 这些不死生物的 [生命值](../Hit_Points.md "Hit Points") 和 [力量](../Strength.md "Strength") 降低，对 ⁠[黯蚀](../Necrotic.md "Necrotic") 伤害 [抗性](../Damage_types.md "Damage Types")，且无法使用 [法术](../Spell.md "Spell") 或 [武器动作](../Weapon_actions.md "Weapon Actions")。

[无尽怒火](../Unbridled_Wrath_(Condition).md "Unbridled Wrath (Condition)")
|

- 在地狱怒火的烈焰中释放十年积压的愤怒。
- 受影响实体对 [火焰](../Fire.md "Fire") 伤害 [免疫](../Immunity.md "Immunity")。
