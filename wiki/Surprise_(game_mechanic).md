# 惊讶（游戏机制）

[惊讶](Surprised_(Condition).md "惊讶（状态）")是一种特殊的[状态](Conditions.md "状态")，可在战斗的第一回合影响[生物](List_of_creature_types.md "生物类型列表")。

## 概述

当战斗遭遇中的一方未察觉另一方时，他们可能会被_惊讶_。被_惊讶_的生物在一回合内无法采取任何[动作](Action.md "动作")或[反应](Reaction.md "反应")。拥有[警觉](Alert.md "警觉")专长、[野性直觉](Feral_Instinct.md "野性直觉")特性或特定物品（如[警觉灵药](Elixir_of_Vigilance.md "警觉灵药")）的角色不会被_惊讶_。

某些战斗遭遇，尤其是发生在游戏过场动画之后的遭遇，总是会令队伍惊讶。通常可以通过利用上述特性或状态（如[隐形](Invisible_(Condition).md "隐形（状态）")）或在触发遭遇前使用[黑暗术](Darkness.md "黑暗术")等法术隐藏在重度遮蔽区域来应对。

## 如何使敌人惊讶

使敌人惊讶的关键是视线：他们必须对惊讶他们的角色有清晰的视线。如果没有，他们会开始搜寻该角色，一旦找到并开始战斗，他们就不会被_惊讶_。

这使得成功惊讶一群敌人变得具有挑战性且反直觉。以下预防措施可能有助于此过程：

1. 使用[躲藏](Hide.md "躲藏")使_整个_队伍进入隐匿状态，包括宠物。
2. 解散队伍（包括宠物），然后选择一名队员作为伏击者，将其移动到触发惊讶的位置。
3. 当伏击者从敌人的视野锥外接近敌人时，让其他队员和宠物保持一定距离。如果他们在[清晰区域](Clear_Area_(Condition).md "清晰区域（状态）")中穿过视野锥，他们会立即被发现；但如果他们处于[轻度遮蔽](Lightly_Obscured_(Condition).md "轻度遮蔽（状态）")或[重度遮蔽](Heavily_Obscured_(Condition).md "重度遮蔽（状态）")状态，则必须成功通过[隐匿](Stealth.md "隐匿")掷骰对抗敌人的[察觉技能](Perception.md "察觉技能")检定。如果掷骰成功，敌人不会注意到该角色；如果失败，敌人会在未被惊讶的情况下发起战斗。
4. 伏击者_必须_在敌人的视线范围内发起战斗。如果做到这一点，敌人应该会被惊讶。然而，如果他们不在视线范围内，敌人会开始搜寻该队员，一旦找到并开始战斗，他们就不会被惊讶。
5. 战斗开始的方式很重要。以下是一些关键点（但测试尚未详尽）：
   1. 近战攻击总是有效。
   2. 在近战范围内进行的远程攻击总是有效。
   3. 法术攻击规则要么有漏洞，要么很复杂。法术攻击在近战范围内几乎总是有效，在非近战范围内效果稍差。以下示例展示了差异：
      1. [魔能爆](Eldritch_Blast.md "魔能爆")（攻击掷骰）在近战和非近战范围内都有效。
      2. [火球术](Fireball.md "火球术")（敏捷豁免）在近战范围内有效，但在非近战范围内无效。
      3. [恐惧](Fear.md "恐惧")（感知豁免）在近战和非近战范围内都有效。
      4. [魔法飞弹](Magic_Missile.md "魔法飞弹")（无攻击掷骰）在近战和非近战范围内均无效。
      5. [巫术箭](Witch_Bolt.md "巫术箭")（攻击掷骰）在近战范围内有效，但在非近战范围内无效。
   4. 如果攻击类型（近战、远程、法术）有效，则是否命中无关紧要。
   5. [手雷](Grenades.md "手雷")无效，即使直接击中敌人。
   6. 对于[伏击](Ambushing_(Condition).md "伏击（状态）")中的敌人，所有类型的攻击（包括手雷）都有效，只要敌人尚未发现队伍。即使[察觉技能](Perception.md "察觉技能")检定失败并攻击敌人当前所在的空位置，这一点也成立。
   7. 一旦敌人被惊讶，其余队员可以移动到位置加入战斗。当他们进入敌人的视野锥或攻击敌人时，会被拉入战斗。

如果角色的隐匿值足够高，敌人有时即使其盟友在远处被攻击和杀死，也无法注意到他们。这甚至适用于[火球术](Fireball.md "火球术")等范围效果法术，只要施法的队员保持隐藏。敌人会四处奔跑，拼命搜寻该队员但无法找到。这可以通过高[敏捷](Dexterity.md "敏捷")值、隐匿的[熟练项](Proficiency.md "熟练项")和[专精](Expertise.md "专精")、敏捷或隐匿检定的[优势](Advantage.md "优势")，以及某些法术和能力（如[行动无踪](Pass_Without_Trace.md "行动无踪")）来实现。

### 从隐形状态触发惊讶

使敌人惊讶的最可靠方法是进入隐形状态并攻击他们。这种隐形可以来自任何来源（例如[隐形药水](Potion_of_Invisibility.md "隐形药水")、[隐形术](Invisibility_(spell).md "隐形术（法术）")、[高等隐形术](Greater_Invisibility.md "高等隐形术")或[魔索布莱城之影](Shadow_of_Menzoberranzan.md "魔索布莱城之影")）。[夸塞魔](Quasit.md "夸塞魔")和[小魔鬼](Imp.md "小魔鬼")宠物也有效，[荒野形态：黑豹](Wild_Shape_colon__Panther.md "荒野形态：黑豹")同样有效。

## 惊讶遭遇列表

### 第一幕

- 潜入[蔓生废墟餐厅](Overgrown_Ruins.md#Refectory "蔓生废墟")会使[安东](Andorn.md "安东")惊讶。
- 接近[纳迪拉](Nadira.md "纳迪拉")会使试图伏击她的[熊地精刺客](Bugbear_Assassin.md "熊地精刺客")惊讶。
- 在[药剂师地窖](Apothecary's_Cellar.md "药剂师地窖")内的洞穴中与某些棺材互动会导致[机敏的守卫者](Agile_Guardian.md "机敏的守卫者")冲出并使队伍惊讶。
- 攻击[复生之路](Risen_Road.md "复生之路")沿线肿胀的[鬣狗](Hyena.md "鬣狗")会使它们惊讶。
- 在[幽暗地域](Underdark.md "幽暗地域")中，接近[石化战场](Underdark.md#Petrified_Battlefield "幽暗地域")会与[观察者眼魔](Spectator.md "观察者眼魔")发起战斗，触发过场动画并使队伍成员惊讶。
- 接近[幽暗地域海滩](Underdark.md#Beach "幽暗地域海滩")会触发与灰矮人的对话。如果立即攻击，海滩上的所有灰矮人都会被惊讶。如果[格克·煤球](Gekh_Coal.md "格克·煤球")被召唤并变得敌对，队伍成员反而会被惊讶。
- 在[复仇之炉](Grymforge.md "复仇之炉")中，攻击三个[拟形怪](Mimic.md "拟形怪")在伪装形态下互动时总是会使队伍成员惊讶。从远程攻击它们会在未被惊讶的情况下开始战斗。
- [岩浆魔蝠](Magma_Mephit.md "岩浆魔蝠")在接近最靠近[精金熔炉](Adamantine_Forge_(location)的秘银矿脉时会使队伍成员惊讶.md)。

### 第二幕

- 在[幽影诅咒之地](Shadow-Cursed_Lands.md "幽影诅咒之地")中，与[枯萎术](Blight_(race)的三次单独遭遇如果队伍靠得太近.md)，可能会使队伍成员惊讶。
- 在[月出之塔](Moonrise_Towers.md "月出之塔")中，攻击[拟形怪](Mimic.md "拟形怪")在伪装形态下互动时总是会使队伍成员惊讶。从远程攻击它们会在未被惊讶的情况下开始战斗。
- 在X: -43 Y: 7的[废弃战场](Ruined_Battlefield.md "废弃战场")中遇到的[鬾魊](Meazel.md "鬾魊")如果未通过DC 18的察觉检定，会使队伍惊讶。

### 第三幕

- 在[马戏团](Circus_of_the_Last_Days.md "马戏团")的“[德里布斯](Dribbles.md "德里布斯")”表演结束时，除非通过被动[察觉技能](Perception.md "察觉技能")检定注意到伏击，否则队伍成员会被惊讶；如果注意到伏击，伏击者反而会被惊讶。
- 在[丹瑟隆的飞斧](Danthelon's_Dancing_Axe.md "丹瑟隆的飞斧")的[恩萨尔的地下室](Entharl's_basement)中，为[高阶竖琴手](The_High_Harper.md "高阶竖琴手")任务，可以与一群伪装成竖琴手的[变形怪](Doppelganger.md "变形怪")交谈。对话检定和选择的组合可能导致队伍成员或伏击者被惊讶。
