## 闲聊

在 [下城区下水道](Lower_City_Sewers.md "下城区下水道") 石桥上的两只魔蝠正在聊天。

- 油脂魔蝠
  我老是忘。我们该干嘛来着？

- 另一只油脂魔蝠
  法师叫我们盯着点捣乱分子。

- 油脂魔蝠
  但我们自己就是捣乱分子啊。

- 另一只油脂魔蝠
  闭嘴，把你那脏兮兮的小眼珠子睁大点。

- 油脂魔蝠
  下面那条河叫什么名字？

- 另一只油脂魔蝠
  法师管它叫“苏-呃”。

- 油脂魔蝠
  不管叫啥，真壮观。而且味道好极了。

- 油脂魔蝠
  你早先去游泳了吗？

- 另一只油脂魔蝠
  没。我不会游泳。

- 油脂魔蝠
  哼。肯定是别的什么流着口水、臭烘烘的家伙顺流漂下去了。

再往西还能找到另外两只魔蝠。

- 油脂魔蝠
  哦，我滑溜溜的屁股。我刚意识到我们在哪儿。

- 另一只油脂魔蝠
  哪儿？

- 油脂魔蝠
  [托瑞尔](Toril.md "托瑞尔"). _博德之门._

- 另一只油脂魔蝠
  呃，不！真恶心！

- 油脂魔蝠
  法师又漏了。

- 另一只油脂魔蝠
  不是漏。是_在哭_。

- 油脂魔蝠
  哭什么？

- 另一只油脂魔蝠
  他们难过的时候就会这样……？

- 油脂魔蝠
  什么是难过？

- 另一只油脂魔蝠
  哦，闭嘴让法师漏去吧。

- 油脂魔蝠
  法师啥时候才_干_点事？上个主人让我往井里下毒。

- 另一只油脂魔蝠
  就这样了。它说我们在守夜。为它的朋友。

- 油脂魔蝠
  上面那堆骨头？那是它朋友？所以……我_不该_啃两口？

- 另一只油脂魔蝠
  呃。不，没事，我想？朋友之间啃两口很正常。
