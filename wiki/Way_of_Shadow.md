# 暗影宗

**暗影宗**是[武僧](Monk.md "武僧")的[副职](Monk.md#Level_3 "武僧")之一。可在3级时选择。这些武僧更擅长利用阴影悄无声息地打击敌人。

> “
>
> 你重视隐匿与诡计的艺术，驾驭阴影为己所用，出其不意地发动攻击。
>
> „

## 副职特性

武僧默认护甲 _此副职除了下述独特特性外，还获得其基础职业[武僧](Monk.md "武僧")的所有特性。_

### 等级 3

[初级幻影](Minor_Illusion.md "初级幻影") ()
创造一个幻象，迫使附近生物进行调查。

[暗影技艺：躲藏](Shadow_Arts_colon__Hide.md "暗影技艺：躲藏") ()
通过成功进行[隐匿](Stealth.md "隐匿")[检定](Ability_checks.md "检定")来躲避敌人。紧贴黑暗并避开敌人的视线。攻击或施放法术会暴露你的位置。

[暗影技艺：行动无踪](Shadow_Arts_colon__Pass_Without_Trace.md "暗影技艺：行动无踪") (+ 2 )
召唤一道阴影与寂静的帷幕，为你和附近盟友提供+10的[隐匿](Stealth.md "隐匿")[属性检定](Ability_Check.md "属性检定")加值。

[暗影技艺：黑暗术](Shadow_Arts_colon__Darkness.md "暗影技艺：黑暗术") (+ 2 )
创造一团魔法黑暗，对范围内的生物造成[重度遮蔽](Heavily_Obscured_(Condition).md "重度遮蔽（状态）")和[目盲](Blinded_(Condition).md "目盲（状态）")效果。生物无法向内或向外进行远程攻击。

[暗影技艺：高级黑暗视觉](Shadow_Arts_colon__Darkvision.md "暗影技艺：高级黑暗视觉") (+ 2 )
赋予生物在黑暗中视物的能力，范围为12米/40英尺。

[暗影技艺：沉默术](Shadow_Arts_colon__Silence.md "暗影技艺：沉默术") (+ 2 )
创造一个隔音球体。所有内部生物均[沉默](Silenced_(Condition).md "沉默（状态）")并[免疫](Damage_types.md#Immunity "伤害类型")[雷鸣](Thunder.md "雷鸣")伤害。

### 等级 5

[幽影斗篷](Cloak_of_Shadows.md "幽影斗篷") ()
用阴影包裹自己，若处于[遮蔽](Obscured.md "遮蔽")状态，则变为[隐形](Cloak_of_Shadows_(Condition).md "幽影斗篷（状态）")。

### 等级 6

[暗影步](Shadow_Step.md "暗影步") ()
从一处[轻度遮蔽](Lightly_Obscured.md "轻度遮蔽")传送至另一处阴影。之后，你的下一次近战[攻击掷骰](Attack_roll.md "攻击掷骰")具有[优势](Advantage.md "优势")。

### 等级 11

[阴影突袭](Shadow_Strike.md "阴影突袭") (+ 3 )
从隐藏位置传送至敌人身边，以阴影本身潜行的心灵凶猛进行打击。

[阴影突袭：徒手](Shadow_Strike_colon__Unarmed.md "阴影突袭：徒手") (+ 3 )
从隐藏位置传送至敌人身边，以阴影本身潜行的心灵凶猛进行打击。
