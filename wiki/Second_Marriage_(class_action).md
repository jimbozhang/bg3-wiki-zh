# 续缘魔杖 (职业动作)

**续缘魔杖**是一种通过使用[续缘魔杖](Second_Marriage.md "续缘魔杖")来召唤[康纳·维德巴德](Connor_Vinderblad.md "康纳·维德巴德")作为可控僵尸的动作。

## 描述

召唤[梅丽娜](Mayrina.md "梅丽娜")的[康纳·维德巴德](Connor_Vinderblad.md "康纳·维德巴德")的遗骸。
