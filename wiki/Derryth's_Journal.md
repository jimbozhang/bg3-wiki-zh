# 德里丝的日志

**德里丝的日志**是常见的[书籍](Books.md "书籍")。它记录了[德里丝·骨篷](Derryth_Bonecloak.md "德里丝·骨篷")对其丈夫[巴伦](Baelen_Bonecloak.md "巴伦")的看法。

这本书散发着纸张和墨水的诱人气息。

## 文本

_如果巴伦未被给予[箴言菇](Noblestalk.md "箴言菇")_ ( UID `UNI_BOOK_LOW_BonecloakApothecary_Journal_BaelenUnhealed` / UUID `4ef75ca7-1f32-4cc7-a490-089aded03514` )：

⁠[杜马索因](https://forgottenrealms.fandom.com/wiki/Dumathoin)带走这个男人吧！我以为当我们回到[城市](Baldur's_Gate.md "博德之门")并再次置身于熟悉的环境中时，他会变得温和一些，重新集中注意力。

但没有。他还是像以前一样糊涂，这只老狼蛛，甚至比在[幽暗地域](Underdark.md "幽暗地域")时更易怒。他需要我的帮助，而他讨厌这样——他一直如此。我不会再忍受了。如果他再打我，第二天早上他的粥里就会发现[翼龙毒素](Wyvern_Toxin.md "翼龙毒素")！

[座狼](Worg.md "座狼")屎，我真的写了这些话吗？我最好把它藏起来。

_如果巴伦被给予了[箴言菇](Noblestalk.md "箴言菇")_ ( UID `UNI_BOOK_LOW_BonecloakApothecary_Journal_BaelenHealed` / UUID `11cf7f6b-fc91-4993-88d5-f68b404527a0` )：

我知道我不是圣人，从来都不是，虽然我总是言辞犀利，但如果你问我，我会告诉你我骨子里没有恶意。无论如何，不是真的恶意。

但看看我变成了多么可悲的一坨屎。我猜嫁给巴伦这么久，留下的印记比我想象的要深。即使巴伦在他愤怒发作时，也从未让我伤得比淤青更重，但现在我感觉心碎了。

而为什么，看在⁠[杜马索因](https://forgottenrealms.fandom.com/wiki/Dumathoin)的份上？巴伦已经痊愈了！他在幽暗地域恢复了神智，又变回了原来的自己。

而我猜这就是问题所在。当他糊涂时，我可以把我们的问题归咎于他的昏沉以及这让他（和我）多么愤怒。

现在他又变回了原来的自己。而原来的他不是我想待在一起的人。

_如果巴伦在[第一幕](Act_One.md "第一幕")中死亡_ ( UID `UNI_BOOK_LOW_BonecloakApothecary_Journal_BaelenDead` / UUID `cee1fe69-bf00-4ddd-b3c2-642831172e9d` )：

巴伦死了，这只老恐爪怪。他死了，而我想念他，我这个傻瓜。他是个脾气暴躁的老座狼崽子，一旦开始失去理智就变得更暴躁，他讨厌需要我的帮助，而我讨厌不得不帮助他，因为他心怀怨恨而非感激。

我以为他走了我不会想念他，我自己也这么告诉过他，但现在我的生活中有一个巨大的空洞，猫是填不满的。该死的傻瓜，那样把自己害死了。

现在我该怎么办？我忍不住想尝尝自己的货来改善心情，但巴伦就是这样开始滑向糊涂的，所以我猜我不会。

我想知道[格鲁姆波](To_Glumbor_from_Derryth.md "格鲁姆波给德里丝的信")还活着吗？当我还是个年轻貌美的人时，他比我大二十岁似乎很重要，但对一个矮人来说，二十年又算什么呢？

不过以我的运气，他大概和巴伦一样死了。
