# 棕色的裤子

**棕色的裤子** 是一件常见的 [营地服装](Camp_Clothing.md#Camp_Clothing "营地服装")。

一套非常适合在篝火旁放松闲坐的服装，人们通常会在此交换故事，一般还会配上小酒壶。

## 属性

- [营地服装](Camp_Clothing.md "营地服装")

- 稀有度：普通

- 重量：1 千克 (2 磅)

- 价格：8 金币

- UID `ARM_Camp_Pants_A` UUID `a5cc728e-6b8a-4458-86cd-7492f7c4a266` Stats `ARM_Vanity_Body_Pants` File `CLT_Pants_A` ## 获取地点

- 在游戏后期作为场景布置和随机战利品出现在多个地点

## 备注

- 与 [莱埃泽尔](Lae'zel.md "莱埃泽尔") 的 [衣物](Lae'zel's_Clothes.md "莱埃泽尔的衣物") 类似，棕色的裤子只覆盖下半身。
- 穿着这些衣物时，某些 [内衣](Underwear.md "内衣") 可能会出现在上半身。
