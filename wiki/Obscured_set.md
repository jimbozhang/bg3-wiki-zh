# 遮蔽套装

**遮蔽**套装是一套主要受益于穿戴者或其目标处于轻度或重度[遮蔽](Obscured.md "遮蔽")状态的装备。还包括提供魔法高级黑暗视觉和目盲免疫的物品，因为它们增强了在遮蔽区域内的行动能力。它与[暗夜法官套装](Dark_Justiciar_Set.md "暗夜法官套装")有部分重叠，但该列表中的物品不会在此重复。
