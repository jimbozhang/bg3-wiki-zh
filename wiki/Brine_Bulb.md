# 卤灯

| | |
| --- | --- |
| 生命值 | 24 |

## 抗性 [(info)](Resistances.md "Resistances")

- - - - - **卤灯**是一种光源，可在[鹦鹉螺](Nautiloid.md "鹦鹉螺")上找到，并散落在[鹦鹉螺残骸](Nautiloid_Wreck.md "鹦鹉螺残骸")各处。被摧毁时，会形成一片[腐蚀性卤水](Caustic_Brine.md "Caustic Brine")地表。部分灯泡在被击中时会掉落到地面。
