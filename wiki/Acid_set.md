# 强酸套装

强酸套装是一套帮助使用者造成 [强酸](Acid.md "强酸") 伤害的装备。它与 [元素生物套装](Elemental_set.md "元素生物套装") 中的物品有协同效果，但此处不重复列出。该套装包含以下部件：

| 名称 | 装备栏位 | 效果 |
| --- | --- | --- |
| 腐蚀指环 | [戒指](Ring.md "戒指") | [邪恶分泌](Malefic_Excretion.md "邪恶分泌") 你的武器攻击额外造成 2⁠⁠[强酸](Acid.md "强酸") 伤害。 |
| 腐蚀链枷 | [主手近战](Main_Hand_Melee.md "主手近战") ([链枷](Flails.md "链枷")) | [腐蚀打击](Corrosive_Strike.md "腐蚀打击") () 造成额外的 ⁠[强酸](Acid.md "强酸") 伤害，伤害值等于你的 [熟练项加值](Proficiency_Bonus.md "熟练项加值")。命中时，在目标周围生成一滩酸液，使其 [护甲等级](Armour_Class.md "护甲等级") 降低 2。 ( 重充：[短休](Short_rest.md "短休")。) |
| 血肉消熔披风 | [披风](Cloak.md "披风") | [腐蚀性掠夺](Caustic_Reprisal.md "腐蚀性掠夺") 当有生物对穿戴者造成近战伤害时，该生物受到 1d4⁠⁠[强酸](Acid.md "强酸") 伤害。 |
| 腐液手套 | [手套](Gloves.md "手套") ([非护甲](Handwear#Non-Armour.md#Non-Armour "手部装备")) | [腐液侵蚀](Ichorous_Corrosion.md "腐液侵蚀") 当穿戴者造成 ⁠[强酸](Acid.md "强酸") 伤害时，他们也会对目标施加 [有毒烟雾](Noxious_Fumes_(Condition).md "有毒烟雾 (状态)")。 |
| 马友夫初始法杖 | [主手近战](Main_Hand_Melee.md "主手近战") ([长棍](Quarterstaves.md "长棍")) | [奥术增强](Arcane_Enchantment_(+1).md "奥术增强 (+1)") 你的 [法术豁免 DC](Spell_save_DC.md "法术豁免 DC") 和 [法术攻击](Spell_Attack.md "法术攻击") 掷骰获得 +1 加值。 [马友夫强酸箭](Melf's_Acid_Arrow.md "马友夫强酸箭") () 以 2 级法术施展 ( 重充：[长休](Long_Rest.md "长休")。) |
| 养育官寇库的靴子 | [鞋类](Footwear.md "鞋类") ([非护甲](Footwear#Non-Armour.md#Non-Armour "鞋类")) | [养育官的关爱](Caretaker's_Care.md "养育官的关爱") [酸性](Acid_(surface).md "酸性 (地表)") 地表不会影响你，并且你对 ⁠[强酸](Acid.md "强酸") 伤害 [抗性](Resistant.md "抗性")。 |
