## 目标

取回被偷的吊坠。

归还被偷的吊坠。

## 流程

当队伍首次进入[空洞](Hollow.md "空洞")时，会与[艾拉丁](Aradin.md "艾拉丁")的一名冒险者[巴斯](Barth.md "巴斯")和一个年轻的提夫林小孩[梅里](Meli.md "梅里")发生事件。接近位于 X: 168 Y: 544 的巴斯时，梅里会扒窃他，偷走一个[黄铜吊坠](Brass_Locket.md "黄铜吊坠")。巴斯意识到自己被抢了，并追赶那个孩子。如果什么都不做，巴斯最终会攻击梅里，并被守卫拖走。

队伍可以通过与巴斯或梅里交谈来干预。梅里一直否认自己有吊坠，而巴斯则坚持是他偷的。对话选项会根据阿拉贝尔是否存活而改变。

### 支持梅里

如果支持梅里，巴斯会被激怒，梅里会逃脱：

- 你敢碰那孩子一根手指试试
  - [游说] 你才是那个在威胁——一个孩子的人。冷静点。（难度等级 10） +1 +1 +1
  - [圣武士] [威吓] 无论你对孩子做什么，我都会十倍奉还。（难度等级 5） +1 +1 +1
  - [野蛮人] [威吓] 打那孩子啊，来啊。**看看会发生什么。**（难度等级 10） +1 +1 +1
  - [威吓] 你敢对那孩子动手，你就别想要那只手了。你自己选。（难度等级 10） +1 +1 +1
  - [提夫林] [游说] 只是警告。我以前对付过你这种人，人类。（难度等级 10） +1 +1 +1

### 支持巴斯

如果队伍决定站在巴斯这边，并通过一个游说属性检定，他们可以问梅里为什么拿吊坠。巴斯起初不屑一顾，但梅里声称它看起来像他妈妈的，巴斯决定让梅里留着吊坠。在这个场景之后梅里笑了，因为他的虚张声势成功了。

- 你最好照他说的做，小子。
  - [游说] 你骗不了任何人，小子。你为什么拿它？（难度等级 5）
  - [野蛮人] [威吓] 那个小玩意儿值得挨一顿**揍**？（难度等级 15）
  - [威吓] 你真的想为了一些珠宝挨顿打吗？（难度等级 15）
  - [提夫林] [游说] 把它还回去，小子。你比这强。（难度等级 5）
- [狂战士] [威吓] _照他说的做，否则就去和那个被蛇咬的女孩一起躺在坟墓里。_（难度等级 5）
- [野蛮人] [威吓] 偷东西的小孩在这附近都得死。照他说的做。（难度等级 5）
- [威吓] 最好照他说的做，小子。今天已经有一个贼要埋了。（难度等级 5） -1 -1 -1
  - [提夫林] [游说] 把它还回去，小子。你比这强。（难度等级 5）
  - [野蛮人] [威吓] 那个小玩意儿值得挨一顿**揍**？（难度等级 15）
  - [威吓] 你真的想为了一些珠宝挨顿打吗？（难度等级 15）
  - [游说] 你骗不了任何人，小子。你为什么拿它？（难度等级 5）

### 武僧选项

武僧可以要求大家冷静下来，然后沉着地要求更冷静的头脑：

- [武僧] 我认为大家都冷静下来对每个人都有好处。发生了什么？_（如果阿拉贝尔存活）_
- [武僧] 偷窃和复仇已经在这里害死了一个孩子。冷静下来——解释一下发生了什么。_（如果阿拉贝尔死亡）_

有与支持巴斯类似的选项，但还有另一个选项：

- [武僧] [游说] 让我们找到平衡。孩子会归还吊坠——而成年人会表现得像个大人。（难度等级 15）

### 什么都不做

或者，队伍可以选择什么都不做，这会导致一场冲突，巴斯会扇梅里耳光，梅里随后会呼叫[伊卡隆](Ikaron.md "伊卡隆")。

- 我给过你机会了。你自生自灭吧。 -1 -1
- _保持安静并观看。_ -1 -1
- 算了。我不想惹麻烦。

### 伊卡隆到来

如果未能威吓梅里或没有介入，[伊卡隆](Ikaron.md "伊卡隆")会过来。他询问发生了什么；队伍可以称梅里为小偷，并通过通过难度等级 15 的[游说](Persuasion.md "游说")或[威吓](Intimidation.md "威吓")[属性检定](Ability_Check.md "属性检定")来说服守卫支持巴斯，或者支持梅里让他自由离开。

此外，队伍可以在与梅里交谈前扒窃他，这样会出现一个新的对话选项：

- 放松。那小子不错，但我更厉害——这是你的吊坠。

如果发生这种情况，梅里会相当佩服并跑开；吊坠可以归还给巴斯。

### 将吊坠归还给巴斯

最后，如果梅里逃脱了，吊坠会存放在[提夫林避难所](Tiefling_Hideout.md "提夫林避难所")的一个盗贼箱子里。撬开箱子并偷回吊坠后，队伍可以将吊坠归还给巴斯。他随后会表示感激，如果队伍阻止了他扇梅里耳光，因为他冷静下来了，他也会感谢队伍。

- 你想要*这个*吊坠？它对你值多少钱？
  - 非常感谢。 +1 -1
  - 忘了钱吧。吊坠是你的了。拿去吧。（无奖励）
- [侦测思想] _读取他的思想。_（难度等级 10）
    - 我认为你可以提高一点报价。毕竟，这是你母亲的礼物。 -1 +1
- 这是吊坠。全是你的了。
- 是传家宝吗？在你家传了很久吗？
  - [游说] 那就不算什么传家宝了……除非背后有个故事，也许？（难度等级 15）
- [侦测思想] _读取他的思想。_（难度等级 10）
    - [游说] 来吧，说出来。关于吊坠，你还有什么没告诉我的？（难度等级 10）

## 任务奖励

### 金币

- 在巴斯与梅里交谈时偷回吊坠并归还：0 金币
- 从提夫林避难所偷回吊坠后归还：
  - 如果不索要任何东西，巴斯会给队伍 25 金币。
  - 如果索要报酬，巴斯支付 50 金币。

### 激励点

- ⁠[血债](Criminal.md#Blood_Price "罪犯")（[罪犯](Criminal.md "罪犯")背景） - 如果告诉巴斯队伍拿回了吊坠，但没有归还/卖回给他，他会变得敌对。
- ⁠[家人第一](Urchin.md#Family_Comes_First "流浪儿")（[流浪儿](Urchin.md "流浪儿")背景） - 如果说服梅里说出关于护符的真相。
- ⁠[护犊子](Urchin.md#Protect_Your_Own "流浪儿")（[流浪儿](Urchin.md "流浪儿")背景） - 为了保护梅里
