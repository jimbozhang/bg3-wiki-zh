# 法术位恢复站 等级 3

**法术位恢复站 等级 3** 是由 [法术症结护符](Spellcrux_Amulet.md "法术症结护符") 赋予的类动作。它是 [法术位恢复站](Spell_Slot_Restoration.md "法术位恢复站") 的选项之一，可恢复一个等级 3 的法术位，每 [长休](Long_Rest.md "长休") 一次。

## 描述

恢复一个等级 3 的 [法术位](Spell_Slot.md "法术位")。

## 属性

消耗
[动作](Actions.md#Resources "动作")
详情
范围：自身
充能：[长休](Long_Rest.md "长休")

## 学习方式

由物品赋予：

- [法术症结护符](Spellcrux_Amulet.md "法术症结护符")

此法术是以下法术的变体：
[法术位恢复站](Spell_Slot_Restoration.md "法术位恢复站")

## 备注

- 此能力可同时恢复一个常规的等级 3 法术位和一个 [邪术师](Warlock.md "邪术师") 等级 3 法术位。
- 如果以出身 [盖尔](Gale.md "盖尔") 游玩，并且你拥有一个 [等级 3 阴影法术位](Permanent_bonuses.md#Consumed_Shadow_Weave "永久增益")，此能力也会同时恢复该法术位。这意味着通过 [身兼多职](Multiclassing.md "身兼多职")，你可以用一个动作恢复最多 3 个等级 3 的法术位。
