# Modding:纹理格式

此页面为 [modding 页面](Modding_colon_Modding_resources.md "Modding:Modding resources")，遵循其自身规则和标准，与Wiki其余部分不同。

本教程来自 [atmosfur](https://www.nexusmods.com/users/14681269)

本指南包含多个与BG3使用的纹理格式相关的部分，以及如何在多个程序中导出和准备它们。本指南主要涉及从自定义3D模型烘焙的地图，目前不包括皮肤纹理。

**本指南涵盖的纹理地图包括：**

- \_BM (基础颜色地图)

- \_NM (法线地图) [如何在Blender中烘焙法线地图](https://styly.cc/tips/maku-blender-normalmap-bake/)

- \_MSK (遮罩地图)

- \_PM (物理地图)

**此外，要创建PM，您将需要：**

- AO (环境光遮蔽地图)

- 粗糙度地图

- 金属度地图

本指南还包含一个部分，介绍如何在GIMP中手动设置地图（如果您没有Substance Painter，或者手动编辑了纹理）。

## Substance Painter 导出预设

如果BG3的Substance Painter导出预设对您不起作用，或者您使用的是Adobe之前的Substance Painter版本，以下是您如何在Substance Painter中设置自己的导出预设。我个人将这些分为两个单独的预设，因为我的基础颜色和遮罩地图都从基础颜色信息的不同图层中提取。

1. 在 文件 -> 导出纹理 窗口中，选择 配置 选项卡。
2. 单击左侧“预设”旁边的小+图标以创建新预设。
3. 为NM、PM和MSK地图分别创建两个R+G+B+A输出地图和一个R+G+B输出地图。
4. 在右侧，您将看到需要拖放到输出地图R、G、B和A插槽中的输入地图。
5. 将输入地图拖放到输出地图选项卡中相应的插槽：
   **法线地图（确保使用转换后的地图 - NormalDirectX）：**
   R: 留空
   G: NormalDirectX，选择绿色通道
   B: NormalDirectX，选择蓝色通道
   A: NormalDirectX，选择红色通道
   **物理地图：**
   R: 金属度，选择灰色通道
   G: 粗糙度，选择灰色通道
   B: 环境光遮蔽，选择灰色通道
   A: 留空
   **遮罩地图：**
   这是专门用于号角纹理的，您需要检查您正在纹理化的任何内容的原版遮罩，以查看哪个通道具有用于自定义颜色的遮罩。在这种情况下，号角仅使用绿色通道。

   R: 对于号角留空
   G: 基础颜色，选择绿色通道
   B: 对于号角留空
6. 如前所述，对于基础颜色地图，我创建一个新的导出预设。为此，您只需要一个RGB+A输出地图，并将基础颜色输入地图拖放到RGB插槽。
7. 返回窗口的“导出”选项卡，从顶部的下拉菜单中选择您新创建的导出预设，然后只需点击导出。在允许您查看通道的程序（如GIMP或Photoshop）中仔细检查您创建的纹理，确保它们都已正确分配。您现在仍需要将纹理转换为DDS。

## 创建DDS文件

### 使用Paint.NET进行简单转换

如果您的纹理的所有通道都已正确设置，您只需在您选择的编辑程序中以正确的DDS压缩格式导出它们。我以Paint.NET为例。

**基础颜色地图设置：**
选择 BC1 (Linear, DTX1)，确保勾选 生成Mip贴图
**遮罩地图设置：**
选择 BC5 (Linear, Unsigned)，确保勾选 生成Mip贴图
**法线地图设置：**
选择 BC3 (Linear, DTX5)，确保勾选 生成Mip贴图
**物理地图设置：**
选择 BC1 (Linear, DTX1)，确保勾选 生成Mip贴图

### 使用GIMP手动分配通道

如果您在Substance Painter以外的程序中创建了纹理，您可能需要手动将正确的通道分配到您的地图。您最好的选择可能是GIMP，如果您从未使用过该程序，我试图使本指南的这一部分尽可能对初学者友好。
如果您正在按照本指南操作，我建议一次处理一种地图类型。同样，这以号角纹理为例，游戏中其他纹理的设置方式可能不同，并且不涉及皮肤纹理的设置方式。

#### 法线地图

1. 在GIMP中打开您的法线地图，单击通道选项卡以查看地图的通道。根据您创建法线地图的程序，您可能已有也可能没有Alpha通道。
2. 如果您的通道中没有看到Alpha通道，请切换回图层选项卡，右键单击您的图层并选择“添加Alpha通道”。这将添加一个空的Alpha通道。
3. 回到通道选项卡，右键单击“红色”通道并选择“通道到选区”。
4. 切换回图层选项卡，然后在菜单栏中转到“编辑 -> 清除”以删除所选区域。这基本上是将红色通道的内容复制到Alpha通道。
5. 在菜单栏中转到“选择 -> 无”以取消选择图像。
6. 检查通道选项卡，您的Alpha通道现在应该看起来像您的红色通道。
7. 在菜单栏中，转到“颜色 -> 组件 -> 分解”。这将创建并打开文件的新版本，所有通道都分离到单独的图层中。
8. 在弹出窗口中选择RGBA作为颜色模式，然后点击确定。
9. 您现在应该看到一个看起来像您的法线地图的文件，但为灰度。在图层选项卡中，您将看到每个通道现在都有自己的图层。
10. 在下一步之前，确保您的前景色设置为黑色。如果不是，您可以单击颜色选择器下方的黑白方块小图标进行重置。
11. 单击“红色”图层将其设置为活动图层，然后在菜单栏中转到“编辑 -> 用前景色填充”以用黑色填充该图层。
12. 在菜单栏中转到“颜色 -> 组件 -> 重新组合”，将图层重新组合回原始文件中的通道。
13. 回到原始文件并检查通道选项卡，确保红色通道现在只是黑色（空）。
14. 如果您的原始文件看起来与之前相同，请确保在通道选项卡中没有通道被选中/高亮显示。如果任何通道被选中，只需再次单击它们以取消选择，然后返回分解图层文件并重复步骤12以重新组合。
15. 您的重新组合文件现在可以导出了。转到“文件 -> 导出为...”
16. 选择文件的保存位置，并根据您的选择命名。检查原版游戏纹理以查看纹理所针对文件的命名约定。我个人还导出一个.png版本作为备份，但您需要.dds文件用于游戏。您可以使用.dds扩展名重命名文件，或者单击底部的“选择文件类型（按扩展名）”以选择.dds文件类型。准备就绪后，只需单击“导出”以保存您的纹理。将弹出一个带有导出选项的窗口。对于.png，只需将所有内容保留为默认值。
17. 要正确将法线地图导出为.dds文件，请确保为压缩选择“BC3/DTX5”，为Mipmap选择“生成mipmap”。将Mipmap选项下的过滤器设置为“Kaiser”。

#### 物理地图

要创建物理地图，您将需要这三个地图：

- AO地图
- 粗糙度地图
- 金属度地图

1. 在GIMP中打开所有三个地图。（如果您的金属度地图只是黑色，则跳过打开它）
2. 转到您的粗糙度地图，在菜单栏中，转到“图像 -> 模式 -> RGB”以将图像从灰度模式转换为RGB模式。这将创建所需的RGB通道。
3. 右键单击您的图层并选择“添加Alpha通道”。这将添加一个空白的Alpha通道。
4. 在菜单栏中，转到“颜色 -> 组件 -> 分解”。这将创建并打开文件的新版本，所有通道都分离到单独的图层中。
5. 在弹出的窗口中，选择“RGBA”作为颜色模式，然后点击确定。
6. 您现在应该看到第二个文件，看起来像您的粗糙度地图，每个通道作为单独的图层。
7. 如果您的金属度地图只是黑色，您可以简单地用黑色填充“红色”图层。如果您的金属度地图包含信息，请按照AO地图的步骤将金属度地图插入“红色”图层。
   要用黑色填充图层，单击“红色”图层将其设置为活动图层，然后在菜单栏中转到“编辑 -> 用前景色填充”，确保您的前景色设置为黑色。
8. 现在是插入AO地图的时候了。为了使操作更简单，通过单击它们旁边的小眼睛图标来隐藏“红色”和“绿色”图层。
9. 现在切换到您的AO地图文件，在菜单栏中转到“选择 -> 全选”以选择整个图像。
10. 在菜单栏中，转到“编辑 -> 复制”（或按Ctrl+C）以复制您的AO地图图像。
11. 切换回分解图层文件，单击“蓝色”图层将其设置为活动图层。
12. 在菜单栏中，转到“编辑 -> 粘贴到位置”以粘贴您复制的AO地图。
13. 您现在应该看到您的AO地图作为“浮动选区”粘贴在文件中。
14. 在图层选项卡中，单击底部的小锚点图标以将粘贴的图层锚定到其下方的图层。
15. 您的“蓝色”图层现在应该与粘贴的图像合并。以这种方式合并图层可确保它们最终能正确重新组合。如果您的金属度地图包含信息，请按照AO地图的方式将其粘贴到“红色”图层。
16. 单击您之前隐藏的图层旁边的小眼睛图标，使它们再次可见。
17. 在菜单栏中转到“颜色 -> 组件 -> 重新组合”，将图层重新组合回原始文件中的通道。
18. 回到原始文件并检查通道选项卡，确保它正确重新组合 - 您的PM地图现在应该看起来像这样。
19. 如果您的原始文件看起来与之前相同，请确保在通道选项卡中没有通道被选中/高亮显示。如果任何通道被选中，只需再次单击它们以取消选择，然后返回分解图层文件并重复步骤17以重新组合。
20. 完成的PM的通道顺序应为：红色为金属度，绿色为粗糙度，蓝色为AO，Alpha中无内容。
21. 您的重新组合文件现在可以导出了。转到“文件 -> 导出为...”
22. 选择文件的保存位置，并根据您的选择命名。检查原版游戏纹理以查看纹理所针对文件的命名约定。我个人还导出一个.png版本作为备份，但您需要.dds文件用于游戏。您可以使用.dds扩展名重命名文件，或者单击底部的“选择文件类型（按扩展名）”以选择.dds文件类型。准备就绪后，只需单击“导出”以保存您的纹理。将弹出一个带有导出选项的窗口。对于.png，只需将所有内容保留为默认值。
23. 要正确将物理地图导出为.dds文件，请确保为压缩选择“BC1/DTX1”，为Mipmap选择“生成mipmap”。将Mipmap选项下的过滤器设置为“Kaiser”。

#### 基础颜色地图

2. 您的文件现在可以导出了。转到“文件 -> 导出为...”
3. 选择文件的保存位置，并根据您的选择命名。检查原版游戏纹理以查看纹理所针对文件的命名约定。我个人还导出一个.png版本作为备份，但您需要.dds文件用于游戏。您可以使用.dds扩展名重命名文件，或者单击底部的“选择文件类型（按扩展名）”以选择.dds文件类型。准备就绪后，只需单击“导出”以保存您的纹理。将弹出一个带有导出选项的窗口。对于.png，只需将所有内容保留为默认值。
4. 要正确将基础颜色地图导出为.dds文件，请确保为压缩选择“BC1/DTX1”，为Mipmap选择“生成mipmap”。将Mipmap选项下的过滤器设置为“Kaiser”。

#### 遮罩地图

游戏中遮罩地图使用的通道根据遮罩的用途而有所不同。此示例仅适用于号角纹理，因此请先检查原版游戏纹理以找出您需要的通道。无论如何，遮罩地图不需要Alpha通道，因此您只需将它们导出为.dds文件即可。
无论如何，检查通道选项卡以确保您只在应包含信息的通道中拥有信息，即对于号角遮罩地图，仅“绿色”通道，这没有坏处。
1. 您的文件现在可以导出了。转到“文件 -> 导出为...”
2. 选择文件的保存位置，并根据您的选择命名。检查原版游戏纹理以查看纹理所针对文件的命名约定。我个人还导出一个.png版本作为备份，但您需要.dds文件用于游戏。您可以使用.dds扩展名重命名文件，或者单击底部的“选择文件类型（按扩展名）”以选择.dds文件类型。准备就绪后，只需单击“导出”以保存您的纹理。将弹出一个带有导出选项的窗口。对于.png，只需将所有内容保留为默认值。
3. 要正确将遮罩地图导出为.dds文件，请确保为压缩选择“BC5/ATI2 (3Dc)”，为Mipmap选择“生成mipmap”。将Mipmap选项下的过滤器设置为“Kaiser”。

## 导出设置TL;DR

如果您已经知道如何进行所有通道编辑，只需要快速参考这些地图的设置和导出方式。

法线地图通道：

- 红色 - 无
- 绿色 - 绿色
- 蓝色 - 蓝色
- Alpha - 红色

物理地图通道：

- 红色 - 金属度
- 绿色 - 粗糙度
- 蓝色 - AO
- Alpha - 无

基础颜色地图通道：

- 红色 - 红色
- 绿色 - 绿色
- 蓝色 - 蓝色
- Alpha - 无

遮罩地图通道：

- 检查您正在遮罩的任何内容的原版纹理。这些通常不需要Alpha通道。

DDS导出设置：

- **法线地图：**
  - BC3 DTX5
- **物理地图：**
  - BC1 DTX1
- **基础颜色地图：**
  - BC1 DTX1
- **遮罩地图：**
  - 这同样取决于您创建的遮罩类型。我将在此处收集我找到的不同遮罩地图的尽可能多的信息。
  - 对于号角遮罩：BC5 Unsigned (Paint.NET) / BC5 ATI2 (GIMP)
  | - BC7 RGBA 8bpp | 显式Alpha (Photoshop) / BC7 sRGB 或 BC7 Linear (Paint.NET) 用于共享伤疤MSK（感谢 [aurebuen](https://www.nexusmods.com/baldursgate3/users/47038262) 提供的信息） |
