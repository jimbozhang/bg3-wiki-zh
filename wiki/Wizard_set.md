# 法师套装

The **Wizard set** includes items that interact with [Wizard](Wizard.md "Wizard") mechanics.

See also: [Weave set](Weave_Set.md "Weave set")

| 名称 | 插槽 | 效果 |
| --- | --- | --- |
| 尖顶帽 | [帽子](Headgear.md "Headgear") ([非护甲](Headwear#Non-Armour.md#Non-Armour "Headwear")) | [华丽尖顶帽](Glorious_Pointy_Hat.md "Glorious Pointy Hat")你获得等于你[智力](Intelligence.md "Intelligence") [调整值](Modifier.md "Modifier")的[游说](Persuasion.md "Persuasion") [属性检定](Ability_Check.md "属性检定")加值。 |
| 弯曲的智力头带 | [帽子](Headgear.md "Headgear") ([非护甲](Headwear#Non-Armour.md#Non-Armour "Headwear")) | 将佩戴者的[智力](Intelligence.md "Intelligence")属性值设为17。如果佩戴者不装备此物品时智力属性值更高，则此[惑控学派](Enchantment.md "Enchantment")效果无效。 |
