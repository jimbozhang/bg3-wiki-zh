# 窥探之心 (状态)

**窥探之心**

- 受影响实体将被维康妮娅的攻击自动[重击](Critical_Hit.md "Critical hit")，且其在感知[豁免检定](Saving_throws.md "Saving Throws")上具有[劣势](Disadvantage.md "Disadvantage")。

| 状态 | 效果 |
| --- | --- |
| [怯懦之心](Craven_Heart_(Condition).md "Craven Heart (Condition)") | 受影响实体将被维康妮娅的攻击自动[重击](Critical_Hit.md "Critical Hit")，且其在感知[豁免检定](Saving_throws.md "Saving Throws")上具有[劣势](Disadvantage.md "Disadvantage")。要结束此效果，生物必须通过在维康妮娅3米（10英尺）范围内勇敢地结束其回合，且不使用任何动作或附赠动作，来否定其怯懦之心。 |
| [虚伪之心](False_Heart_(Condition).md "False Heart (Condition)") | 受影响实体将被维康妮娅的攻击自动[重击](Critical_Hit.md "Critical Hit")，且其在感知[豁免检定](Saving_throws.md "Saving Throws")上具有[劣势](Disadvantage.md "Disadvantage")。要结束此效果，生物必须通过[沉默](Silenced_(Condition).md "Silenced (Condition)")自身来否定其虚伪之心。 |
| [嫉妒之心](Jealous_Heart_(Condition).md "Jealous Heart (Condition)") | 受影响实体将被维康妮娅的攻击自动[重击](Critical_Hit.md "Critical hit")，且其在感知[豁免检定](Saving_throws.md "Saving Throws")上具有[劣势](Disadvantage.md "Disadvantage")。要结束此效果，生物必须通过[协助](Help.md "Help")一名盟友来否定其嫉妒之心。 |
| [忧郁之心](Melancholy_Heart_(Condition).md "Melancholy Heart (Condition)") | 受影响实体将被维康妮娅的攻击自动[重击](Critical_Hit.md "Critical hit")，且其在感知[豁免检定](Saving_throws.md "Saving Throws")上具有[劣势](Disadvantage.md "Disadvantage")。要结束此效果，生物必须通过[表演](Perform.md "Perform")、聆听表演或目睹另一项使其快乐的行为来否定其忧郁之心。 |
| 窥探之心 | 受影响实体将被维康妮娅的攻击自动[重击](Critical_Hit.md "Critical hit")，且其在感知[豁免检定](Saving_throws.md "Saving Throws")上具有[劣势](Disadvantage.md "Disadvantage")。要结束此效果，生物必须通过[中毒](Poisoned_(Condition).md "Poisoned (Condition)")自身来否定其窥探之心。 |
| [暴戾之心](Violent_Heart_(Condition).md "Violent Heart (Condition)") | 受影响实体将被维康妮娅的攻击自动[重击](Critical_Hit.md "Critical Hit")，且其在感知[豁免检定](Saving_throws.md "Saving Throws")上具有[劣势](Disadvantage.md "Disadvantage")。要结束此效果，生物必须通过治疗一名敌人来否定其暴戾之心。 |
| [迷失之心](Wayward_Heart_(Condition).md "Wayward Heart (Condition)") | 受影响实体将被维康妮娅的攻击自动[重击](Critical_Hit.md "Critical Hit")，且其在[感知](Wisdom.md "Wisdom")[豁免检定](Saving_throws.md "Saving Throws")上具有[劣势](Disadvantage.md "Disadvantage")。要结束此效果，生物必须通过顺从莎尔并在战斗中协助维康妮娅来否定其迷失之心。 |
