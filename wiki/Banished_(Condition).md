# 流放 (状态)

"Banished" redirects here. For the note by the same name, see [Banished (note)](Banished_(note)..md)

**流放**

- 从当前存在位面被放逐。
- 无法被选中为目标。
- 无法移动或执行 [动作](Action.md "Action")、[附赠动作](Bonus_action.md "Bonus Action") 或 [反应](Reaction.md "Reaction")
