# 虚假之心 (状态)

**虚假之心**

- 受影响实体将被维康妮亚的攻击自动[重击](Critical_Hit.md "重击")，且其在感知[豁免检定](Saving_throws.md "豁免检定")上具有[劣势](Disadvantage.md "劣势")。

| 状态 | 效果 |
| --- | --- |
| [懦弱之心](Craven_Heart_(Condition).md "懦弱之心 (状态)") | 受影响实体将被维康妮亚的攻击自动[重击](Critical_Hit.md "重击")，且其在感知[豁免检定](Saving_throws.md "豁免检定")上具有[劣势](Disadvantage.md "劣势")。要结束此效果，生物必须通过在维康妮亚3米（10英尺）范围内勇敢地结束其回合，且不使用任何动作或附赠动作，来否认其懦弱之心。 |
| 虚假之心 | 受影响实体将被维康妮亚的攻击自动[重击](Critical_Hit.md "重击")，且其在感知[豁免检定](Saving_throws.md "豁免检定")上具有[劣势](Disadvantage.md "劣势")。要结束此效果，生物必须通过[沉默](Silenced_(Condition).md "沉默 (状态)")自身来否认其虚假之心。 |
| [嫉妒之心](Jealous_Heart_(Condition).md "嫉妒之心 (状态)") | 受影响实体将被维康妮亚的攻击自动[重击](Critical_Hit.md "重击")，且其在感知[豁免检定](Saving_throws.md "豁免检定")上具有[劣势](Disadvantage.md "劣势")。要结束此效果，生物必须通过[协助](Help.md "协助")一名盟友来否认其嫉妒之心。 |
| [忧郁之心](Melancholy_Heart_(Condition).md "忧郁之心 (状态)") | 受影响实体将被维康妮亚的攻击自动[重击](Critical_Hit.md "重击")，且其在感知[豁免检定](Saving_throws.md "豁免检定")上具有[劣势](Disadvantage.md "劣势")。要结束此效果，生物必须通过[表演](Perform.md "表演")、聆听表演或目睹另一件令其愉悦的举动来否认其忧郁之心。 |
| [窥探之心](Prying_Heart_(Condition).md "窥探之心 (状态)") | 受影响实体将被维康妮亚的攻击自动[重击](Critical_Hit.md "重击")，且其在感知[豁免检定](Saving_throws.md "豁免检定")上具有[劣势](Disadvantage.md "劣势")。要结束此效果，生物必须通过[中毒](Poisoned_(Condition).md "中毒 (状态)")自身来否认其窥探之心。 |
| [暴戾之心](Violent_Heart_(Condition).md "暴戾之心 (状态)") | 受影响实体将被维康妮亚的攻击自动[重击](Critical_Hit.md "重击")，且其在感知[豁免检定](Saving_throws.md "豁免检定")上具有[劣势](Disadvantage.md "劣势")。要结束此效果，生物必须通过治疗一名敌人来否认其暴戾之心。 |
| [歧途之心](Wayward_Heart_(Condition).md "歧途之心 (状态)") | 受影响实体将被维康妮亚的攻击自动[重击](Critical_Hit.md "重击")，且其在[感知](Wisdom.md "感知")[豁免检定](Saving_throws.md "豁免检定")上具有[劣势](Disadvantage.md "劣势")。要结束此效果，生物必须通过顺从莎尔并在战斗中协助维康妮亚来否认其歧途之心。 |
