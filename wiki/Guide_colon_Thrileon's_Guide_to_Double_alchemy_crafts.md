## **如何获得99.75%几率每次制造双倍药水（炼金术指南）**

在本指南中，我将展示如何在使用炼金术时达到**99.75%的几率每次制造双倍药水**。

你需要一个**10级角色**来达到最大成功率，尽管在此之前你就有很好的机会。

（*英语不是我的母语，如有错误请见谅。*）

我们首先需要一个用于制造的角色。我建议使用**雇佣兵**，因为这个角色将留在营地，不用于战斗。

以下是等级进度和重要选择。

- 等级1 - 法师等级1 - 16感知 + 医药熟练项
- 等级2 - 法师等级2 - 副职：变化学派
- 等级3 - 牧师等级1 - 神导术戏法
- 等级4 - 牧师等级2
- 等级5 - 牧师等级3 - 法术：强化属性
- 等级6 - 牧师等级4 - 专长 -> 属性值提升 -> +2感知
- 等级7 - 吟游诗人等级1
- 等级8 - 吟游诗人等级2
- 等级9 - 吟游诗人等级3 - 医药专精
- 等级10 - 吟游诗人等级4 - 专长 -> 属性值提升 -> +2感知

## **工作原理**

**变化学派法师**副职在等级2时获得特性**试验炼金术**。

这允许你在合并提取物时，如果成功通过**DC 15 医药检定（感知）**，则酿造**两种炼金溶液而非一种**。

构建的其余部分专注于增加此检定的加值并提高成功概率。

______________________________________________________________________

## **步骤1：法师等级**

我们从**2级法师**开始，以解锁**试验炼金术**。

此时的加值为：

- +2 医药熟练项
- +3 感知调整值

因为检定是**DC 15**，你需要在1d20上掷出**10或更高**。

这大约有**55%的成功几率**。

______________________________________________________________________

## **步骤2：牧师等级**

接下来我们兼职**牧师4级**。

- 牧师等级1给予**神导术**戏法，为**属性检定**添加**1d4**。
- 牧师等级3授予**强化属性**法术。

对自己使用*强化属性 -> 枭之感知*以获得**感知检定优势**。

优势意味着掷**两个d20并取较高结果**，这通常比神导术的+1d4更好。因为两个法术都需要专注，你通常使用**枭之感知**而非神导术。

（*如果另一个角色对你的制造者施放神导术，你可以同时受益。*）

在**牧师等级4**时，选择：

- 专长 -> 属性值提升 -> +2感知

此时你的加值为：

- +3 医药
- +4 感知
- 检定优势

这意味着你现在需要在骰子上掷出**8或更高**。

______________________________________________________________________

## **步骤3：吟游诗人等级**

最后我们兼职**吟游诗人4级**。

前两个吟游诗人等级不为此构建提供重要加值。

在**吟游诗人等级3**时选择：

- **专精：医药**

这使你的医药检定熟练项加值翻倍。

在**吟游诗人等级4**时选择：

- 专长 -> 属性值提升 -> +2感知

最终加值：

- +5 感知
- +8 医药
- 检定优势

这提供了**总计+13的加值**。

因为检定是**DC 15**，任何**2或更高的掷骰都会成功**。

然而，**自然1总是失败**。在优势下，失败的唯一方式是**两个骰子都掷出1**。

失败几率：

- 1/20 x 1/20 = **0.25%**

成功几率：

- **99.75%**

## **替代构建（半身人）**

另一个可能的构建使用**半身人雇佣兵**，其具有**半身人幸运**特性（重掷属性检定上的自然1）。

- 等级1 - 法师等级1 - 16感知 + 医药熟练项
- 等级2 - 法师等级2 - 副职：变化学派（57.75%成功几率）
- 等级3 - 游荡者等级1 - 专精：医药（68.25%成功几率）
- 等级4 - 法师等级3
- 等级5 - 法师等级4 - 专长 -> 属性值提升 -> +2感知（84%成功几率）
- 等级6 - 游荡者等级2
- 等级7 - 游荡者等级3
- 等级8 - 游荡者等级4 - 专长 -> 属性值提升 -> +2感知（89.25%成功几率）
- 等级9 - 任意职业 - 额外熟练项（99.75%成功几率）

## **额外思路**

一个**吉斯洋基人**角色使用**星界知识（感知）**可以获得以下熟练项：

- 驯兽
- 洞悉
- 医药
- 察觉技能
- 求生

还有一个**吉斯洋基武僧雇佣兵**，可以用守墓人重新指定为**法师2 / 牧师X**，这非常适合药水制造。

一些玩家在营地保留几个**牧师雇佣兵**以维持长期增益，例如：

- 守护之链
- 防护毒素
- 死亡防护
- 昼明术
- 支援术

这可以大大减少休息需求并提高整体队伍耐久度。
