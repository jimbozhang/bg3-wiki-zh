# 火焰套装

火焰套装是一套造成 [火焰](Fire.md "火焰") 伤害并施加 [燃烧](Burning_(Condition).md "燃烧（状态）") 状态的装备。与 [灼热套装](Heat_Set.md "灼热套装")、[地狱火套装](Helldusk_set.md "地狱火套装") 和 [元素套装](Elemental_set.md "元素套装") 的物品协同作用，这些套装在此不重复列出。它包含以下部件：

- 仅限利用。
- 造成额外 1d6 [火焰](Fire.md "火焰") 伤害。
