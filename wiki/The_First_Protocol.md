# 第一协议

另请参阅：[沃斯的信](Letter_from_Voss.md "沃斯的信")

**第一协议**是玩家角色在[终章](Epilogue.md "终章")中可以收到的信件之一。它有两种变体，取决于[莱埃泽尔](Lae'zel.md "莱埃泽尔")是否离开[托瑞尔](Toril.md "托瑞尔")去对抗[维拉基斯](Vlaakith.md "维拉基斯")。

这块[石盘](Githyanki_Slates.md "吉斯洋基石板")上刻有圆形符文和象形文字。

- 价格：14 金币

## 文本

_如果莱埃泽尔与俄耳甫斯一同对抗维拉基斯_ ( UID `S_EPI_Letter_OrpheusWithLaezel` / UUID `493ced21-50b6-4dbb-ad8d-aa4013e4a7bc` ):

致解放我的 _Mla'ghir_。致莱埃泽尔。_T'lak'ma Ghir_。我的话语刻于石板之上，因其永恒不变。

你们不仅是解放者——无论你们行至何方土地，跨越何方海洋，踏足何方位面，你们皆是彗星战士。

维拉基斯的爪牙正逐一倒在我忠诚信徒的手中。但[死亡之城](https://forgottenrealms.fandom.com/wiki/Tu%27narath)依然矗立，女王仍端坐于其中心的邪恶王座之上。但我们正立于悬崖边缘。莱埃泽尔：多亏了你，我们即将恢复古老的亲缘关系，而维拉基斯将在由此点燃的火焰中燃烧。协议已被重写，而第一协议，我献予你，_Mla'ghir_：

_所有自称为吉斯者，将团结于同一片天空之下。_

_Ghith'ka tavkim krash'ht._

俄耳甫斯

_如果莱埃泽尔未与俄耳甫斯同行或未被招募_ ( UID `S_EPI_Letter_Orpheus` / UUID `e6bb88cf-c5cc-4345-8135-537ee8f2a2ef` ):

_Mla'ghir_。我的话语刻于石板之上，因其永恒不变。

你不仅是解放者——无论你行至何方土地，跨越何方海洋，踏足何方位面，你皆是彗星战士。

维拉基斯的爪牙正逐一倒在我忠诚信徒的手中。但死亡之城依然矗立，女王仍端坐于其中心的邪恶王座之上。但我们正立于悬崖边缘。我们即将恢复古老的亲缘关系，而维拉基斯将在由此点燃的火焰中燃烧。协议已被重写，而第一协议，我献予你，_Mla'ghir_：

_所有自称为吉斯者，将团结于同一片天空之下。_

_Ghith'ka tavkim krash'ht._

俄耳甫斯
