# 皮肤松弛的脑袋

| | |
| --- | --- |
| 配音 | - [Ellie Heydon](https://www.imdb.com/name/nm8703995/) |

## 数据

### 种族

精灵

**皮肤松弛的脑袋**和大脑档案室接口可以在[夺心魔殖民地](Mind_Flayer_Colony.md "夺心魔殖民地")中并排找到，前提是击败了[死亡牧者](Death_Shepherd.md "死亡牧者")、一些飞天恐魔和各种僵尸。

## 参与

可以在绿色罐子中找到各种“意志”，并将其插入大脑档案室接口，这使得玩家角色可以通过皮肤松弛的脑袋与不幸的生物进行对话。

**在[鹦鹉螺](Nautiloid.md "鹦鹉螺")的[序幕](Prologue.md "序幕")中有2个意志可供寻找：**

- [黑暗的大脑](Dark_Mind.md "黑暗的大脑")
- [被奴役的大脑](Slave_Mind.md "被奴役的大脑")

**在[第二幕](Act_Two.md "第二幕")的[夺心魔殖民地](Mind_Flayer_Colony.md "夺心魔殖民地")内有6个意志可供寻找：**

- [摧残意志](Butchered_Mind.md "摧残意志") - 在[停尸房](Mind_Flayer_Colony.md#Morgue "夺心魔殖民地")中一堆带有废弃护甲的架子顶部，[切割者](Chop.md "切割者")在此工作
- [封闭意志](Closed_Mind.md "封闭意志") - 在[铁指](Ironfinger.md "铁指")房间的架子上，位于[夺心魔殖民地](Mind_Flayer_Colony.md "夺心魔殖民地")的西北部
- [新鲜意志](Fresh_Mind.md "新鲜意志") - 在通往最终区域的电梯旁一堆粘液盒的顶部
- [真实意志](True_Mind.md "真实意志") - 在[黯蚀实验室](Mind_Flayer_Colony.md#Necrotic_Laboratory "夺心魔殖民地")中脑袋的前方
- [苏醒意志](Waking_Mind.md "苏醒意志") - 在[黯蚀实验室](Mind_Flayer_Colony.md#Necrotic_Laboratory "夺心魔殖民地")的一扇门后，**仅**在解决[大脑测绘谜题](Brain_Mapping_Puzzle.md "大脑测绘谜题")后方可进入
- [自愿意志](Willing_Mind.md "自愿意志") - 在[黯蚀实验室](Mind_Flayer_Colony.md#Necrotic_Laboratory "夺心魔殖民地")中脑袋西侧的桌子上

苏醒意志是独特的，因为与脑袋交谈可以获得永久增益[吉斯泽莱人的精神屏障](Githzerai_Mind_Barrier_(Condition)..md)
