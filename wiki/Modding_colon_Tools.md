## 工具

一些非常有用的工具，将使您的 BG3 模组制作生活更加轻松。

- [Lslib](https://github.com/Norbyte/lslib) by Norbyte
  - 模组制作中许多必要对话类型所需。

  - [BG3 Multi-Tool](https://github.com/ShinyHobo/BG3-Modders-Multitool) by ShinyHobo
  - 允许您打包和解包模组，搜索游戏的资源文件等！

  - [BG3 Mini tool](https://www.nexusmods.com/baldursgate3/mods/502) by Padme4000
  - 具有多种有用功能，包括为您创建 Meta.lsx 文件。

  - [BG3 Creator's Cauldron](https://www.nexusmods.com/baldursgate3/mods/1808) by Keefr
  - 使用 Creator's Cauldron 深入 BG3 自制世界！从自定义魔法护甲到强大法术，这款工具让您轻松创建自己的 BG3 内容。

  - [Dice Sprite Generator](https://www.nexusmods.com/baldursgate3/mods/1325) by Anecx
  - 这些工具允许您一键生成 25 个动画骰子精灵，供您的骰子使用！

  - [Language Support for BG3 VSCode Extension](https://www.nexusmods.com/baldursgate3/mods/2594) by Chromosome16
  - 在 Visual Studio Code 中为 bg3 文本文件提供文本高亮显示。

  - [BG3 Search Engine](https://bg3.norbyte.dev/search) by Norbyte
  - 允许您在线搜索游戏文件，无需在计算机上索引或解包任何内容。

  - 代码编辑器，以下是一些选项：
  - Windows: [VS Code](https://code.visualstudio.com/), [Notepad++](https://notepad-plus-plus.org/), [Brackets](https://brackets.io/), [Atom Community](https://atom-community.github.io/)
  - Mac: [Espresso](https://espressoapp.com/), [BBedit](https://www.barebones.com/)
  - Linux: [KWrite](https://apps.kde.org/en-gb/kwrite/), [Kate](https://apps.kde.org/en-gb/kate/), [Gedit](https://wiki.gnome.org/Apps/Gedit), [VS Codium](https://vscodium.com/)
  - Linux 用户还需要安装最新版本的 [WINE](https://wiki.winehq.org/Download)。通过您所选发行版的应用商店或 CLI 使用包管理器安装它。

### Blender 插件

  - [网格导入/导出插件](https://github.com/Norbyte/dos2de_collada_exporter) by Norbyte
  - 如果您想在 Blender 中处理 .GR2 文件（BG3 使用的格式），则需要此插件。

  - [Padme's Addons](https://www.nexusmods.com/baldursgate3/mods/346) by Padme4000
  - 包括用于头部导出顺序、轻松 LOD 创建、法线贴图传输、快速 UV 按钮和重置变换的插件。

  - [Armature Plugin](https://www.nexusmods.com/baldursgate3/mods/464) by Aloija
  - Blender 插件，帮助编辑 BG3 模型的骨架。

    - [Outfit Builder](https://www.nexusmods.com/baldursgate3/mods/3683) by LazyIcarus
  - 给定基础身体（例如 HUM_F）和 HUM_F 的身体变体（例如 HUM_FS，或此处的某个身体模组）中的护甲，为这些身体变体生成护甲网格。

  - [Volnos Texture Toolbox](https://www.nexusmods.com/baldursgate3/mods/4310) by Volno
  - 一个模组资源，包含用于修改 BG3 皮肤纹理的工具集合，以及 Baldur's Gate 3 皮肤着色器的重现。
