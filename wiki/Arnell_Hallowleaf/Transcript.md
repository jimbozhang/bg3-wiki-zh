# 阿内尔·圣叶/对话记录

## 营地对话

影心父亲的默认营地对话。他已开始从莎尔洞穴的折磨中恢复。他有针对影心玩家和非影心玩家的独特分支，以及在第一次重逢前后发生的分支。

### 影心，重逢

**阿内尔·圣叶:** 珍妮薇尔。我仍然无法相信你解救了我们——我就这样和你站在一起。我真想让你打我一巴掌，好让我知道这是真的，但我现在还没那么强壮，承受不住。

- **影心** : 我有很多想问的事。
  - **阿内尔·圣叶:** 我能想象。不过，我可能还需要一点时间才能给出许多答案。四十年的囚禁让我的头脑...布满了蛛网。但只要休息一下，就会清晰起来。
- **影心** : 那个名字。我不知道该怎么看待它。
  - **阿内尔·圣叶:** 给它点时间。给我们点时间。这一下子要接受太多东西了——对我也一样。
  - 我该积蓄力量了。我等这一刻等了这么久，再等一小会儿也无妨...
- **影心** : 妈妈怎么样了？
  - **阿内尔·圣叶:** 她很坚强...但她也是人类。这些年对她来说比我更艰难，无论是身体还是精神。莎尔的深渊夺走了我们永远无法挽回的宝贵时光。
  - 但别担心。我们又在一起了——我们会更加珍惜每一刻。
- **影心** : 他们让我对你们俩做了那些事，这么多年。拜托——我需要知道。
  - **阿内尔·圣叶:** 也许...也许晚点吧，我的孩子。那件事太难开口。我恐怕现在还没那个力气。
- _[(继续)](#questions)_
- **影心** : 你该休息了。我们以后有的是时间谈。
  - **阿内尔·圣叶:** 你有你母亲的智慧。是的。我该积蓄力量了。我等这一刻等了这么久，再等一小会儿也无妨...
- _(对话结束)_
- **阿内尔·圣叶:** 我该积蓄力量了。我等这一刻等了这么久，再等一小会儿也无妨...
- _(对话结束)_

### 影心，第二次对话

**阿内尔·圣叶:** 影心，我亲爱的。一切都好，我希望。
我答应过你，等我休息好了就跟你叙旧——也许你有什么想问的。

- **影心** : 那天晚上在树林里到底发生了什么，当我被绑架的时候？
  - **阿内尔·圣叶:** 那天晚上你本该成年——独自留在树林里，只靠塞伦涅的神导术找到回家的路。
  - 但你并不是独自在树林里。我们在村子附近抓到了一个莎尔教徒间谍，得知他的同伙企图绑架你。
  - 我必须找到你——带你回家——而且必须迅速。匆忙中，我犯了一个可怕的错误。
  - 我化身为狼形，以便更快地赶路。这奏效了——我循着你的气味，在莎尔教徒之前找到了你。
  - 但当我找到你时，你并没有看到赶来救你的父亲——你只看到一头狼，在黑暗中追猎你。
  - 我本该在那晚之前就告诉你我的诅咒。那样的话...一切都会不同。
  - 你惊慌失措，我还没来得及解释，莎尔教徒就找到了我们。父亲和女儿。狼和猎物。
  - 他们把我们俩都抓走了。
  - 当我醒来时，你已经不见了，但你母亲和我在一起。他们正把我们偷偷运往城市，藏匿并束缚着。
  - 那本该是你成年、进入月之少女教诲的日子。
  - 相反，莎尔教徒利用我们来给你上不同的课。痛苦的课。
- **影心** : 你怎么能随意变成狼？
      - **阿内尔·圣叶:** 月之少女的祝福有很多。她教会我控制诅咒——但她无法教会我控制我的父性本能。
      - 这是我的失败。
- **影心** : 对不起。如果我没有害怕，如果我没有逃跑，这一切就不会发生。
      - **阿内尔·圣叶:** 你不能责怪自己。你当时还是个孩子；保护你本该是你母亲和我的责任。
      - 我们变得自满——低估了莎尔怨恨的深度，她追随者的狂热。
      - 但也许这一切本就是注定的。也许月之少女为你铺设的道路比我们知道的更长。
- _[(继续)](#wishtoaskabout)_
- **影心** : 为什么没有其他人帮助你们？或者来寻找我们？
      - **阿内尔·圣叶:** 也许他们找了——我们的社区很小，人手分散。
      - 我们变得自满——低估了莎尔怨恨的深度，她追随者的狂热。
      - 但也许这一切本就是注定的。也许月之少女为你铺设的道路比我们知道的更长。
- _[(继续)](#wishtoaskabout)_
- **影心** : 这么多年...你们离我这么近。而我却什么都没做。
      - **阿内尔·圣叶:** 你当时什么也做不了——除非把自己置于危险之中。月之少女在她自己的时间里指引着你。
- _[(继续)](#wishtoaskabout)_
- **影心** : 你没生气吧？关于我的名字？
  - **阿内尔·圣叶:** 名字不重要。你的幸福才重要。只是...原谅你父亲偶尔的口误。给我点时间，我会适应的。
- _[(继续)](#wishtoaskabout)_
- **影心** : 如果你是狼人，那我也是吗？
  - **阿内尔·圣叶:** 嗯，这要看情况——你是否在某些时候发现自己毛发更浓密，渴望温热的血肉？其实不用——别让你父亲知道答案。
  - 嗯，这要看情况——你是否在某些时候发现自己毛发更浓密，渴望温热的血肉？其实不用——别让你父亲知道答案。
  - 我是狼人，但你母亲不是。你是否继承我这一面，就像抛硬币一样。
  - 如果你到现在都没有变化，我相信你是安全的。
- _[(继续)](#wishtoaskabout)_
- **影心** : 我们接下来该怎么办？
  - **阿内尔·圣叶:** 我听了足够多关于这个“绝对”威胁的事，明白它不容忽视。你母亲和我可以等到你胜利——我知道你会的。
  - 在那之后，你母亲需要我的照顾——如果你愿意加入我们，也需要你的照顾。
  - 在某个安静、靠近自然的地方。那里阳光能温暖我们，月光能在我们入睡时守护我们。
- **影心** : 你答应过会告诉我我对你们俩做了什么。我现在准备好了。
  - **阿内尔·圣叶:** 你确定吗，影心...？揭开旧伤疤可能没什么意义...
- **影心** : 拜托。我需要理解，才能继续前进。
- _[(与相同回应)](#practice)_
- **影心** : 这些伤疤并不旧——它们从未被允许愈合。我必须知道。
- _[(与相同回应)](#practice)_
- **影心** : 他们让我对你们俩做了那些事，这么多年。拜托——我需要知道。
  - **阿内尔·圣叶:** 很好...但我不会沉溺于丑陋的细节。那正是他们想要的——让他们的肉体折磨活在我们的精神里。我不会允许那样。
  - 他们让你用我们来...练习。所有莎尔教徒引以为豪的技巧。诡计、审讯...酷刑。他们甚至让你，我们自己的女儿，试图转化我们。
  - 最糟糕的不是你对我们做的任何事——而是他们对你做的事。你认不出我们了。那双我曾擦拭过泪水的眼睛，现在看我就像看一个陌生人。
  - 我...我那时差点就屈服了，但你母亲的力量支撑着我。无论你说什么或做什么，她从未放弃叫你女儿。
  - 在那个房间里，我们仍然是一家人。受苦。忍受。但我们没有放弃。你母亲和我发誓，我们会再次看到你眼中认出我们的神情。而我们做到了。
- **影心** : 对不起。
      - **阿内尔·圣叶:** 别道歉。知道你还在外面，给了我们坚持下去的力量。而你回到了我们身边，就像很久以前你本该做的那样。
- _[返回节点](#practice)_
- **影心** : 你们俩是怎么坚持下来的？我永远不会有那种力量...
      - **阿内尔·圣叶:** 知道你还在外面，就足以支撑我们。而你比我们俩都坚强，女儿。我们只是不得不忍受——而你必须选择你的道路。一条最终把你带回我们身边的道路。
- _[返回节点](#practice)_
- **影心** : 我手上的伤呢？是为了伤害我，还是伤害你们？
      - **阿内尔·圣叶:** 两者都有。他们在我们眼前给你打上标记。他们说这是为了惩罚你偏离正道，同时也延长我们的折磨。
      - 不仅是通过痛苦，还通过让我们知道，每次我们感到疼痛时，你真实的自我都在因试图打破他们的灌输而受罚。
- _[返回节点](#practice)_
- **影心** : 我听够了。
      - **阿内尔·圣叶:** 是的。现在你知道所有值得知道的事了。让我们把这件事永远抛在脑后吧。

- 离开。
- _(对话结束)_
