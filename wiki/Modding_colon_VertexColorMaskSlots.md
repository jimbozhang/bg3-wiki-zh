# 顶点颜色遮罩槽

在创建服装/护甲/头盔等时，我们有时希望隐藏身体的某些部分，或者让这些部分不被隐藏，例如希望头饰不隐藏头发。

您会在合并的.lsf文件中看到类似这样的内容：

`<node id="VertexColorMaskSlots"> <attribute id="Object" type="FixedString" value="Torso" /> </node>`

您可以使用的值如下：（注意：这些值区分大小写）

## 顶点颜色遮罩槽表

| 固定字符串 | 描述 | 颜色十六进制代码 |
| --- | --- | --- |
| Torso | 隐藏躯干的大部分，用于大多数身体护甲 | #013201 |
| Shoulders |  | #011E01 |
| upperarm |  | #014101 |
| lowerarm |  | #015001 |
| wrists | 通常用于手套 | #013A01 |
| hands | 通常用于手套 | #015F01 |
| Thighs | 使用时要小心，因为某些较短的裤子在使用时会出现间隙 | #01B901 |
| knees |  | #011001 |
| shins |  | #012401 |
| feet | 通常用于靴子 | #019601 |
| decolletage_01 | 隐藏服装的低领口 | #018201 |
| decolletage_02 | 隐藏服装的第二个低领口 | #016E01 |
| Underwear_Bra | 隐藏上身内衣 | #01A001 |
| Underwear_Panties | 隐藏下身内衣 | #01AA01 |
| Underwear_Panties_Tail | 同上，但我认为是专门用于隐藏提夫林内衣的 | #01AA01 |
| Private_Parts | 隐藏生殖器 | #022902 |
| ModestyLeaf | 在游戏未开启全裸时隐藏叶子 | ? |
| Nipple Covers | 在游戏未开启全裸时隐藏乳头盖 | ? |

## 隐藏网格部分的其他方法

游戏中某些头盔会隐藏耳朵/胡须/头部。

首先，这是由gr2完成的，因此如果您不小心将头部的任何部分命名为Ears/Head/Beard（除了预期的网格本身），这些头盔也会隐藏它们。

如果您想有意这样做，请确保在Blender中网格的数据选项卡和对象选项卡中具有正确的名称。

请记住将这些标签放在visualbank中的node id="base"标签中，这与“hide hair”的位置不同。

`<node id="base"> <children> <node id="Tags"> <attribute id="Object" type="FixedString" value="Hide Head/Beard/Ears" />` 在visualbank中隐藏头部部分，就像顶点遮罩槽一样。
