# 绝望 (状态)

**绝望**

- 来自堕影冥界深处的绝望感已渗透你的存在。

## 属性

[堆叠ID](Stack_ID.md "Stack ID"): `DESPAIR` [堆叠优先级](Stack_priority.md "Stack priority"): 0

## 状态：绝望：冷漠

| 状态 | 效果 |
| --- | --- |
| 绝望 | 来自堕影冥界深处的绝望感已渗透你的存在。 |
| [绝望：冷漠](Despair_colon__Apathy_(Condition).md "绝望：冷漠 (状态)") | 使用敏捷的[属性检定](Ability_checks.md "属性检定")和[豁免检定](Saving_throws.md "豁免检定")具有[劣势](Disadvantage.md "劣势")。 |
| [绝望：恐惧](Despair_colon__Dread_(Condition).md "绝望：恐惧 (状态)") | 所有[属性检定](Ability_checks.md "属性检定")和[豁免检定](Saving_throws.md "豁免检定")具有[劣势](Disadvantage.md "劣势")。 |
| [绝望：疯狂](Despair_colon__Madness_(Condition).md "绝望：疯狂 (状态)") | 你被绝望的疯狂所折磨，对敏捷[检定](Checks.md "检定")以及感知、智力、魅力和敏捷[豁免检定](Saving_throws.md "豁免检定")施加[劣势](Disadvantage.md "劣势")。 |

持续时间：2 驱散

- 使用敏捷的[属性检定](Ability_checks.md "属性检定")和[豁免检定](Saving_throws.md "豁免检定")具有[劣势](Disadvantage.md "劣势")。
