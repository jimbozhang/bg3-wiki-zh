## 出场

### 第二幕

馨妲姐妹在治疗中心的前台工作，并在队伍进入时与他们交谈。她认为队伍“不舒服”，但足够在一条不存在的队伍中等待。

队伍可以通过通过检定来说服她，他们的疾病足够重要，需要立即处理：

- 我没时间等——我现在就要看医生。
  - [牧师] [欺瞒] 我不是病人——我是被雇来的新治疗者。(DC 16 )
  - [武僧] [欺瞒] 我不是病人——我是被雇来的灵性顾问。(DC 16 )
  - [法师] [欺瞒] 我不是病人——我刚被聘为适应不良死灵病痛的专家。(DC 16 )
  - [游荡者] [欺瞒] 我不是病人，你的外科医生听说我有费伦大陆这一侧最稳的手。(DC 16 )
  - [欺瞒] 我不是病人——我是医生的新助手。(DC 16)
- [游荡者] [表演] _模仿_刺客之血_这种知名毒药的症状。_(DC 16 )
- [吟游诗人] [表演] _假装患上某种最可怕的疾病。_(DC 16 )
- [欺瞒] _假装生病。_(DC 16)
  - [游说] 等等！我的伤口可能在外面看不见，但我仍然需要帮助。(DC 16)
  - [游侠] [游说] 你能想象到的每一种怪物，人或野兽，都试图杀死我。我真的该看看医生。(DC 16 )
  - [术士] [威吓] 你可以让我进去，或者我烧掉、电击或冻结这个地方——也许三样都来。你选。(DC 16 )
- [野蛮人] [威吓] _**对这种不公怒吼。**_(DC 16 )
  - [古神] [欺瞒] 我得了一种深沉的病痛，来自星辰之外的地方。(DC 14)
  - [至高妖精] [欺瞒] 我身上长出了奇怪的、会唱歌的荨麻疹——这是精灵感染的明确迹象。(DC 14)
  - [邪魔] [欺瞒] 我染上了一种恶魔般的疾病——症状相当地狱。(DC 14)
  - [魔契刃] [欺瞒] 我被一把异界武器割伤——我想伤口感染了。(DC 14)
- [阿斯代伦] 亲爱的女士——看看我的脸色。我已命悬一线。_(自动通过)_

如果通过检定，辛妲会指引队伍去找医生，[马鲁斯·索姆](Malus_Thorm.md "马鲁斯·索姆")。否则，她会坚持让队伍“排队等候”，阻止他们进入她身后的门厅。队伍可以通过从任何侧门进入建筑，或者穿过馨妲姐妹右侧的门（通往外部一条通往儿童病房的短路径）来绕过这一点。儿童病房可以通过打破南侧被封锁的门进入，或者使用通往屋顶的大藤蔓坡道到达北侧。或者，可以杀死她，或由[弃誓者圣武士](Paladin.md#Becoming_an_Oathbreaker "圣武士")[控制不死生物](Control_Undead.md "控制不死生物")；后者在效果激活时让队伍获得她强大的[心灵尖啸](Mind_Screech.md "心灵尖啸")能力。

如果选择[邪念](Dark_Urge.md "邪念")作为[出身](Origin.md "出身"), 则会出现独特的互动。他们可以坚持自己有问题，并允许馨妲姐妹检查。事实上，她注意到了邪念心智的损伤，并允许队伍在不进行[技能检定](Skill_Check.md "技能检定")的情况下前往马鲁斯·索姆。这也是确认邪念历史中一个关键事件的事件之一：它所遭受的失忆是实际颅脑创伤的结果，与[夺心魔](Mind_flayer.md "夺心魔")蝌蚪无关。

如果队伍在未与辛妲交战的情况下击败[马鲁斯·索姆](Malus_Thorm.md "马鲁斯·索姆")，返回她那里会揭示一条新的对话线：_“医生今天不在。改天再来。”_
