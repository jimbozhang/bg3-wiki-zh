# 失能 (状态)

**失能**

- 无法采取 [动作](Action.md "Action") 或 [反应](Reaction.md "Reaction")。
- 如果正在维持专注法术，则失去 [专注](Concentration.md "Concentration")。

| 状态 | 效果 |
| --- | --- |
| 失能 | 无法采取 [动作](Action.md "Action") 或 [反应](Reaction.md "Reaction")。如果正在维持专注法术，则失去 [专注](Concentration.md "Concentration")。 |

## 属性

[堆叠 ID](Stack_ID.md "Stack ID"): `INCAPACITATED_DND` [状态组](Status_groups.md "Status groups"): [SG_Incapacitated](SG_Incapacitated.md "SG Incapacitated"), [SG_Condition](SG_Condition.md "SG Condition")

## 备注

- 受影响实体无法执行*任何*动作，包括 [动作](Action.md "Action")、[附赠动作](Bonus_action.md "Bonus Action") 或 [反应](Reaction.md "Reaction")。
- 失能实体无法移动。
