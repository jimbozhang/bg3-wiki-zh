# Modding:打包模组

此页面为 [模组页面](Modding_colon_Modding_resources.md "Modding:模组资源")，并遵循其自身规则和标准，与Wiki其余部分不同。

此页面解释如何打包（pak）您的模组，并使其准备好导入到您选择的模组管理器中。

## 使用 Multitool

1. 将您的文件夹拖放到 Multitool 中的蓝色方块中。
1. 您的模组现在应该已在原始文件夹的同一位置被打包并压缩。您已完成！

## 使用 LSLIB

1. 确保程序顶部的游戏设置为 Baldur's Gate 3。
1. 在 PAK / LSV Tools 选项卡中，专注于 Create Package 部分。
1. 在 Source path 中填写您要打包的文件夹的位置。
1. 在 Package path 中填写您的模组文件夹的位置，或您要打包模组到的位置。
1. 将您的版本设置为最新版本（Baldur's Gate 3 Release）。
1. 点击 Create Package。
