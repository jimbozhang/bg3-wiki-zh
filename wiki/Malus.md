## 概述

生前，马鲁斯将治疗中心的宗旨从照料雷斯文民众中的病患和伤者，转变为专门服务于[凯瑟里克·索姆](Ketheric_Thorm.md "凯瑟里克·索姆")军队的[暗夜法官](Dark_Justiciar.md "暗夜法官")，因为他相信保持后者的健康有助于他们侍奉莎尔，而其他人则最好任其湮灭。<sup>[\[3\]](#cite_note-3)</sup><sup>[\[4\]](#cite_note-4)</sup>

尽管身处[幽影诅咒之地](Shadow-Cursed_Lands.md "幽影诅咒之地")，他不知何故保留了人格，他相信优雅的痛苦和精心计算的残害是真正拥抱其黑暗信仰理想的关键。在他那堕落的医院里，他孜孜不倦地训练他的[不死生物](Undead.md "不死生物")护士磨练此类技能，然而她们空洞的状态使她们除了尽职的助手外别无他能。

与马鲁斯·索姆以及凯瑟里克·索姆的其他家庭成员互动，会为拥有[智者](Sage.md "智者")背景的队伍成员授予激励点[索姆家族的秘密](Sage.md#Thorm_Family_Secrets "智者")。

## 参与

抵达[治疗中心](House_of_Healing.md "治疗中心")并进入手术室后，可以发现马鲁斯正在一个可怜的“病人”身上教育他的亡灵护士。他拥有[破旧鲁特琴](Battered_Lute.md "破旧鲁特琴")，这是解决[唤醒阿尔特·库拉](Wake_up_Art_Cullagh.md "唤醒阿尔特·库拉")所必需的。起初他彬彬有礼，将会说话的队伍成员视为新学生，宣扬他关于“虚无”是理解莎尔之道的观点。在残害了“病人”后，他敦促对队伍也做同样的事。除了攻击之外，还可以进行一系列[技能检定](Skill_Check.md "技能检定")：

- [调查] _检查姐妹们及其工具。_ (DC 14)
  - 她们的刀具不均匀。高效的手术需要进一步训练。
- [游荡者] [调查] _审视姐妹们的刀具。_ (DC 14 )
  - 你的刀具状况极差——用这些根本无法干净利落地切割。
- [宗教] _回想你所知的莎尔教义。_ (DC 14) (非牧师)
- [牧师] [宗教] _回想你所知的莎尔亵渎的教义。_ (DC 14 ) _(仅限非莎尔牧师)_ (为拥有[侍僧](Acolyte.md "侍僧")背景的队伍成员授予[完成作业](Acolyte.md#Did_My_Homework "侍僧"))
- [游说] 姐妹们还没准备好。她们会让我恶心而不是治愈我。(DC 16)

如果任何检定成功，**马鲁斯**会询问如何正确训练护士，这会导致两个不同的游说检定：

- [游说] 何不让她们在彼此身上磨练技能？(DC 21)
- [战士] [游说] 对练对进步至关重要。让她们在彼此身上练习。(DC 21 )
- [游荡者] [游说] 练习——我就是这样精通刀法的。她们应该在彼此身上对练。(DC 21 )
- [法师] [游说] 在我多年的学习中，我发现没有比榜样更好的老师了。也许_你_可以自愿？(DC 21 )
- [武僧] [游说] 当拳头颤抖时，有时需要另一个人来稳住它。一个好的大师会自愿献身。(DC 21 )
- [吟游诗人] [游说] 我乐于与我技艺的大师一起表演——我相信姐妹们也一样。何不_你_自愿？(DC 21 )
- [游说] 她们需要一个更好的练习对象。不是学生，而是大师。(DC 21)

如果队伍阅读了**马鲁斯**身后房间里的_[外科手术与生理学：莎尔信徒入门](Surgery_and_Physiology_colon__A_Sharran's_Primer.md "外科手术与生理学：莎尔信徒入门")_一书，则会提供两个额外选项，两者都会导致护士杀死**马鲁斯**：

- [威吓] 我读过你的论文。开始最后一课，否则就死在我手上。(DC 18)
- [游说] “唯有牺牲方能精通。”为了让姐妹们学习，你必须屈服。(DC 18)

如果护士被说服互相战斗，拥有[公会工匠](Guild_Artisan.md "公会工匠")背景的队伍成员会获得激励点[不人道的试验](Guild_Artisan.md#Inhumane_Trials "公会工匠")，而疯狂的外科医生会敦促接下来照料队伍。可以攻击他，在没有护士支持的情况下会更容易击败。或者，可以说服他自杀：

- [吟游诗人] [游说] 我不配！请向我展示大师是如何工作的——我将深感荣幸。(DC 21 )
- [牧师] [欺瞒] 你给了我荣誉，但我不配。向我展示莎尔之道——我恳求你。(DC 21 )
- [战士] [欺瞒] 在大师剑客面前，我只是个笨拙的侍从。请示范一下，让我学习...(DC 21 )
- [法师] [术士] [欺瞒] 我的魔法蒙蔽了我——我现在明白莎尔是唯一的道路。请向我展示如何——我恳求你。(DC 21 )
- [德鲁伊] [欺瞒] 我在自然世界中寻求意义——我现在明白莎尔是唯一的道路。请向我展示如何——我恳求你。(DC 21 )
- [游说] 我宁愿亲自尝试——如果你向我展示如何做。(DC 21)

被队伍的勤勉所折服，**马鲁斯**高兴地以自己为例，刺穿自己的眼睛并自杀。

成功让他自杀会授予以下激励点：<sup>[\[5\]](#cite_note-5)</sup>
