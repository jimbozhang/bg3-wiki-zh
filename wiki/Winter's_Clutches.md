# 寒冬之握

对于被动特性，请参见 [寒冬之握（被动特性）](Winter's_Clutches_(passive_feature)..md)

寒冬之握是一双[手套](Gloves.md "手套")，属于不常见物品。当穿戴者造成[寒冷](Cold.md "寒冷")伤害时，可对敌人施加[冷冻](Encrusted_with_Frost_(Condition)状态.md)。

这些手套很可能源自[冰风谷](https://forgottenrealms.fandom.com/wiki/Icewind_Dale)，在那里，脾气如同炉火般炽热，冰冻的河流以蓝白色的支流网络在山谷臂弯中蜿蜒。

## 属性

- [手套](Gloves.md "手套")
- 稀有度：不常见
- 重量：0.5 千克 (1 磅)
- 价格：90 金币 / 170 金币（荣誉模式）
- UID `MAG_Frost_GenerateFrostOnDamage_Gloves` UUID `e6bb26ab-2987-4061-9c5c-764b377f91b8` Stats `MAG_Frost_GenerateFrostOnDamage_Gloves` File `ARM_Gloves_Leather_F` ### 特殊

穿戴此物品者获得：

[寒冬之握](Winter's_Clutches_(passive_feature).md)
当穿戴者造成[寒冷](Cold.md "寒冷")伤害时，对目标施加 2 回合的[冷冻](Encrusted_with_Frost_(Condition).md "冷冻（状态）")状态。

- [瑰晨修道院小径](Rosymorn_Monastery_Trail.md "瑰晨修道院小径") X: -43 Y: -129：由[埃斯特女士](Esther.md "埃斯特女士")出售，位于特瑞尔塔峭壁传送点东北方
- [巫术杂物店](Sorcerous_Sundries.md "巫术杂物店") X: -2 Y: -66：在入口附近的[岩浆元素](Lava_Elemental.md "岩浆元素")和[水元素](Water_Elemental.md "水元素")后方的柜台上
