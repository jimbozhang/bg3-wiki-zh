# 巴尔之恩赐

| | |
| --- | --- |
| 体型 | 中级 |

## 参与

当奥林未被一对一对决时，它将与[奥林](Orin.md "奥林")一同加入战斗。巴尔之恩赐总是先攻顺序的最后行动。

## 战斗

### 攻击与能力

[不可撼动之恩赐](Boon_of_the_Unstoppable.md "不可撼动之恩赐")
巴尔以可怕的力量灌注你，使你[不可撼动](Unstoppable_(Condition).md "不可撼动（状态）")。

射程：50米（167英尺）

### 荣誉模式

[巴尔之谕](Bhaal's_Edict.md "巴尔之谕")
强制一个生物成为你的谋杀者——如果它在下一个回合未杀死另一个生物，它将立即死亡。
