# 骨刺靴

骨刺靴是一双[非常稀有](Footwear.md "Footwear")的[靴子](Footwear.md "Footwear")，能增强跳跃距离并让角色获得[凶蛮跳跃](Brutal_Leap.md "Brutal Leap")能力。

专为标准野蛮人设计的护甲，适用于空气动力学般的跳跃、打击以及普遍的咆哮。

## 属性

- [靴子](Boots.md "Boots")
- 稀有度：非常稀有
- 重量：0.55 kg (1.1 lb)
- 价格：70 gp
- UID `ARM_Shoes_Barbarian_Magic_B` UUID `23806fd3-f5f6-4ff3-826d-054afe03b7c0` Stats `MAG_Barbarian_BoneSpike_Shoes` File `ARM_BarbarianMagical_B_Footwear` ### 特殊

此物品的穿戴者获得：

[凶蛮跳跃](Brutal_Leap.md "Brutal Leap") (+ )
跃向一个目标，并可能使其[倒伏](Prone_(Condition).md "Prone (Condition)")。（充能：每回合。）

[直觉闪避](Evasive_Instinct.md "Evasive Instinct")
只要你不穿戴[护甲](Armour.md "Armour")或手持[盾牌](Shields.md "Shields")，你的[护甲等级](Armour_Class.md "Armour Class")和[豁免检定](Saving_throw.md "Saving throw")获得+1加值。

[灵巧跳跃](Refined_Vaulting.md "Refined Vaulting")
[跳跃](Jump.md "Jump")距离增加1.5米（5英尺）。
