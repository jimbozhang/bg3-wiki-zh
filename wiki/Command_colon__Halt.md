# 命令：定身

**命令：定身**是一个[1级惑控学派法术](Spells.md "Spells")。该法术是[命令术](Command.md "Command")的一个变体。它允许施法者命令目标暂时无法移动或使用[动作](Action.md "Action")、[附赠动作](Bonus_action.md "Bonus Action")或[反应](Reaction.md "Reaction")。

## 描述

命令一个生物定身，阻止其移动或进行任何类型的行动。

## 属性

消耗
[动作](Actions.md#Resources "动作") + [1级法术位](Spells.md#Spell_slots "法术")
详情
[感知](Wisdom.md "WIS") [豁免检定](Saving_throws.md "Saving throws")（[法术豁免DC](Dice_rolls.md#Spell_save_DC "骰子检定")）
射程：18米（60英尺）

## 升环施法

[升环施法](Upcasting.md "Upcasting")：每比1环高1环，影响一个额外目标。
