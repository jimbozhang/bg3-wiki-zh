# 病弱 (状态)

| 状态 | 效果 |
| --- | --- |
| [刺客之血](Assassin's_Blood_(Condition).md "Assassin's Blood (Condition)") | 生物必须通过一次体质[豁免检定](Saving_throw.md "Saving Throw")，否则受到1d12⁠⁠[中毒](Poison.md "Poison")伤害，并在下一次长休前处于[中毒](Poisoned_(Condition).md "Poisoned (Condition)")状态。即使豁免成功，仍会受到一半伤害。 |
| [有毒烟雾](Noxious_Fumes_(Bibberbang)_(Condition).md "Noxious Fumes (Bibberbang) (Condition)") | 受影响实体每回合受到1d4⁠⁠[中毒](Poison.md "Poison")伤害。必须通过一次[DC](Dice_rolls#Save_DCs.md#Save_DCs "Dice rolls") 15的[体质](Constitution.md "Constitution")[豁免检定](Saving_throw.md "Saving Throw")，否则额外受到2d4⁠⁠[中毒](Poison.md "Poison")伤害，豁免成功时伤害减半。 |
| [中毒](Poisoned_(Condition).md "Poisoned (Condition)") | 在[攻击掷骰](Attack_roll.md "Attack Roll")和[属性检定](Checks.md "Checks")上承受[劣势](Disadvantage.md "Disadvantage")。 |
| [毒雾](Poisonous_Fumes_(Condition).md "Poisonous Fumes (Condition)") | 每回合受到1d4⁠⁠[中毒](Poison.md "Poison")伤害。在[攻击掷骰](Attack_rolls.md "Attack Rolls")和[属性检定](Checks.md "Checks")上承受[劣势](Disadvantage.md "Disadvantage")。 |
| 病弱 | 感觉极度不适——可疑地不适。 |
