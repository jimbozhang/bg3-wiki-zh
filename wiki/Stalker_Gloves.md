# 追猎者手套

追猎者手套是一双稀有[手套](Handwear.md "Handwear")，能提升穿戴者的[偷袭](Sneak_Attack.md "Sneak Attack")和[先攻](Initiative.md "Initiative")。

作为一名现代游荡者，生活十分艰难——每个人都预料到会遭到突袭。这些手套能确保被突袭者来不及决定自己是否受惊就一命呜呼。

## 属性

- [手套](Gloves.md "Gloves")
- 稀有度：稀有
- 重量：0.5 千克（1 磅）
- 价格：400 金币 / 580 金币（荣誉模式）
- UID `MAG_Stalker_Gloves` UUID `dd99248c-fe42-4d07-9861-853e9291ea51` Stats `MAG_Stalker_Gloves` File `ARM_Gloves_Leather_F` ### 特殊

穿戴此物品获得：

[鲜少措手不及](Seldom_Caught_Unawares_(+1).md)
你获得 +1 的[先攻掷骰](Initiative.md "Initiative")加值。

[阴招](Skullduggery_Attack.md "Skullduggery Attack")
你的[偷袭](Sneak_Attack.md "Sneak Attack")造成额外 1d4⁠⁠[力场](Force.md "Force")[DRS](Damage_rider_as_source.md "Damage rider as source")伤害。

## 获取地点

- [利文顿杂货店](Rivington_General.md "Rivington General") X: 7 Y: -35：在[第三幕](Act_Three.md "Act Three")期间，由[利文顿](Rivington.md "Rivington")的[埃克维凯普](Exxvikyap.md "Exxvikyap")出售

## 备注

- 尽管[先攻](Initiative.md "Initiative")加值对所有角色都有帮助，但这些手套对[游荡者](Rogue.md "Rogue")更具价值，因为这是唯一能使用[偷袭](Sneak_Attack.md "Sneak Attack")的职业。

## 错误

_关于阴招：_

- **阴招**特性无法与[潇洒偷袭](Rakish_Sneak_Attack.md "Rakish Sneak Attack")配合使用，除非偷袭是作为反应使用。
