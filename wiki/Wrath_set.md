# 怒火套装

| 名称 | 部位 | 效果 |
| --- | --- | --- |
| 狂怒头盔 | [帽子](Headgear.md "Headgear") ([非护甲](Headwear#Non-Armour.md#Non-Armour "Headwear")) | [背水一战](Last_Stand.md "Last_Stand")战斗中，当穿戴者回合开始时生命值低于或等于50%，获得[怒火](Wrath_(Condition).md "Wrath (Condition)")，持续2回合。 |
| 吸血服 | [护甲](Armour.md "Armour") ([服装](Clothing.md "Clothing")) | [以牙还牙](Grievous_Retribution.md "Grievous_Retribution")当敌人对穿戴者造成伤害时，穿戴者获得[怒火](Wrath_(Condition).md "Wrath (Condition)")，持续2回合。 |
| 破阵战靴 | [靴子](Footwear.md "Footwear") ([非护甲](Footwear#Non-Armour.md#Non-Armour "Footwear")) | [兴奋激发](Adrenaline_Rush.md "Adrenaline_Rush")当你[疾走](Dash.md "Dash")时，获得[怒火](Wrath_(Condition).md "Wrath (Condition)")，持续3回合。[[参见：错误](Adrenaline_Rush#Bugs.md#Bugs "Adrenaline_Rush")] |
| 愤怒之心 | [护甲](Armour.md "Armour") ([服装](Clothing.md "Clothing")) | [怒从心生](Ruintamer_Heart.md "Ruintamer_Heart")当[狂暴](Rage_(Condition).md "Rage (Condition)")时，穿戴者产生2回合的[怒火](Wrath_(Condition).md "Wrath (Condition)")。[[参见：错误](Ruintamer_Heart#Bugs.md#Bugs "Ruintamer_Heart")] |

**怒火**套装是一套能够产生[怒火](Wrath_(Condition).md "Wrath (Condition)")的装备，这是一种可叠加的状态，最多可提供+7近战伤害加成。

## 备注

- [野蛮人套装](Barbarian_set.md "Barbarian set")中的[理智之握](Reason's_Grasp.md "Reason's_Grasp")在视觉上与该套装相配。
