# 指南：箱塔

这是一份[社区指南](Guide_colon_Index.md "Guide:Index")，任何人都可以编辑。请参阅[样式手册](Help_colon_Style_manual.md "Help:Style manual")了解指南。
营地中的箱塔

**箱塔**是由[木箱](Wooden_Crate_(regular).md "Wooden Crate (regular)")、[大木箱](Wooden_Trunk.md "Wooden Trunk")和类似可堆叠物品<sup>[\[1\]](#cite_note-1)</sup>建造的结构，用于创建人工抬高的<sup>[\[2\]](#cite_note-2)</sup>平台。它们可以在任何时候搭建，包括在战斗中，因为放置物品不会消耗动作。箱塔可用于：

- 让[伙伴](Companions.md "Companions")和/或[雇佣兵](Hirelings.md "Hirelings")跳向他们的[死亡](Dead_(Condition).md "Dead (Condition)")，如果需要的话<sup>[\[3\]](#cite_note-3)</sup>
- 通过[坠落伤害](Falling_damage.md "Falling damage")机制增加掉落物品造成的伤害<sup>[\[4\]](#cite_note-4)</sup>
- 通过足够的距离让使用投掷物品的角色保持在战斗之外<sup>[\[5\]](#cite_note-5)</sup>
- 为计划中的遭遇战做准备，以寻求特定的战术优势：
  - 对[翠绿林地](Emerald_Grove.md "Emerald Grove")的[第一次](Emerald_Grove.md#Emerald_Grove_ramparts "Emerald Grove")或[击败地精](Defeat_the_Goblins.md "击败地精")进攻
  - [莱埃泽尔](Lae'zel.md "Lae'zel")的[浪漫战斗](Lae'zel/Romance.md#Act_Two "Lae'zel/Romance")
  - 在[复仇之炉](Grymforge.md "Grymforge")与[尼讷](Nere.md "Nere")战斗

## 警告

与大多数漏洞利用一样，建造箱塔可能会导致意外结果。已知的潜在问题包括：

- 在特定高度下<sup>[\[6\]](#cite_note-6)</sup>，会出现短暂的图形故障，偶尔会导致现有游戏进程和/或存档软锁定，包括荣誉模式运行。解决此问题可能需要加载早期存档，或者——在[荣誉模式](Honour_Mode.md "Honour mode")运行的情况下——[绕过](Guide_colon_Circumventing_Honour_Mode_Save_File_Restrictions.md "Guide:Circumventing Honour Mode Save File Restrictions")存档文件限制。
- 如果在访问另一个营地时，某个营地留有足够高的箱塔，部分<sup>[\[7\]](#cite_note-7)</sup>箱子会被收集到营地箱子中。如果小队返回搭建箱塔的营地，剩余结构将瞬间坍塌。这不仅会对当时站在其下的任何角色造成大量伤害，而且在其残骸中移动的角色可能会随机<sup>[\[8\]](#cite_note-8)</sup>受到大量钝击伤害，仿佛箱塔再次坍塌一样。这对四处游荡的营地宠物尤其危险。
- 在回合制模式下下塔时，角色偶尔会穿过箱塔故障，掉落到下方地面。
