# 罗丝

| | |
| --- | --- |
| 种族 | [神祇](Deity.md "Deity") |

**罗丝**，又名_蛛后_，是[卓尔](Drow.md "Drow")最具影响力的[女神](Deities.md "Deities")，也是⁠[黑暗席德瑞恩](https://forgottenrealms.fandom.com/wiki/Dark_Seldarine)万神殿的领袖。

> “
>
> 赞美罗丝，所有卓尔之母。她的蛛网束缚着我们的身体与灵魂，并在那些不配者呼吸之前将其扼杀。
>
> „

— [《忠诚教育》](On_Educating_the_Faithful.md "On Educating the Faithful")

## 概述

杀戮、残忍、背叛与绝对崇拜是罗丝所沉醉的方面，这很大程度上解释了卓尔社会。她与[诡术领域](Trickery_Domain.md "Trickery Domain")和[战争领域](War_Domain.md "War Domain")相关联。

任何玩家角色[牧师](Cleric.md "Cleric")都可以选择这位神祇，但[席德瑞恩卓尔](Seldarine_Drow.md "Seldarine Drow")除外。对于[罗丝誓约卓尔](Lolth-Sworn_Drow.md "Lolth-sworn Drow")牧师来说，这是强制性的。
