## 有人知道为什么全队死亡仍然会清除先前荣誉模式运行的状态吗？

如果在全队死亡后替换存档文件，这不仍然是一个没有死亡的存档文件吗？[Ankou](https://bg3.wiki/w/index.php?title=User:Ankou&action=edit&redlink=1 "User:Ankou (页面不存在)") ([讨论](https://bg3.wiki/w/index.php?title=User_talk:Ankou&action=edit&redlink=1 "User talk:Ankou (页面不存在)")) [2024年7月25日 21:20 (CEST)](https://bg3.wiki/wiki/Guide_talk:Circumventing_Honour_Mode_Save_File_Restrictions#c-Ankou-20240725192000-Anyone_know_why_the_whole_party_dying_still_wipes_the_prior_honor_mode_run's_sta)回复

很遗憾，不行。一旦发生全队死亡，游戏会在整体保险柜文件结构中更改标志，以表明从“单存档文件” > “普通游戏”的转变。
这无法通过切换存档文件来更改/还原。[Valk](User_colon_Valk.md "User:Valk") ([讨论](https://bg3.wiki/w/index.php?title=User_talk:Valk&action=edit&redlink=1 "User talk:Valk (页面不存在)")) [2024年9月5日 14:56 (CEST)](https://bg3.wiki/wiki/Guide_talk:Circumventing_Honour_Mode_Save_File_Restrictions#c-Valk-20240905125600-Ankou-20240725192000)回复

那么，情况是否是这样的：只要避免全队死亡，你总是可以重新加载存档（例如，你得到一个坏骰子导致一个伙伴死亡），如果你替换存档文件，那么备份荣誉模式存档文件还是有一些有限的价值？[Ankou](https://bg3.wiki/w/index.php?title=User:Ankou&action=edit&redlink=1 "User:Ankou (页面不存在)") ([讨论](https://bg3.wiki/w/index.php?title=User_talk:Ankou&action=edit&redlink=1 "User talk:Ankou (页面不存在)")) [2024年9月8日 22:29 (CEST)](https://bg3.wiki/wiki/Guide_talk:Circumventing_Honour_Mode_Save_File_Restrictions#c-Ankou-20240908202900-Valk-20240905125600)回复

确实如此。
