# 卷轴

另见：[卷轴（消歧义）](Scroll_(disambiguation).md)>)

**卷轴**是魔法[消耗品](Consumables.md "消耗品")，允许角色在不消耗[法术位](Spells.md#Spell_Slots "法术位")或没有施法职业的情况下施放[法术](Spells.md "法术")。当你没有任何剩余的法术位、需要施放你未准备的法术，或者队伍中没有所需的施法者时，卷轴会非常有用。

## 卷轴的工作原理

博德之门3中的卷轴遵循与第五版龙与地下城规则集不同的规则。以下是BG3中卷轴工作原理的一些要点：

- 任何职业的角色都可以从卷轴施放任何[法术](Spell.md "法术")，即使他们没有施法职业，或者法术的等级高于角色的最高等级法术位。

  - 然而，阻止施法的条件，如[狂暴](Rage.md "狂暴")和[荒野形态](Wild_Shape.md "荒野形态")，仍然适用于卷轴。

| [智力](Intelligence.md "智力") | [感知](Wisdom.md "感知") | [魅力](Charisma.md "魅力") |
| --- | --- | --- |
| [战士](Fighter.md "战士") / [游荡者](Rogue.md "游荡者") / [法师](Wizard.md "法师") / | [牧师](Cleric.md "牧师") / [德鲁伊](Druid.md "德鲁伊") / [武僧](Monk.md "武僧") / [游侠](Ranger.md "游侠") / | [野蛮人](Barbarian.md "野蛮人") / [吟游诗人](Bard.md "吟游诗人") / [圣武士](Paladin.md "圣武士") / [术士](Sorcerer.md "术士") / [邪术师](Warlock.md "邪术师") / |

- 在[身兼多职](Multiclassing.md "身兼多职")的情况下，卷轴的施法关键属性由最近添加的职业决定（角色最近达到1级的职业）。战士和游荡者是此规则的例外，因为这两个职业没有定义施法关键属性，但在单职业时使用默认属性智力。当与这两个职业兼职时，在确定施法关键属性时会忽略它们。

  - 然而，如果这两个职业在3级时选择了[奥法骑士](Eldritch_Knight.md "奥法骑士")或[诡术师](Arcane_Trickster.md "诡术师")子职业，它们会获得适当的施法关键属性（仍然是智力）。例如：

    - 假设一个角色有5级牧师，因此他们的施法关键属性是感知。如果他们兼职并获得3级邪术师，那么他们使用卷轴时的施法关键属性变为魅力，因为邪术师是最近添加的。

    - 假设一个角色有5级牧师，然后兼职并获得1级游荡者。在这种情况下，卷轴的施法关键属性仍然是感知，因为游荡者最初不提供施法关键属性。如果在达到游荡者3级时选择了诡术师子职业，那么卷轴的施法关键属性会变为智力，因为诡术师为游荡者提供了施法关键属性。

      - 请注意，最近的职业仍然是由哪个职业最近达到1级决定的。因此，如果一个5级牧师获得1级游荡者，然后获得1级术士，他们的卷轴施法关键属性是魅力。即使他们随后再获得两级游荡者并成为诡术师，他们的施法关键属性仍然是魅力，因为他们最近达到1级的是术士，而不是游荡者。

- **抄录卷轴：** 法师如果找到一个包含他们不知道的[法师法术](List_of_Wizard_spells.md "法师法术列表")的卷轴，可以永久复制（抄录）该法术到他们的法术书上，每环法术花费50金币；这也会消耗卷轴。法师可以抄录卷轴，将法术添加到他们的法术书中。

  - 对于与法师子职业属于同一学派的法术，费用降低至每环法术25金币（*子职业选项见[法师](Wizard.md#Level_2 "法师")*）。[剑咏者](Bladesinging.md "剑咏者")没有任何折扣。

  - 如果角色改变职业，不再是法师但后来又成为法师，之前从卷轴学会的法术仍然会被记住。只要角色至少有一级法师等级，他们就可以访问所有已学会的法术。

  - 与D&D不同，卷轴无法被创造。

  - 如果队伍成员试图抄录一个等级过高的法师法术卷轴，它会说他们的法师等级不够高。这是误导性的，因为要抄录卷轴，唯一的要求是角色的**综合施法者等级**足够高以施放该法术（即他们必须至少有一个该法术等级的法术位）。角色只需要一级法师就可以抄录卷轴上包含的任何法师法术。例如，一个*1级法师/2级吟游诗人*的角色有2环法术位，因此可以抄录2环法师法术[侦测思想](Detect_Thoughts.md "侦测思想")（尽管在这种情况下，他们在升级时无法从任何一个职业学会任何2环法术）。

    - 在考虑综合施法者等级和抄录卷轴时，记住哪些[施法职业](Spellcasting_classes.md "施法职业")是完整施法者、半施法者和三分之一施法者可能会有所帮助。

- 用于施放具有不同变体的法术的卷轴，例如[繁彩球](Chromatic_Orb.md "繁彩球")的各种元素，不能直接从角色的物品栏中使用，必须先放在快捷栏上。

- 如果被[法术反制](Counterspell.md "法术反制")阻挡，卷轴不会被消耗。

- 购买卷轴的等级要求如下：

- **角色等级1**：1环和2环法术（以及[死者交谈](Speak_with_Dead.md "死者交谈")，这是一个3环法术）。<sup>[\[1\]](#cite_note-1)</sup><sup>[\[2\]](#cite_note-2)</sup>

- **角色等级3**：3环法术

- **角色等级5**：4环法术

- **角色等级7**：5环法术

- **角色等级9**：6环法术

- [术士](Sorcerer.md "术士")可以对卷轴应用超魔，其工作方式与通常的超魔略有不同：

  - 具有子法术的法术卷轴（如[繁彩球](Chromatic_Orb.md "繁彩球")）会错误地报告其术法点成本是子法术数量的倍数。例如，对繁彩球卷轴应用[成双法术](Metamagic_colon__Twinned_Spell.md "超魔：成双法术")会报告需要7点术法点，而不是1点。在应用超魔之前先打开子法术菜单将允许使用它。

  - 容器内的卷轴无法应用超魔；它们必须在施法者的主要物品栏中。
