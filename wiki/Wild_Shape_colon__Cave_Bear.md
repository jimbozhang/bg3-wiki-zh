# 荒野形态：穴熊

注意：请勿与 [荒野形态：熊](Wild_Shape_colon__Bear.md "荒野形态：熊") 混淆

**荒野形态：穴熊**是一种特殊的 [荒野形态](Wild_Shape.md "荒野形态") 形态，仅 [哈尔辛](Halsin.md "哈尔辛") 可用。

## 描述

| [力量](Strength.md "力量") 20 (+5) | [敏捷](Dexterity.md "敏捷") 10 | [体质](Constitution.md "体质") 16 (+3) |
| --- | --- | --- |
|  |  |  |
| 生命值 | 30 |  |
| 护甲等级 | 10 |  |
| 体型 | 大型 |  |

化身为穴熊形态，可进行 [多重攻击](Multiattack_(Cave_Bear_Wild_Shape).md "多重攻击 (穴熊荒野形态)")。穴熊的初始生命值为36点，每提升两级德鲁伊等级会增加生命值。

## 属性

消耗：
[附赠动作](Actions.md#Resources "动作") + [荒野形态充能](Druid.md#Level_2 "德鲁伊")

## 更高等级

在德鲁伊等级4、6、8、10或12时，生命值分别提升至45、57、72、92或117点。

## 状态：荒野形态：穴熊

**[荒野形态：穴熊](Wild_Shape_colon__Cave_Bear_(Condition).md "荒野形态：穴熊 (状态)")**

- 已变形为穴熊

### 生物：穴熊

| [力量](Strength.md "力量") 20 (+5) | [敏捷](Dexterity.md "敏捷") 10 | [体质](Constitution.md "体质") 16 (+3) |

用致命的爪击进行攻击。

[解除变形](Dismiss_Wild_Shape.md "解除变形")
解除你的荒野形态并变回原始形态。

[月光修复](Lunar_Mend.md "月光修复") (+ )
消耗 [法术位](Spells.md#Spell_slots "法术") 以在荒野形态下恢复 [生命值](Hit_Points.md "生命值")。每消耗一个法术位，恢复1d8点[治疗](Healing.md "治疗")。

[多重攻击](Multiattack_(Cave_Bear_Wild_Shape).md "多重攻击 (穴熊荒野形态)")
对目标发动多次攻击。（重置：每场战斗。）

## 如何习得

由以下生物使用：[哈尔辛](Halsin.md "哈尔辛")

## 备注

- 仅哈尔辛可用；他作为职业动作永久知晓此技能，但除非拥有 [荒野形态](Wild_Shape.md "荒野形态") 充能，否则无法使用。
  - 通常需要2级以上 [德鲁伊](Druid.md "德鲁伊") 等级，但 [变形生物帽子](Shapeshifter_Hat.md "变形生物帽子") 提供的 [荒野形态充能](Druid.md#Level_2 "德鲁伊") 同样适用，允许任何构建激活此技能。
- 由于此技能_始终_需要附赠动作激活，[大地结社](Circle_of_the_Land.md "大地结社") 或 [孢子结社](Circle_of_the_Spores.md "孢子结社") 德鲁伊将能够以 [动作](Actions.md#Resources "动作")（用于常规列表，_不包括_穴熊）或 [附赠动作](Actions.md#Resources "动作")（_仅_用于穴熊）进行荒野形态变形。
