# 劣势地形 (状态)

**劣势地形**

- [移动速度](Movement_speed.md "移动速度") 减半

## 属性

[堆叠 ID](Stack_ID.md "堆叠 ID"): `DIFFICULT_TERRAIN` [状态组](Status_groups.md "状态组"): [SG_Surface](SG_Surface.md "SG Surface"), [SG_DifficultTerrain](SG_DifficultTerrain.md "SG DifficultTerrain")

[更多属性](Status_properties.md "状态属性"):

- [禁用战斗日志](Status_properties/DisableCombatlog.md "状态属性/禁用战斗日志")
- [禁用头顶显示](Status_properties/DisableOverhead.md "状态属性/禁用头顶显示")

## 备注

- 移动速度并未实际减半，而是在劣势地形中移动时，移动消耗加倍。生物必须花费 2 米（7 英尺）的移动距离才能在 **劣势地形** 区域中移动 1 米（3 英尺）。
| 状态 | 效果 |
| --- | --- |
| 劣势地形 | [移动速度](Movement_speed.md "移动速度") 减半 |
| [劣势地形：深水](Difficult_Terrain_colon__Deep_Water_(Condition).md "劣势地形：深水 (状态)") | [移动速度](Movement_speed.md "移动速度") 减半。 |
| [劣势地形：熔岩](Difficult_Terrain_colon__Lava_(Condition).md "劣势地形：熔岩 (状态)") | [移动速度](Movement_speed.md "移动速度") 减半。 |
| [劣势地形：泥巴](Difficult_Terrain_colon__Mud_(Condition).md "劣势地形：泥巴 (状态)") | [移动速度](Movement_speed.md "移动速度") 减半。 |
| [劣势地形：藤蔓](Difficult_Terrain_colon__Vines_(Condition).md "劣势地形：藤蔓 (状态)") | [移动速度](Movement_speed.md "移动速度") 减半。 |
| [劣势地形：蛛网](Difficult_Terrain_colon__Web_(Condition).md "劣势地形：蛛网 (状态)") | [移动速度](Movement_speed.md "移动速度") 减半。 |
| [植物滋长](Plant_Growth_(Condition).md "植物滋长 (状态)") | 地面被植物和杂草覆盖。[移动速度](Movement_speed.md "移动速度") 减为四分之一。 |

  - 移动消耗加倍不仅适用于移动，也适用于 [备战（近战）](Brace_(Melee).md "备战（近战）")、[备战（远程）](Brace_(Ranged).md "备战（远程）") 和 [准备](Prepare.md "准备") 等动作。
  - 一旦离开该地表，移动消耗将恢复正常。
- 你可以通过以下方式获得对 **劣势地形** 效果的免疫：[友好步伐之靴](Boots_of_Genial_Striding.md "友好步伐之靴")、[黯狱之靴](Helldusk_Boots.md "黯狱之靴")、[自由动作戒指](Ring_of_Free_Action.md "自由动作戒指")、[行动自如](Freedom_of_Movement.md "行动自如")、[陆地步伐：劣势地形](Land's_Stride_colon__Difficult_Terrain.md "陆地步伐：劣势地形")、[增强无甲移动](Advanced_Unarmoured_Movement.md "增强无甲移动")、[灵活移动：回避劣势地形](Mobile_colon__Evade_Difficult_Terrain.md "灵活移动：回避劣势地形")。
  - 请注意，许多地表还具有其他效果，例如 [倒伏](Prone_(Condition).md "倒伏 (状态)")，而你 **不会** 对这些效果免疫。
