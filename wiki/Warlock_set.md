# 邪术师套装

**邪术师套装**包含与[邪术师](Warlock.md "邪术师")机制互动的物品。

| 名称 | 装备栏位 | 效果 |
| --- | --- | --- |
| 卑鄙法杖 | [主手近战](Main_Hand_Melee.md "主手近战") ([长棍](Quarterstaves.md "长棍")) | [奥术增强](Arcane_Enchantment_(+1).md "奥术增强 (+1)")你获得+1加值至[法术豁免DC](Spell_save_DC.md "法术豁免DC")和[法术攻击](Spell_Attack.md "法术攻击")掷骰。[凯蒂夫的修复术](Caitiff's_Restoration.md "凯蒂夫的修复术") 补充1个已消耗的邪术师[法术位](Spell_Slot.md "法术位"). ( 重充: [长休](Long_Rest.md "长休").) |
| 闪电誓缚战锤 | [主手近战](Main_Hand_Melee.md "主手近战") ([战锤](Warhammers.md "战锤")) | [心仪武器](Favoured_Weapon.md "心仪武器")此武器具有+1伤害加值和[攻击掷骰](Attack_roll.md "攻击掷骰")加值。[动电电流](Galvanic_Currents.md "动电电流")此武器造成额外1d6⁠⁠[闪电](Lightning.md "闪电")伤害。此锤子的魔法能力仅当它被[奥法骑士](Eldritch_Knight.md "奥法骑士")誓缚或作为[邪术师](Warlock.md "邪术师")的魔契武器时才生效。这些效果是武器+1增强的额外加成。 |
| 灾祸短剑 | [主手近战](Main_Hand_Melee.md "主手近战") ([短剑](Shortswords.md "短剑")) | [心仪武器](Favoured_Weapon.md "心仪武器")此武器具有+1伤害加值和[攻击掷骰](Attack_roll.md "攻击掷骰")加值。[灾祸连接](Baneful_Bond.md "灾祸连接")命中时，可能对目标施加[灾祸术](Bane_(Condition).md "灾祸术 (状态)")。此短剑的魔法能力仅当它被[奥法骑士](Eldritch_Knight.md "奥法骑士")誓缚或作为[邪术师](Warlock.md "邪术师")的魔契武器时才生效。这些效果是武器+1增强的额外加成。 |
