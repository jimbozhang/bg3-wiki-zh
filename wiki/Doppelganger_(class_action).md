# 变形怪 (职业动作)

**变形怪**是[变形怪](Doppelganger.md "变形怪")使用的职业动作。此动作允许变形怪变形成另一种生物。

## 描述

变形成你见过的小型或中级生物。

## 属性

消耗
[附赠动作](Actions.md#Resources "动作")
详情
范围：自身
充能：每驱散
