# Modding:创建发型模组

此页面为[模组页面](Modding_colon_Modding_resources.md "Modding:模组资源")，遵循其独立于Wiki其余部分的规则和标准。
本页面将阐述创建您专属的《博德之门3》发型模组的步骤。

1.  在开始模组制作前，如果您使用非原版素材，请务必检查[权限](https://bg3.wiki/wiki/Modding:Getting_Started_with_BG3_Modding#example-of-permissions)。
2.  下载以下必备的发型模组工具：
    1.  [Blender](https://www.blender.org/download/lts/3-6/)（推荐版本 3.6）
        *   用于导入、编辑和导出发型网格本身
    2.  [Lslib](https://github.com/Norbyte/lslib)（作者：Norbyte）
        *   用于将您的自定义发型适配至原版发型
    3.  [BG3 Multi-Tool](https://github.com/ShinyHobo/BG3-Modders-Multitool)（作者：ShinyHobo）
        *   用于搜索游戏文件
    4.  [Collada/GR2 导入/导出 Blender 插件](https://github.com/Norbyte/dos2de_collada_exporter)（作者：Norbyte）
        *   用于导入/导出 BG3 使用的 .GR2 文件
    5.  [VSCode](https://code.visualstudio.com/) 或 [Notepad++](https://notepad-plus-plus.org/)
        *   用于编辑代码文件
3.  [发型模组网格设置](Modding_colon_Hair_Mod_Mesh_Setup.md "Modding:发型模组网格设置")
    *   在此设置网格本身，即您发型的可见部分。
4.  [发型模组文件设置](Guide_colon_Hair_Mod_File_Setup.md "Guide:发型模组文件设置")
    *   在此设置代码以将模组纳入游戏。
5.  [打包您的模组以供安装](Modding_colon_Packaging_mods.md "Modding:打包模组")
6.  收益！
