# 沉默

**沉默** 是一项被动功能，可防止生物施放法术。

## 描述

此生物无法说话或施放具有言语成分的法术

## 如何习得

由以下生物使用：

- [血髓骷髅](Bloodmarrow_Skeleton.md "血髓骷髅"), [死亡牧者](Death_Shepherd.md "死亡牧者"), [堕落斗士](Fallen_Champion.md "堕落斗士"), [堕落战士](Fallen_Fighter.md "堕落战士"), [死去的游侠](Fallen_Ranger.md "死去的游侠"), [加雷特·多特迈尔](Garret_Dortmell.md "加雷特·多特迈尔"), [巨骷髅](Giant_Skeleton.md "巨骷髅"), [戈迪](Godey.md "戈迪"), [死灵侍僧](Necromite.md "死灵侍僧"), [骷髅](Skeleton.md "骷髅")

## 备注

- 与 [沉默](Silenced_(Condition).md "沉默 (状态)") 不同，此状态没有固有的 [雷鸣](Thunder.md "雷鸣") 伤害免疫。
