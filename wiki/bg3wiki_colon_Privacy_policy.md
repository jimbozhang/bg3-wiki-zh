## bg3.wiki 收集的数据

Bg3.wiki 本身不会收集超出 Web 服务器和 MediaWiki 软件正常操作范围的用户数据。

唯一的例外是，如果您[申请付款](bg3wiki_colon_Payment.md "bg3wiki:Payment")以获取您过去的贡献，在这种情况下，您在申请表中输入的信息将被存储以处理您的付款，且不用于任何其他目的。

### 服务器日志

Web 请求（包括页面访问以及加载相关资源，如 CSS 或 JavaScript 代码、图像等）可能会记录在私有服务器日志中。

这些日志包含以下信息：

- 请求的时间和日期
- 请求来源的 IP 地址
- 请求的页面或资源
- 用户设备发送的 "User-Agent" HTTP 标头
- 用户设备发送的 "Referrer" HTTP 标头

此类日志通常在服务器上保留最多一个月。除非法律要求，否则不会与任何第三方共享。

### MediaWiki 日志

创建或编辑页面，包括上传文件（这会创建一个文件页面），会导致在相关页面的关联**历史**页面上**公开**记录该操作，这是 MediaWiki 的标准做法。

此公共日志包含执行操作的用户的用户名（如果已登录）或 IP 地址（如果未登录），以及操作发生的时间和日期。

这些公共日志作为 MediaWiki 数据库的一部分无限期保留。

### 付款申请表

如果您填写了从页面 [bg3wiki:Payment](bg3wiki_colon_Payment.md "bg3wiki:Payment") 链接的付款申请表，则输入的数据将被存储，唯一目的是处理您当前和未来的付款。该数据绝对不用于任何其他目的，也不会传递给任何第三方，除非是您选择的付款处理商，如 Wise 或 PayPal。

## 来自广告服务器的第三方 Cookie

bg3.wiki 上出现的广告可能由广告合作伙伴提供，他们可能会设置 Cookie。这些 Cookie 允许广告服务器在每次向您发送在线广告时识别您的计算机，以汇编关于您或使用您计算机的其他人的信息。此信息使广告网络能够（除其他外）提供他们认为您最感兴趣的定向广告。

由上述第三方广告服务器提供的 Cookie 不受本隐私政策的管辖，而是受各自合作伙伴的隐私政策管辖。

### Publift

bg3.wiki 上的一些广告由 Publift Pty Limited 管理。如果使用 Publift 的广告服务，Publift Pty Limited 可能会收集和使用某些聚合和匿名数据用于广告目的。要了解有关收集的数据类型、数据使用方式以及您作为用户的选择的更多信息，请访问：

<https://www.publift.com/privacy-policy>

### Playwire

bg3.wiki 上的一些广告由 Playwire LLC 管理。如果使用 Playwire 发布商广告服务，Playwire LLC 可能会收集和使用某些聚合和匿名数据用于广告目的。要了解有关收集的数据类型、数据使用方式以及您作为用户的选择的更多信息，请访问：

<https://www.playwire.com/privacy-policy>
