| | |

**灾祸豕**是通过[驯兽师](Beast_Master.md "驯兽师")的[野猪伙伴](Boar_Companion.md "野猪伙伴")能力召唤的[野猪](Boar.md "野猪")，是[游侠伙伴](Ranger's_Companion.md "游侠伙伴")的变体。灾祸豕是[野猪座](Verres.md "野猪座")和[野猪座](Verres_Major.md "野猪座")在驯兽师达到 11 级后的完全升级形态。

## 战斗

### 攻击与能力

[野猪冲撞](Boar_Charge.md "野猪冲撞") ()

伤害：2~5

1d4 + 1⁠[挥砍](Slashing.md "挥砍")

向前冲锋，攻击路径上的所有敌人，并可能将其击至[倒伏](Prone_(Condition).md "倒伏（状态）")。

不会引发[借机攻击](Opportunity_Attacks.md "借机攻击")。

[力量](Strength.md "力量") [豁免检定](Saving_throws.md "豁免检定")

[疾走](Dash_(bonus_action).md "疾走（附赠动作）") ()
本回合移动更远距离：双倍你的[移动速度](Movement_speed.md "移动速度")。

范围：自身

[撤离：附赠动作](Disengage_colon__Bonus_Action.md "撤离：附赠动作") ()
安全撤退。移动不会引发[借机攻击](Attack_of_Opportunity.md "借机攻击")。

范围：自身

[狂怒攻击](Frenzied_Strike_(Boar).md "狂怒攻击（野猪）") ()

伤害：3~8

1d6 + 2⁠[挥砍](Slashing.md "挥砍")

在狂怒状态下以附赠动作刺穿敌人。

近战：1.5 米 (5 英尺)

[协助](Help_(bonus_action).md "协助（附赠动作）") ()
帮助需要帮助的盟友。移除[燃烧](Burning_(Condition).md "燃烧（状态）")、[倒地](Downed_(Condition).md "倒地（状态）")、\_(状态)[诱捕](Ensnared_(Ensnaring_Strike)_(Condition).md "诱捕（诱捕打击）（状态）")、[缠绕](Entangled_(Condition).md "缠绕（状态）")、[网缚](Enwebbed_(Condition).md "网缚（状态）")、[倒伏](Prone_(status_group).md), [沉睡](Sleeping_(Condition).md "沉睡（状态）")。

范围：1.5 米 (5 英尺)

[激起淤泥](Kick_Up_Muck.md "激起淤泥") ()
将目标裹满泥巴，使其[浑身泥泞](Mud-Drenched.md "浑身泥泞")。

范围：自身

[狂暴](Rage_(Boar).md "狂暴（野猪）") ()

进入狂乱状态，增大体型并通过原始愤怒变得更强。

狂暴时，你的近战攻击造成额外 2+1d4⁠⁠[挥砍](Slashing.md "挥砍") _\[[参见注释](Rage_(Boar).md#Notes).md#Notes> "狂暴（野猪）")\]_ 伤害，并且你处于[巨化](Enlarged_(Condition).md "巨化（状态）") _\[[参见注释](Rage_(Boar).md#Notes).md#Notes> "狂暴（野猪）")\]_ 状态。

你获得对物理伤害的[抗性](Resistance.md "抗性")，以及力量[检定](Checks.md "检定")和[豁免检定](Saving_throws.md "豁免检定")的[优势](Advantage.md "优势")。

范围：自身
充能：每场战斗

[牙撕攻击](Tusk_Attack_(Boar).md "牙撕攻击（野猪）") ()

伤害：3~8

1d6 + 2⁠[挥砍](Slashing.md "挥砍")

刺穿目标。

近战：1.5 米 (5 英尺)

由于[猎物的气味](Prey's_Scent.md "猎物的气味")被动，驯兽师游侠可能希望攻击被[猎人印记](Hunter's_Mark.md "猎人印记")标记的目标以获得最佳伤害。
