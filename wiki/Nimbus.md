# 尼姆巴斯

| | |
| --- | --- |
| 体型 | [中级](Medium.md "中级") |
| 类型 | [怪兽](Monstrosity.md "怪兽") |
| 种族 | [幽影獒犬](Shadow_Mastiff.md "幽影獒犬") |
| 生命值 | 33 |
| 护甲等级 | 18 |
| 先攻 | +2 |
| 移动速度 | 15 米 (50 英尺) |

**尼姆巴斯**是通过[暗影魔法](Shadow_Magic.md "暗影魔法")术士的[凶兆猎犬](Hound_of_Ill_Omen.md "凶兆猎犬")能力召唤的[幽影獒犬](Shadow_Mastiff.md "幽影獒犬")。

## 战斗

[不祥啃咬](Ominous_Bite.md "不祥啃咬")

伤害：3~18

3d6⁠[穿刺](Piercing.md "穿刺")

啃咬目标并可能对其施加[凶兆](Hound's_Omen_(Condition).md "凶兆（状态）")。

[体质](Constitution.md "体质") [豁免检定](Saving_throws.md "豁免检定")
近战：1.5 米 (5 英尺)

[暗影缠绕](Umbral_Tangle.md "暗影缠绕")

伤害：2~12

2d6⁠[黯蚀](Necrotic.md "黯蚀")

用暗影藤蔓困住你的敌人。

豁免检定成功时，目标仍会受到一半伤害。

[力量](Strength.md "力量") [豁免检定](Saving_throws.md "豁免检定")
范围：自身

[暗影分裂](Splinter_Shadow.md "暗影分裂")
当你受到不造成[光耀伤害](Radiant_damage.md "光耀伤害")的近战攻击时，分裂出一部分自身，从暗影中生成一个新的版本。

此外，当移动到友方[被诅咒的幽鬼](Accursed_Spectre.md "被诅咒的幽鬼")相邻位置时，尼姆巴斯会获得 \_(状态)[暗影灌注](Shadow_Infusion_(Hound_of_Ill_Omen)_(Condition).md "暗影灌注（凶兆猎犬）（状态）") 状态，该状态会赋予幽鬼一个特殊反应，直到下一次长休。当召唤者用近战武器攻击击中目标时，会触发该反应。

[在你身后](Right_Behind_You.md "在你身后")

伤害：1~12 + 调整值

1d12 + 召唤者魅力调整值⁠[黯蚀](Necrotic.md "黯蚀")

在召唤者的受害者身旁显形。

范围：18 米 (60 英尺)

## 备注

- 尽管不祥啃咬是武器攻击，但[巨像灵药](Elixir_of_the_Colossus.md "巨像灵药")和其他武器伤害调整值不适用于它。相反，一些对尼姆巴斯有效的[灵药](Elixirs.md "灵药")包括[战斗法师之力灵药](Elixir_of_Battlemage's_Power.md "战斗法师之力灵药")、[嗜血灵药](Elixir_of_Bloodlust.md "嗜血灵药")和[警觉灵药](Elixir_of_Vigilance.md "警觉灵药")。
- 尽管没有明确说明，但尼姆巴斯可以看到魔法黑暗。这意味着他不会受到[失明术](Blindness_(Condition).md "失明术（状态）")的影响，并且在施法者的[黑暗术](Darkness_(cloud).md "黑暗术（云）")云中战斗时具有优势。
